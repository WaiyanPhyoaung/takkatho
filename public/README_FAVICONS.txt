Generate raster favicons from favicon.svg
Suggested (locally using svgexport via npx):
  npx svgexport favicon.svg favicon-32x32.png 32:32
  npx svgexport favicon.svg favicon-16x16.png 16:16
  npx svgexport favicon.svg android-chrome-192x192.png 192:192
  npx svgexport favicon.svg android-chrome-512x512.png 512:512
  npx svgexport favicon.svg apple-touch-icon.png 180:180

Or use https://realfavicongenerator.net with favicon.svg and copy outputs here.
