### Python ဆိုတာ ဘာလဲ?

Python ဆိုတာ ဖတ်ရလွယ်ကူပြီး ရေးရလွယ်ကူတဲ့ Programming Language တစ်ခုပါ။ လူတွေ နားလည်တဲ့ အင်္ဂလိပ်စာနဲ့ အတော်လေး ဆင်တူပါတယ်။

### ဘာလို့ Python ကို လေ့လာသင့်တာလဲ?

- **လွယ်ကူတယ်:** Code တွေက ရှင်းလင်းပြီး ဖတ်ရလွယ်ပါတယ်။ Beginner တွေအတွက် အသင့်တော်ဆုံးပါပဲ။
- **နေရာတိုင်းမှာ သုံးတယ်:** Web Development, Data Science, AI (Artificial Intelligence), Automation စတဲ့ နေရာပေါင်းစုံမှာ သုံးပါတယ်။
- **Community ကြီးတယ်:** အခက်အခဲရှိရင် မေးစရာ နေရာတွေ၊ အဖြေရှာစရာ နေရာတွေ အများကြီး ရှိပါတယ်။

### ဒီ Course မှာ ဘာတွေ သင်ရမလဲ?

1. **Variables & Data Types:** Data တွေကို ဘယ်လို သိမ်းမလဲ။
2. **Control Flow:** အခြေအနေပေါ်မူတည်ပြီး အလုပ်လုပ်ခိုင်းတာတွေ (If/Else) နဲ့ ထပ်ခါထပ်ခါ လုပ်ခိုင်းတာတွေ (Loops)။
3. **Functions:** Code တွေကို ပြန်သုံးလို့ရအောင် ဘယ်လို စုစည်းမလဲ။
4. **Data Structures:** Data အများကြီးကို စနစ်တကျ ဘယ်လို သိမ်းမလဲ (Lists, Dictionaries)။
5. **Object-Oriented Programming (OOP):** Code တွေကို ပိုမို စနစ်ကျအောင် ဘယ်လို တည်ဆောက်မလဲ။
6. **File Handling & Errors:** File တွေ ဖတ်တာ၊ ရေးတာနဲ့ Error တွေကို ဘယ်လို ဖြေရှင်းမလဲ။
