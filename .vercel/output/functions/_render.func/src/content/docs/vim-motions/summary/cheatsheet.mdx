သင်နေ့စဉ် Code ရေးတဲ့အခါ ဘေးမှာဖွင့်ထားပြီး ခဏခဏ ပြန်ကြည့်လို့ရမယ့် အသုံးအများဆုံး Vim Motions အကျဉ်းချုပ် ဖြစ်ပါတယ်။

### အခြေခံ သွားလာခြင်း (Basic Navigation)
| Command | လုပ်ဆောင်ချက် (Action) |
| :---: | :--- |
| `h` `j` `k` `l` | ဘယ်၊ အောက်၊ အပေါ်၊ ညာ (Left, Down, Up, Right) |
| `w` | နောက်စကားလုံး၏ အစသို့ |
| `b` | ယခင်စကားလုံး၏ အစသို့ |
| `e` | လက်ရှိ/နောက်စကားလုံး၏ အဆုံးသို့ |
| `0` | စာကြောင်း၏ အစဆုံးသို့ |
| `^` | စာကြောင်း၏ ပထမဆုံးစာလုံးသို့ (Space များကျော်၍) |
| `$` | စာကြောင်း၏ အဆုံးသို့ |

### အဆင့်မြင့် သွားလာခြင်း / ရှာဖွေခြင်း (Advanced / Searching)
| Command | လုပ်ဆောင်ချက် (Action) |
| :---: | :--- |
| `gg` | ဖိုင်၏ အပေါ်ဆုံးသို့ သွားရန် |
| `G` | ဖိုင်၏ အောက်ဆုံး (နောက်ဆုံး) သို့ သွားရန် |
| `Ctrl + u` | မျက်နှာပြင် တစ်ဝက် အပေါ်သို့ တက်ရန် (Up) |
| `Ctrl + d` | မျက်နှာပြင် တစ်ဝက် အောက်သို့ ဆင်းရန် (Down) |
| `f[char]` | စာကြောင်းအတွင်း ကိုယ်လိုချင်သော စာလုံးသို့ သွားရန် |
| `t[char]` | ကိုယ်လိုချင်သော စာလုံး၏ အရှေ့တွင် ရပ်ရန် |
| `;` | `f` သို့မဟုတ် `t` ဖြင့် ရှာထားသည်ကို ထပ်၍ ရှာရန် |
| `%` | အဖွင့်/အပိတ် ကွင်းများ `{}`, `()` ရှာရန် |
| `/[text]` | ဖိုင်တစ်ခုလုံးအတွင်း စာလုံးရှာရန် (`n` ဖြင့် နောက်တစ်ခုသို့ ဆက်သွားနိုင်သည်) |

### ပြင်ဆင်ခြင်း Operators (Editing Operators)
| Command | လုပ်ဆောင်ချက် (Action) |
| :---: | :--- |
| `d` | ဖျက်ရန် (Delete / Cut) |
| `c` | ဖျက်ပြီး အသစ်ပြင်ရန် (Change -> ချက်ချင်း Insert mode ဝင်သွားမည်) |
| `y` | ကူးယူရန် (Yank / Copy) |
| `p` | ပြန်ချရန် (Paste) |
| `x` | စာတစ်လုံးတည်း ဖျက်ရန် (Delete character) |
| `u` | Undo လုပ်ရန် (Ctrl + Z ကဲ့သို့) |
| `Ctrl + r` | Redo လုပ်ရန် |

### Vim ၏ မှော်ဆန်သောတွဲဖက်မှုများ (Text Objects & Combos)
*(အထက်ပါ `d`, `c`, `y` တို့နှင့် တွဲသုံးရမည်။ `i` = inner, `a` = around)*

| Command | လုပ်ဆောင်ချက် (Action) |
| :---: | :--- |
| `dd` | စာတစ်ကြောင်းလုံး အပြည့် ဖျက်ရန် |
| `yy` | စာတစ်ကြောင်းလုံး အပြည့် Copy ကူးရန် |
| `C` သို့ `c$` | လက်ရှိနေရာမှ စာကြောင်းအဆုံးထိလွတ်ပြီး အသစ်ပြင်ရန် |
| `diw` | စကားလုံးတစ်လုံးလုံးကို လှလှပပ ဖျက်ရန် |
| `ci"` | `""` ထဲရှိ စာများကို ဖျက်ပြီး အသစ်ပြင်ရန် |
| `ya{` | `{}` အပိတ်အဖွင့် အတွင်းရောအပြင်ပါ အားလုံး Copy ကူးရန် |
| `cap` | စာပိုဒ် (Paragraph) တစ်ခုလုံး ဖျက်ပြီး အသစ်ပြန်စီရန် |
