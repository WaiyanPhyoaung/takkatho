Programmer တော်တော်များများက **Vim** လို့ကြားတာနဲ့ "သုံးရခက်တယ်၊ အပြင်ထွက်ဖို့တောင် မလွယ်ဘူး" လို့ ထင်တတ်ကြပါတယ်။ ဒါပေမယ့် Vim ရဲ့ "Keyboard ကနေ လက်မခွာဘဲ စာတွေကို မြန်မြန်ဆန်ဆန် ပြင်နိုင်တဲ့ စွမ်းရည် (Vim Motions)" ကို တတ်သွားပြီဆိုရင်တော့ တခြားနည်းလမ်းတွေကို ပြန်သွားချင်တော့မှာ မဟုတ်ပါဘူး။

### Vim Editor ကို သုံးစရာမလိုပါဘူး!
ဒီ Course မှာ ကျွန်တော်တို့ သင်သွားမှာက Terminal ထဲကခက်ခဲတဲ့ Vim Editor အကြောင်း မဟုတ်ပါဘူး။ **Vim Motions** လို့ခေါ်တဲ့ **"စာစီစာရိုက် မြန်ဆန်စေတဲ့ Keyboard Shortcuts အစုအဝေး"** အကြောင်းဖြစ်ပါတယ်။

သင်က **VS Code**, **IntelliJ**, **WebStorm** ဘာပဲသုံးသုံး Extension ဆိုင်ကနေ "Vim" လို့ရှာပြီး Install လုပ်လိုက်တာနဲ့၊ သင်အကြိုက်ဆုံး Editor ထဲမှာ Vim ရဲ့ အစွမ်းတွေကို အပြည့်အဝ အသုံးပြုလို့ ရသွားပါပြီ။

### Modal Editing ဆိုတာ ဘာလဲ?
ပုံမှန် Text Editor တွေ (ဥပမာ - Notepad, VS Code အဟောင်း) မှာ စာရိုက်ရင် စာတွေဝင်လာပါတယ်။ ဒါပေမယ့် Vim မှာတော့ Mode (အခြေအနေ) ကွဲပြားမှုတွေရှိပါတယ်။

အဓိက Mode (၃) မျိုး ရှိပါတယ် -

1. **Normal Mode (ပံမှန်အခြေအနေ):**
   * Vim ကို ဖွင့်ဖွင့်ချင်း ရောက်နေတဲ့ Mode ဖြစ်ပါတယ်။
   * ဒီ Mode မှာ Keyboard က ခလုတ်တွေဟာ စာမရိုက်ပါဘူး။ **"Command (အမိန့်ပေးစနစ်)"** တွေအဖြစ် အလုပ်လုပ်ပါတယ်။ (ဥပမာ - `w` ကိုနှိပ်ရင် ရှေ့ကိုတစ်လုံးကျော်မယ်၊ `d` ကိုနှိပ်ရင် ဖျက်မယ် စသဖြင့်)
   * *မှတ်ချက်: တခြား Mode တွေကနေ Normal Mode ပြန်သွားချင်ရင် `Esc` ကို နှိပ်ရပါတယ်။*

2. **Insert Mode (စာရိုက်သည့်အခြေအနေ):**
   * ပုံမှန် Notepad တွေလို စာသုံးစွဲရိုက်ချင်ရင် သုံးတဲ့ Mode ဖြစ်ပါတယ်။
   * Normal Mode ကနေ `i` (Insert) ကို နှိပ်လိုက်ရင် ဒီ Mode ထဲ ရောက်သွားပါမယ်။

3. **Visual Mode (Select မှတ်သည့်အခြေအနေ):**
   * Mouse နဲ့ စာတွေကို Select မှတ်သလိုမျိုးလုပ်ချင်ရင် သုံးပါတယ်။
   * Normal Mode ကနေ `v` ကို နှိပ်ပြီး ဝင်ရပါတယ်။

### ဥပမာ - Mouse မလိုတဲ့ ဘဝ
ပုံမှန် သင်က "hello world" ဆိုတဲ့ စာကြောင်းကို ဖျက်ချင်ရင် Mouse ကိုကိုင်၊ လိုက်ဆွဲ (Select မှတ်) ပြီး `Backspace` နှိပ်ရပါတယ်။ 
Vim Motions တတ်သွားရင်တော့ စာကြောင်းပေါ်ရောက်နေတာနဲ့ `dd` လို့ ကီးဘုတ်ကနေ မြန်မြန်လေး ၂ ချက်နှိပ်လိုက်ရုံနဲ့ အဲဒီစာကြောင်း တစ်ကြောင်းလုံး ပျက်သွားပါလိမ့်မယ်။

ကဲ... နောက်သင်ခန်းစာတွေမှာ တကယ့် လက်တွေ့ Motions တွေကို စတင်လေ့လာကြရအောင်။
