---

Terminal မှာ အလုပ်လုပ်နေရင်း ကိုယ့်ကွန်ပျူတာရဲ့ အခြေအနေ၊ Manual (လမ်းညွှန်စာအုပ်) တွေ ဖတ်ချင်တဲ့အခါ အောက်ပါ Commands လေးတွေက အရမ်းအသုံးဝင်ပါတယ်။

### 1. `clear`
Terminal screen ကြီး အပေါ်နားမှာ စာတွေအရမ်းများပြီး ရှုပ်နေရင် ရှင်းပစ်ဖို့ `clear` ကို သုံးနိုင်ပါတယ်။ (Keyboard Shortcut အနေနဲ့ `Ctrl + L` ကိုလည်း သုံးလို့ရပါတယ်။)

```bash
clear
```

### 2. `history`
အရင်က သင်ရိုက်ခဲ့ဖူးတဲ့ Command အဟောင်းတွေကို ပြန်ကြည့်ချင်ရင် သုံးပါတယ်။

```bash
history

# Output နမူနာ -
#  121  cd Desktop
#  122  mkdir project
#  123  ls -a
```

### 3. `man` (Manual)
Command တစ်ခုရဲ့ "အသုံးပြုပုံ လမ်းညွှန်စာအုပ်" ထဲကို ဝင်ဖတ်ချင်ရင် သုံးပါတယ်။ **Command တွေ ဘယ်လိုအလုပ်လုပ်လဲ မမှတ်မိတော့ရင် Google ခေါက်စရာမလိုဘဲ `man` နဲ့ ရှာဖတ်လို့ရပါတယ်။**

```bash
# ls command ကို ဘယ်လိုသုံးရမလဲ ဖတ်ရန်
man ls

# mkdir အကြောင်း ဖတ်ရန်
man mkdir
```
*(ထွက်ချင်လျှင် `q` ကို နှိပ်ပါ)*

### 4. `whoami`
ယခု Terminal ကို အသုံးပြုနေတဲ့ လက်ရှိ User ရဲ့ နာမည်ကို သိချင်ရင် သုံးပါတယ်။

```bash
whoami
```

### 5. `top` သို့မဟုတ် `htop`
ကွန်ပျူတာရဲ့ CPU, RAM အသုံးပြုမှုတွေနဲ့ လက်ရှိ Run နေတဲ့ Programs (Processes) တွေကို Task Manager လိုမျိုး ကြည့်ချင်ရင် သုံးပါတယ်။ အပြင်ထွက်ဖို့ `q` (Quit) ကို နှိပ်ပါ။

```bash
top
```

### 6. အင်တာနက် ချိတ်ဆက်မှု စစ်ဆေးခြင်း (`ping`)
ကိုယ့်စက်ကနေ အင်တာနက် (သို့) ပေးထားတဲ့ Website ဆီကို Network ရောက်/မရောက် စစ်ဆေးချင်ရင် သုံးပါတယ်။

```bash
# Google သို့ မက်ဆေ့ခ်ျပို့၍ စမ်းသပ်ခြင်း
ping google.com
```
*(Ping ထွက်နေတာကို အဆုံးသတ်ရပ်တန့်ဖို့ `Ctrl + C` ကိုနှိပ်ပါ)*
