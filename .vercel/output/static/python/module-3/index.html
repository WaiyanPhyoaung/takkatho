Function ဆိုတာ ဘာလဲ?

Section titled “Function ဆိုတာ ဘာလဲ?”

Function ဆိုတာ Code တွေကို အစုလိုက် ဖွဲ့ထားတာပါ — “ဟင်းချက်နည်း (Recipe)” တစ်ခုနဲ့ တူပါတယ်။ ဟင်းချက်နည်း တစ်ကြိမ်ရေးထားရင် ကြိုက်တဲ့အချိန်တိုင်း အဲ့ဒီနည်းအတိုင်း ချက်လို့ ရသလိုပဲ — Function ကို တစ်ကြိမ် ဖန်တီးပြီးရင် လိုတဲ့အချိန်တိုင်း ခေါ်သုံးနိုင်ပါတယ်။

ဘာကြောင့် Functions သုံးမလဲ? — DRY (Don’t Repeat Yourself) အတွက်ပါ။ Code တွေ ထပ်ခါထပ်ခါ မရေးဘဲ Function ထဲ ထည့်ပြီး လှမ်းခေါ်သုံးလိုက်တာ ဖြစ်ပါတယ်။
