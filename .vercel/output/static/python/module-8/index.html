Mini Project: Number Guessing Game

ဒီ Module ပြီးရင် ခင်ဗျားလုပ်နိုင်မှာများ

  • random module သုံးပြီး Number Guessing Game ဆောက်နိုင်မယ်
  • while Loop, try/except နဲ့ Functions တွေကို ပေါင်းစပ်အသုံးပြုနိုင်မယ်
  • Difficulty Level ရွေးနိုင်တဲ့ Game Version ကို တည်ဆောက်နိုင်မယ်
  • Code တစ်ကြောင်းချင်းစီရဲ့ ရှင်းလင်းချက်ကို ဖတ်ပြီး ဘယ် Module က ဘာလုပ်သလဲဆိုတာ သိနိုင်မယ်

ဒီ Module မစခင်

Module 1-7 ကို ပြီးမှသာ ဆက်ကြည့်ပေးပါ။ ဒီ Mini Project မှာ Variables, Loops, Functions, Modules နဲ့ Error Handling တွေကို ပေါင်းစပ်အသုံးပြုသွားပါမယ်။

ဒီ Course မှာ သင်ယူခဲ့တဲ့ Variables, Loops, If/Else တွေနဲ့ Modules တွေကို ပေါင်းစပ်ပြီး “ဂဏန်းခန့်မှန်းတဲ့ ဂိမ်း (Number Guessing Game)” ကို ဆောက်ကြည့်ကြရအောင်။

ဂိမ်းရဲ့ စည်းမျဉ်းများ

Section titled “ဂိမ်းရဲ့ စည်းမျဉ်းများ”
  1. ကွန်ပျူတာက 1 ကနေ 100 ကြား ဂဏန်းတစ်ခုကို ကျပန်း ရွေးထားပါမယ်
  2. ကစားသမားက မှန်းပြီး ရိုက်ထည့်ရပါမယ်
  3. မှန်းတဲ့ ဂဏန်းက ကြီးနေရင် “ကြီးနေတယ်”၊ ငယ်နေရင် “ငယ်နေတယ်” ဆိုပြီး Hint ပေးပါမယ်
  4. အဖြေမှန်ရင် ဘယ်နှစ်ကြိမ်မြောက်မှာ မှန်သလဲဆိုတာ ပြပေးပါမယ်

Challenge Extensions — ကိုယ်တိုင် ထပ်မံဖြည့်စွက်ကြည့်ပါ

Section titled “Challenge Extensions — ကိုယ်တိုင် ထပ်မံဖြည့်စွက်ကြည့်ပါ”

ဒီ Game ကို ပိုကောင်းအောင် ဆက်လက် ကြိုးစားကြည့်ပါ:

  • High Score — အနည်းဆုံး ကြိမ်နဲ့ အဖြေမှန်တဲ့ Record ကို File ထဲမှာ သိမ်းပါ
  • Timer — time Module သုံးပြီး အချိန် ဘယ်လောက်ကြာသလဲ တွက်ပြပါ
  • Hint System — “ဒုတိယ Hint ဝယ်မလား?” ဆိုတဲ့ Option မျိုး ထည့်ပါ
  • Multiplayer — ကစားသမား ၂ ယောက် အလှည့်ကျ ကစားတဲ့ Mode ထည့်ကြည့်ပါ

နောက်တစ်ဆင့်

Game ကို သေချာ နားလည်အောင် ဆောက်နိုင်ပြီဆိုရင်တော့ Module 9 — Debugging မှာ Error တွေကို ပိုမိုထိရောက်စွာ ဘယ်လို ဖြေရှင်းရမလဲဆိုတာ သင်ကြားပေးသွားပါမယ်။ အဲ့ဒီနောက် Module 10 — Projects မှာ ပိုမိုကြီးမားတဲ့ Projects တွေကို တည်ဆောက်ကြပါမယ်!

Module 8 ပြီးဆုံးပါပြီ — မဆုတ်မနစ် ဆက်လက် လေ့လာကြပါစို့! 💪
