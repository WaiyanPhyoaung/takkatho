Setup & Installation

ဒီ Module ပြီးရင် ခင်ဗျားလုပ်နိုင်မှာများ

  • Python ကို ကွန်ပျူတာမှာ မှန်ကန်စွာ တပ်ဆင်နိုင်မယ်
  • VS Code Editor ကို Python အတွက် Setup လုပ်နိုင်မယ်
  • ကိုယ်ပိုင် .py ဖိုင် ဖန်တီးပြီး Run နိုင်မယ်
  • Python REPL ကို Scratchpad အဖြစ် သုံးနိုင်မယ်

လိုအပ်သောအရာများ

  • ကွန်ပျူတာ (Windows, Mac, Linux) တစ်လုံး
  • Internet ချိတ်ဆက်မှု (Download အတွက်)
  • Python နဲ့ Programming အကြောင်း ဘာမှမသိသေးလည်း စတင်လို့ရပါတယ် — ဒီ Module မှာ အကုန် ရှင်းပြပေးသွားမှာပါ

Python ဆိုတာ ဘာ?

Section titled “Python ဆိုတာ ဘာ?”

မစတင်ခင် Python ကို ဘာကြောင့် လေ့လာသင့်သလဲ ဆိုတာ တစ်ခွန်းတည်းနဲ့ ရှင်းပြရရင် — Python ဟာ လက်ရှိ ကမ္ဘာမှာ အသုံးအများဆုံး Programming Language တစ်ခုဖြစ်ပြီး Web, AI, Data Science, နဲ့ Automation တွေမှာ နေရာအနှံ့ သုံးနေကြလို့ပါပဲ။

အဆင့် ၁ — Python တပ်ဆင်ခြင်း

Section titled “အဆင့် ၁ — Python တပ်ဆင်ခြင်း”

python.org ဝဘ်ဆိုဒ်ကနေ Python ကို အခမဲ့ Download လုပ်ယူနိုင်ပါတယ်။

“Download Python 3.x.x” ဆိုတဲ့ ခလုတ်ကြီးကို နှိပ်ပြီး Installer ကို Download ယူလိုက်ပါ။

Windows အတွက် အရေးကြီးသောအချက်

Installer ကို Double-click နှိပ်ပြီး ဖွင့်တဲ့အခါ “Add Python to PATH” ဆိုတဲ့ Checkbox ကို မဖြစ်မနေ ✓ ထားပေးပါ။ ဒါမလုပ်ဘဲ Install လုပ်မိရင် Terminal မှာ python command ကို သုံးလို့ ရမှာမဟုတ်ပါဘူး။ အဲဒီအခါကျရင် Uninstall လုပ်ပြီး ပြန် Install ရမှာမို့လို့ ဒါကို အရင်ဆုံး သေချာစစ်ဆေးပါ။

Python မှန်မှန်ကန်ကန် တပ်ဆင်ပြီးပြီလားဆိုတာ စစ်ဆေးနည်း:

Terminal (Windows မှာ Command Prompt သို့မဟုတ် PowerShell) ကို ဖွင့်ပြီး အောက်ပါ Command ကို ရိုက်ထည့်ကြည့်ပါ:

ဒီလို Python version number ပြပေးတယ်ဆိုရင်တော့ မှန်ကန်စွာ တပ်ဆင်ပြီးသွားပါပြီ။

အဆင့် ၂ — VS Code တပ်ဆင်ခြင်း

Section titled “အဆင့် ၂ — VS Code တပ်ဆင်ခြင်း”

VS Code (Visual Studio Code) ဆိုတာ Microsoft က ဖန်တီးထားတဲ့ Code Editor တစ်ခုဖြစ်ပြီး Python ရေးဖို့အတွက် အသင့်တော်ဆုံး Editor ပါပဲ။ Code တွေ Highlight ပြပေးတာ၊ Error တွေ ချက်ချင်းညွှန်ပြတာ၊ Terminal ပါ built-in ပါဝင်တာတွေကြောင့် Beginner တွေ အတွက် အလွန် ကောင်းမွန်ပါတယ်။

  1. VS Code ကို Download လုပ်ပြီး Install လုပ်ပါ
  2. VS Code ကို ဖွင့်ပါ
  3. ဘယ်ဘက် Sidebar မှာ Extensions Icon (⊞) ကို နှိပ်ပါ
  4. Search Box မှာ Python လို့ ရိုက်ရှာပါ
  5. Microsoft ရဲ့ Python Extension ကို Install နှိပ်လိုက်ပါ

အဆင့် ၃ — ပထမဆုံး Python Program ရေးခြင်း

Section titled “အဆင့် ၃ — ပထမဆုံး Python Program ရေးခြင်း”

Folder ဖွင့်ခြင်း

Section titled “Folder ဖွင့်ခြင်း”

VS Code မှာ File → Open Folder ကနေ ခင်ဗျားရဲ့ Projects Folder ကို ဖွင့်ပါ (ဥပမာ Desktop/my-python-projects)။

File အသစ် ဖန်တီးခြင်း

Section titled “File အသစ် ဖန်တီးခြင်း”

Folder ထဲမှာ File အသစ် ဖန်တီးပြီး hello.py လို့ အမည်ပေးပါ (.py Extension ကို မမေ့ပါနဲ့ — Python ဖိုင် ဆိုတာ ဒါနဲ့ သိနိုင်တာပါ)။

Code ရေးခြင်း

Section titled “Code ရေးခြင်း”

Program Run ခြင်း

Section titled “Program Run ခြင်း”

VS Code Terminal ကိုဖွင့်ပါ (View → Terminal) ပြီးရင်:

ဒီလို ပြပေးရပါမယ်:

ဒါဆိုရင်တော့ ခင်ဗျားအနေနဲ့ Python လေ့လာဖို့ ပြင်ဆင်မှု အပြည့်အစုံ ပြီးသွားပါပြီ!

Python REPL — ချက်ချင်း စမ်းသပ်နိုင်သော Scratchpad

Section titled “Python REPL — ချက်ချင်း စမ်းသပ်နိုင်သော Scratchpad”

Terminal မှာ python လို့ ရိုက်ပြီး Enter နှိပ်လိုက်ရင် Python REPL (Read-Eval-Print Loop) ထဲ ရောက်သွားပါမယ်။ REPL ဆိုတာ Code တစ်ကြောင်းကို ချက်ချင်း Run ကြည့်လို့ ရတဲ့ ပုံစံပါ — Calculator တစ်ခုနဲ့ တူပါတယ်။

>>> ကို မြင်ရရင် REPL ထဲ ရောက်နေတာပါ။ ထွက်ချင်ရင်တော့ exit() လို့ ရိုက်ပါ။

REPL ကို ဘယ်အချိန်သုံးမလဲ?

Code တစ်ကြောင်းကို “ဒါ ဘယ်လို Output ထွက်မလဲ?” ဆိုပြီး စမ်းကြည့်ချင်တဲ့အခါ REPL ကနေ ချက်ချင်း မြင်နိုင်ပါတယ်။ ဖိုင်ဖွင့်ပြီး Run တာထက် ပိုမြန်ပါတယ်။

Error တွေ တွေ့ရင်

Section titled “Error တွေ တွေ့ရင်”

Setup လုပ်တာနဲ့ပတ်သက်ပြီး အဖြစ်များတဲ့ Error တွေကတော့:

Errorအဓိပ္ပာယ်ဖြေရှင်းနည်း
python is not recognizedPATH မထည့်ဘဲ Install လုပ်ထားလို့ပါPython ကို Uninstall လုပ်ပြီး “Add to PATH” check ထားပြီး ပြန် Install ပါ
SyntaxErrorCode ရေးပုံ မှားနေလို့ပါError message ရဲ့ line number ကို ကြည့်ပြီး ပြန်စစ်ဆေးပါ
File မတွေ့Terminal folder မှားနေလို့ပါhello.py ရှိတဲ့ Folder ထဲ Terminal ရောက်နေဖို့ လိုပါတယ်
