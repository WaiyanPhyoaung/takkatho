Data Structure ရွေးချယ်ရာ Myanmar ဥပမာ

  • List — ဆိုင်မှာ မှာထားသော ဟင်းစာများ (အစဉ်လိုက်၊ ထပ်ဝယ်လည်း ရ)
  • Dict — Customer Profile (ဖုန်းနံပါတ်နဲ့ နာမည် တွဲထားတာ)
  • Tuple — GPS Location (ရန်ကုန်ရဲ့ Latitude/Longitude — ပြောင်းလို့မရ)
  • Set — တက်ထားသောကျောင်းသားများ (ထပ်တူ မဖြစ်နိုင်)

ဘယ် Data Structure ကို ဘယ်အချိန် ရွေးမလဲ?

Section titled “ဘယ် Data Structure ကို ဘယ်အချိန် ရွေးမလဲ?”
Data Structureရွေးချယ်သင့်သောအချိန်ဥပမာ
Listအစဉ်လိုက် ဖြစ်ရမည်၊ ပြင်ရမည်၊ ထပ်သောတန်ဖိုး ပါနိုင်မှာထားသော Items, ကျောင်းသားစာရင်း
DictKey-Value တွဲနဲ့ ရှာနိုင်ရမည်User Profile, Settings Config
Tupleပြောင်းလဲမည် မဟုတ်သောတန်ဖိုးGPS Coordinates, RGB
SetUnique တန်ဖိုးသာ၊ Duplicate မလိုLogin Sessions, Tag List
