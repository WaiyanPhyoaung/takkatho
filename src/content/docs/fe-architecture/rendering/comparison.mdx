### ဘယ်ဟာကို ရွေးချယ်မလဲ?

#### ၁။ Admin Dashboard / User Profile
**CSR** ကို သုံးပါ။
- SEO လုံးဝ မလိုပါဘူး။ (Admin မျက်နှာပြင်ကို Google Search မှာ ပေါ်စရာ မလိုလို့ပါ)
- User က Login ဝင်ပြီးမှ သုံးတာဖြစ်လို့ စစချင်း Initial Load နည်းနည်း ကြာရင်လည်း ပြဿနာ မရှိပါဘူး။
- Data တွေက အမြဲတမ်း တပြေးညီ (Real-time) Update ဖြစ်နေဖို့ လိုအပ်ပါတယ်။

#### ၂။ News Portal / Social Media Feed
**SSR** ကို သုံးပါ။
- သတင်းတွေက အချိန်နဲ့ တပြေးညီ အမြဲတမ်း တက်နေဖို့ လိုအပ်ပါတယ်။
- Google News လို နေရာမျိုးမှာ ပါဖို့အတွက် SEO အထူး ကောင်းမွန်ဖို့ လိုအပ်ပါတယ်။

#### ၃။ Personal Blog / Documentation
**SSG** ကို သုံးပါ။
- Content တွေက နေ့တိုင်း၊ အချိန်တိုင်း မပြောင်းလဲပါဘူး။
- User အတွက် အမြန်ဆုံး (Fastest Experience) ဖြစ်ချင်ပါတယ်။
- Server ဘက်က အလုပ်မလုပ်ရတဲ့အတွက် Hosting စရိတ် အသက်သာဆုံး ဖြစ်ချင်ပါတယ်။

#### ၄။ E-commerce Product Listing (ကုန်ပစ္စည်းပြသသော စာမျက်နှာ)
**ISR** ကို သုံးပါ။
- ပစ္စည်းဈေးနှုန်းတွေက အချိန်မရွေး ပြောင်းလဲနိုင်ပါတယ်။
- ဒါပေမဲ့ SSR လောက် Server ကို ဝန်မပိစေချင်ပါဘူး။ (ပစ္စည်းတွေ အများကြီး ရှိလို့ပါ)
- ၁ နာရီ ဒါမှမဟုတ် ၁၀ မိနစ်လောက်နေမှ Price Update ပြန်ဖြစ်ရင် လုံလောက်ပါတယ်။
