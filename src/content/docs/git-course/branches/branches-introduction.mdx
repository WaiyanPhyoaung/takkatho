### Branch ဆိုတာ ဘာလဲ? (သီးခြား အလုပ်လုပ်သည့် လမ်းကြောင်း)

Project မှာ Feature အသစ် ထပ်ထည့်ချင်တဲ့အခါ အဓိက လမ်းကြောင်းကြီး (Main Branch) ကို မထိခိုက်စေဘဲ သီးသန့် အလုပ်လုပ်ချင်ရင် Branch တွေကို သုံးနိုင်ပါတယ်။ Branch ဆိုတာ ကိုယ့် Project Code ရဲ့ မိတ္တူပွားထားတဲ့ လမ်းကြောင်းခွဲလေး တစ်ခုပါပဲ။ ဒီလမ်းကြောင်းပေါ်မှာ အဓိက Code ကို မထိခိုက်စေဘဲ လွတ်လွတ်လပ်လပ် အလုပ်လုပ်လို့ ရပါတယ်။

Branch တွေ သုံးရတဲ့ အဓိက အကြောင်းရင်းကတော့ -
- Feature အသစ်တွေကို စိတ်ကြိုက် စမ်းသပ် ရေးသားနိုင်ဖို့
- အဆင်ပြေနေပြီးသား Code တွေကို မထိခိုက်စေဘဲ အမှား (Bug) တွေ ပြင်ဆင်နိုင်ဖို့ ဖြစ်ပါတယ် ခင်ဗျာ။

### Branch များကို ကြည့်ရှုခြင်း (`git branch`)

Project မှာ ဖန်တီးထားတဲ့ Branch တွေကို ကြည့်ချင်ရင် Terminal မှာ `git branch` လို့ ရိုက်ပါ။ Branch အားလုံးကို ပြပေးမှာဖြစ်ပြီး၊ လက်ရှိ ရောက်နေတဲ့ Branch ကိုတော့ `*` (ကြယ်ပွင့်) လေးနဲ့ ပြပေးပါလိမ့်မယ်။

### Branch အသစ် ဖန်တီးခြင်း (`git branch <branch-name>`)

Branch အသစ် ဖန်တီးချင်ရင် `git branch <Branch နာမည်>` လို့ ရိုက်ပါ။ ဥပမာ - `git branch add-user-login` လို့ ရိုက်လိုက်ရင် Branch အသစ်တစ်ခု ရပါပြီ။ ဒါပေမဲ့ အဲဒီ Branch ဆီကို ချက်ချင်း ရောက်သွားမှာတော့ မဟုတ်သေးပါဘူး။

### Branch ပြောင်းခြင်း (`git switch <branch-name>`)

ဖန်တီးထားတဲ့ Branch ဆီ သွားချင်ရင် `git switch <Branch နာမည်>` ကို သုံးရပါတယ်။ ဥပမာ - `git switch add-user-login` လို့ ရိုက်လိုက်ရင် အဲဒီ Branch ဆီ ရောက်သွားပါပြီ။ (Project ဖိုင်တွေကလည်း အဲဒီ Branch ရဲ့ အခြေအနေအတိုင်း အလိုအလျောက် ပြောင်းလဲသွားမှာ ဖြစ်ပါတယ်)။

### Branch အသစ်ဖန်တီးပြီး ချက်ချင်းပြောင်းခြင်း (Shortcut)

Branch အသစ် ဖန်တီးပြီးတာနဲ့ ချက်ချင်း ပြောင်းသွားချင်ရင် `git switch -c <Branch နာမည်>` Command ကို သုံးနိုင်ပါတယ်။ ဥပမာ - `git switch -c new-idea-branch` လို့ ရိုက်လိုက်ရင် "new-idea-branch" ကို ဖန်တီးပြီး ချက်ချင်း အဲဒီ Branch ဆီ ရောက်သွားမှာ ဖြစ်ပါတယ် ခင်ဗျာ။
