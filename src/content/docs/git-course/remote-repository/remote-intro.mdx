### Remote Repository ဆိုတာဘာလဲ?

Remote Repository ဆိုတာ ကိုယ့်ရဲ့ Project ကို အင်တာနက် သို့မဟုတ် ကွန်ရက်ပေါ်က Server တစ်ခုမှာ သိမ်းထားတာပါ။ အများအားဖြင့် GitHub, GitLab, Bitbucket စတဲ့ ဝဘ်ဆိုဒ်တွေမှာ သိမ်းကြပါတယ်။ ဥပမာ - ကိုယ့်ကွန်ပျူတာထဲက Git Project က ကိုယ်ပိုင်မှတ်စုစာအုပ်လိုဆိုရင်၊ Remote Repository က အဲဒီမှတ်စုကို အွန်လိုင်းပေါ်မှာ သိမ်းထားတာပါ။ ဒီလိုသိမ်းထားတော့ တခြားသူတွေ ဖတ်နိုင်သလို၊ ကိုယ်လည်း ဘယ်နေရာကမဆို ပြန်ယူနိုင်ပါတယ်။

### Remote Repository တွေကို ဘာကြောင့် သုံးရလဲ?

- **`Collaboration` အတူတကွ အလုပ်လုပ်ဖို့**: လူအများ၊ ကွန်ပျူတာ အမျိုးမျိုးကနေ Project တစ်ခုတည်းကို အတူတကွ အလုပ်လုပ်နိုင်ပါတယ်။
- **`Backup` အရန်သိမ်းဆည်းဖို့**: ကိုယ့်ကွန်ပျူတာ ပျက်သွားရင်တောင် Project က လုံခြုံစွာ သိမ်းဆည်းထားပြီး ဖြစ်ပါတယ်။
- **`Accessibility` ဘယ်နေရာကမဆို ရယူနိုင်ဖို့**: အင်တာနက်ရှိတဲ့ ဘယ်နေရာကမဆို ကိုယ့် Code တွေကို ရယူနိုင်ပါတယ်။

### GitHub အကြောင်း

GitHub ဆိုတာ Git Repository တွေကို သိမ်းဆည်းပေးတဲ့ website တစ်ခုပါ။ အဖွဲ့လိုက် အလုပ်လုပ်ဖို့ Pull Request လိုမျိုး Tool တွေလည်း ပါဝင်ပြီး လူအများစု အသုံးပြုကြပါတယ်။

- အကောင့်မရှိသေးရင် https://github.com မှာ အခမဲ့ အကောင့်တစ်ခု ဖွင့်ထားပါ။
