## Remote Repository ဆိုတာ ဘာလဲ?

**Remote Repository** ဆိုတာ ကိုယ့် Project ကို အင်တာနက်ပေါ်မှာ သိမ်းဆည်းထားတာကို ဆိုလိုပါတယ်။ ဥပမာ - GitHub, GitLab, Bitbucket စတဲ့ Website တွေမှာ သိမ်းဆည်းလေ့ ရှိကြပါတယ်။

ကိုယ့် ကွန်ပျူတာထဲက Git Project က ကိုယ်ပိုင် မှတ်စုစာအုပ်နဲ့ တူတယ်ဆိုရင်၊ Remote Repository က အဲဒီ မှတ်စုကို အွန်လိုင်းပေါ်မှာ အလွယ်တကူ ပြန်ကြည့်နိုင်အောင် သိမ်းထားတာနဲ့ တူပါတယ်။

## Remote Repository ရဲ့ အရေးပါပုံ

Remote Repository သုံးခြင်းအားဖြင့် အောက်ပါ အကျိုးကျေးဇူးတွေ ရရှိနိုင်ပါတယ် -

1. **အတူတကွ အလုပ်လုပ်နိုင်ခြင်း**: လူအများ၊ ကွန်ပျူတာ အမျိုးမျိုးကနေ Project တစ်ခုတည်းကို အလွယ်တကူ ပူးပေါင်း လုပ်ဆောင်နိုင်ပါတယ်။
2. **အရန် သိမ်းဆည်းနိုင်ခြင်း (Backup)**: ကိုယ့် ကွန်ပျူတာ ပျက်သွားရင်တောင် Project က အွန်လိုင်းမှာ လုံခြုံစွာ ရှိနေပါတယ်။
3. **ဘယ်နေရာကမဆို ရယူနိုင်ခြင်း**: အင်တာနက် ရှိတဲ့ ဘယ်နေရာကမဆို ကိုယ့် Code တွေကို အလွယ်တကူ ရယူနိုင်ပါတယ်။

## GitHub အကြောင်း

**GitHub** ဆိုတာ Git Repository တွေကို သိမ်းဆည်းပေးတဲ့ နာမည်ကြီး Website တစ်ခု ဖြစ်ပါတယ်။

- အဖွဲ့လိုက် အလုပ်လုပ်ဖို့အတွက် Pull Request လိုမျိုး အသုံးဝင်တဲ့ Tool တွေ ပါဝင်ပါတယ်။
- ကမ္ဘာတစ်ဝှမ်းက Developer အများစု အသုံးပြုကြပါတယ်။
- အခမဲ့ အကောင့်ဖွင့်ပြီး လွယ်ကူစွာ အသုံးပြုနိုင်ပါတယ်။

> အကောင့် မရှိသေးဘူးဆိုရင် [GitHub](https://github.com) မှာ အခမဲ့ အကောင့်တစ်ခု အရင်ဖွင့်ထားပါ ခင်ဗျာ。
