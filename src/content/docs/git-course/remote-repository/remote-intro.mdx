## Remote Repository ဆိုတာဘာလဲ?

**Remote Repository** ဆိုတာ ကိုယ့် Project ကို အင်တာနက်ပေါ်မှာ သိမ်းထားတာပါ။ ဥပမာ GitHub, GitLab, Bitbucket စတဲ့ websites တွေမှာ သိမ်းကြပါတယ်။

ကိုယ့်ကွန်ပျူတာထဲက Git Project က ကိုယ်ပိုင်မှတ်စုစာအုပ်လိုဆိုရင်၊ Remote Repository က အဲဒီမှတ်စုကို အွန်လိုင်းပေါ်မှာ သိမ်းထားတာပါ။

## Remote Repository ရဲ့ အရေးပါပုံ

Remote Repository သုံးခြင်းရဲ့ အဓိက အကျိုးကျေးဇူးများ:

  1. **အတူတကွ အလုပ်လုပ်နိုင်ခြင်း**
      - လူအများ၊ ကွန်ပျူတာ အမျိုးမျိုးကနေ Project တစ်ခုတည်းကို အတူတကွ အလုပ်လုပ်နိုင်
  2. **အရန်သိမ်းဆည်းခြင်း**
      - ကိုယ့်ကွန်ပျူတာ ပျက်သွားရင်တောင် Project က လုံခြုံစွာ သိမ်းဆည်းထားပြီး ဖြစ်
  3. **ဘယ်နေရာကမဆို ရယူနိုင်ခြင်း**
      - အင်တာနက်ရှိတဲ့ ဘယ်နေရာကမဆို ကိုယ့် Code တွေကို ရယူနိုင်

## GitHub အကြောင်း

**GitHub** ဆိုတာ Git Repository တွေကို သိမ်းဆည်းပေးတဲ့ website တစ်ခုပါ။ ၎င်းတွင် -

- အဖွဲ့လိုက် အလုပ်လုပ်ဖို့ Pull Request လိုမျိုး Tool များပါဝင်
- ကမ္ဘာတစ်ဝှမ်းက Developer များ အသုံးပြု
- အခမဲ့ အကောင့်ဖွင့်နိုင်

> အကောင့်မရှိသေးရင် [GitHub](https://github.com) မှာ အခမဲ့ အကောင့်တစ်ခု ဖွင့်ထားပါ။
