ပတ်ဝန်းကျင်သစ်မှာ CLI ကို အသုံးပြုရတာ အစပိုင်းမှာ မှတ်ရခက်နေနိုင်ပါတယ်။ ဒီ Cheat Sheet လေးကို လိုအပ်တဲ့အချိန် ပြန်ကြည့်လို့ရအောင် သိမ်းထားပါ။

### Navigation (သွားလာခြင်း)
| Command | အဓိပ္ပါယ် | ဥပမာ |
| :--- | :--- | :--- |
| `pwd` | လက်ရှိဘယ်နေရာရောက်နေသလဲ စစ်ရန် | `pwd` |
| `ls` | လက်ရှိဖိုင်/ဖိုဒါများကို စာရင်းထုတ်ပြရန် | `ls` |
| `ls -a` | Hidden files များအပါအ၀င် အားလုံးကိုပြရန် | `ls -a` |
| `cd` | Folder အရွှေ့အပြောင်းလုပ်ရန် | `cd Desktop` |
| `cd ..` | နောက်ထပ် တစ်ဆင့်(Folder) အပြင်သို့ထွက်ရန် | `cd ..` |
| `cd ~` | Home Directory သို့ အမြန်သွားရန် | `cd ~` |

---

### File & Folder Management (ဖန်တီး၊ ပြင်ဆင်၊ ဖျက်)
| Command | အဓိပ္ပါယ် | ဥပမာ |
| :--- | :--- | :--- |
| `mkdir` | ဖိုဒါ (Directory) အသစ်ဆောက်ရန် | `mkdir new_folder` |
| `touch` | ဖိုင်အလွတ်အသစ် ဖန်တီးရန် | `touch index.html` |
| `cp` | ဖိုင်ကို ကူးယူရန် (Copy) | `cp file1.txt file2.txt` |
| `cp -r` | ဖိုဒါကို ကူးယူရန် (Copy Folder) | `cp -r folder1 folder2` |
| `mv` | အရွှေ့အပြောင်းလုပ်ရန် သို့မဟုတ် နာမည်ပြောင်းရန် | `mv old_name.txt new_name.txt` |
| `rm` | ဖိုင်ကို အပြီးတိုင် ဖျက်ရန် | `rm unwanted.txt` |
| `rm -r` | ဖိုဒါနှင့်အထဲရှိအရာအားလုံး အပြီးဖျက်ရန် | `rm -r old_folder` |

### Reading, Searching & Filtering (ဖတ်ရှု၊ ရှာဖွေခြင်း)
| Command | အဓိပ္ပါယ် | ဥပမာ |
| :--- | :--- | :--- |
| `cat` | ဖိုင်ပါ အကြောင်းအရာအားလုံးကို ဖတ်ရန် | `cat text.txt` |
| `less` | စာဖိုင်အရှည်များကို တစ်မျက်နှာချင်းဖတ်ရန် | `less long_log.txt` |
| `head` | ဖိုင်၏ ထိပ်ဆုံး စာကြောင်းများကို ဖတ်ရန် | `head -n 5 file.txt` |
| `tail` | ဖိုင်၏ အောက်ဆုံး စာကြောင်းများကို ဖတ်ရန် | `tail -n 5 file.txt` |
| `find` | ဖိုင်နှင့် ဖိုဒါများကို အမည်ဖြင့်ရှာရန် | `find . -name "*.js"` |
| `grep` | ဖိုင်များအတွင်းမှ စာသားကို ရှာရန် | `grep "error" log.txt` |
| `\|` (Pipe) | Command တစ်ခု၏ ရလဒ်ကို နောက်တစ်ခုသို့ ပို့ရန် | `ls \| grep ".md"` |
| `>` သို့ `>>` | ရလဒ်များကို ဖိုင်ထဲသို့ သိမ်းရန် | `echo "hi" > text.txt` |

### System & Others (အခြား အသုံးများသော Command များ)
| Command | အဓိပ္ပါယ် | ဥပမာ |
| :--- | :--- | :--- |
| `clear` | Terminal အမည်းရောင် screen ကို ရှင်းထုတ်ရန် | `clear` |
| `history` | ယခင်က ရိုက်ခဲ့ဖူးသော command များကို ပြန်ကြည့်ရန် | `history` |
| `man` | Command များ၏ Manual လမ်းညွှန်ကို ဖတ်ရန် | `man ls` |
| `whoami` | လက်ရှိ အသုံးပြုနေသူ user အမည်ကို ကြည့်ရန် | `whoami` |
| `top` | ကွန်ပျူတာ၏ CPU/RAM လုပ်ဆောင်ချက်များကိုကြည့်ရန် | `top` |
| `ping` | အင်တာနက်/ကွန်ရက် ချိတ်ဆက်မှု စမ်းသပ်ရန် | `ping google.com` |
| `Ctrl + C` | လက်ရှိ run နေသော အရာများကို ဖြတ်တောက် (cancel) ရန် | နှိပ်ပါ |
