### Java ဆိုတာ ဘာလဲ?

Java ဆိုတာ ၁၉၉၅ ခုနှစ်ကတည်းက ပေါ်ပေါက်ခဲ့ပြီး၊ ဒီနေ့အချိန်ထိ ကမ္ဘာပေါ်မှာ အသုံးအများဆုံး Programming Language တွေထဲက တစ်ခု ဖြစ်ပါတယ်။ သူရဲ့ အဓိက ဆောင်ပုဒ်ကတော့ **"Write Once, Run Anywhere" (တစ်ခါရေးထားရင် ဘယ်နေရာမှာမဆို အလုပ်လုပ်တယ်)** ဆိုတာပါပဲ။

### ဘာလို့ Java ကို လေ့လာသင့်တာလဲ?

- **Platform Independent:** Windows, Mac, Linux ဘယ် OS မှာမဆို အလုပ်လုပ်ပါတယ်။ (JVM - Java Virtual Machine ကြောင့်ပါ)
- **Object-Oriented:** အရာအားလုံးကို Object အနေနဲ့ မြင်ပြီး ရေးရတဲ့အတွက် Code တွေက စနစ်ကျပြီး ပြန်သုံးရ လွယ်ကူပါတယ်။
- **လုံခြုံရေး (Security):** ဘဏ်တွေ၊ ငွေကြေးဆိုင်ရာ Application တွေမှာ Java ကို အဓိက သုံးကြပါတယ်။
- **အလုပ်အကိုင် အခွင့်အလမ်း:** ကုမ္ပဏီကြီးတွေ (Enterprise) တော်တော်များများက သူတို့ရဲ့ Backend System တွေအတွက် Java (Spring Boot) ကို သုံးကြတဲ့အတွက် အလုပ်အကိုင် ပေါများပါတယ်။

### Python နဲ့ ဘာကွာလဲ?

Python က ရေးရတာ လွယ်ကူပြီး တိုတောင်းပေမယ့်၊ Java ကတော့ စည်းကမ်းကြီးပါတယ်။ Variable တစ်ခု ကြေညာရင်တောင် ဘာအမျိုးအစားလဲ ဆိုတာ အတိအကျ ပြောပေးရပါတယ်။ (Strongly Typed လို့ ခေါ်ပါတယ်)။ စည်းကမ်းကြီးတဲ့အတွက် အမှား (Error) တွေကို Code ရေးနေတုန်းမှာတင် ကြိုသိနိုင်ပါတယ်။

### ဒီ Course မှာ ဘာတွေ သင်ရမလဲ?

1. **Syntax & Data Types:** Java ရဲ့ အခြေခံ ရေးသားပုံများ။
2. **Control Flow:** If/Else နဲ့ Loops များ။
3. **Methods (Functions):** Code တွေကို ခွဲခြားရေးသားခြင်း။
4. **Object-Oriented Programming (OOP):** Java ရဲ့ အသက်သွေးကြောဖြစ်တဲ့ Classes, Objects, Inheritance, Polymorphism စတာတွေ။
5. **Collections Framework:** Data တွေကို စနစ်တကျ သိမ်းဆည်းခြင်း (Lists, Maps, Sets)။
6. **Exception Handling:** Error တွေကို ထိန်းချုပ်ခြင်း။

ကဲ... Java လောကထဲကို ဝင်ရောက်ကြည့်ကြရအောင်!
