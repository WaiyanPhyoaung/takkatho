<Aside title="Tooltip များ ပိုမို လွယ်ကူလာခြင်း (Tooltips Made Easy)">
အရင်တုန်းက Tooltip တွေ၊ Dropdown Menu တွေကို သက်ဆိုင်ရာ Button နားမှာ သေသေချာချာ ကပ်ထားဖို့ဆိုရင် Popper.js သို့မဟုတ် Floating UI လိုမျိုး JavaScript Library တွေ မဖြစ်မနေ သုံးခဲ့ရပါတယ်။ အခုအခါမှာတော့ CSS အသစ်နဲ့တင် အလွယ်တကူ လုပ်လို့ ရသွားပါပြီ။
</Aside>

### အခြေခံ သဘောတရား
အဓိက အစိတ်အပိုင်း (၂) ခု ပါဝင်ပါတယ်။
1.  **Anchor Element:** မှီခိုအားထားရမယ့် တိုင် (ဥပမာ - Button)။ သူ့ကို `anchor-name` နာမည်တစ်ခု ပေးထားရပါမယ်။
2.  **Positioned Element:** သွားကပ်မယ့် ကောင် (ဥပမာ - Tooltip)။ သူကနေပြီး `position-anchor` နဲ့ ဟိုဘက်က နာမည်ကို လှမ်းချိတ်ရပါမယ်။

### Code ရေးသားပုံ ဥပမာ

```css
/* ၁။ Anchor (မှီခိုရမည့် Button) */
.trigger-btn {
  anchor-name: --my-anchor; /* နာမည်တစ်ခု ပေးလိုက်တယ် */
}

/* ၂။ Positioned Element (သွားကပ်မည့် Tooltip) */
.tooltip {
  position: absolute;
  position-anchor: --my-anchor; /* Button ရဲ့ နာမည်ကို လှမ်းချိတ်လိုက်တယ် */
  
  /* Button ရဲ့ အပေါ် (top) မှာ ကပ်မယ် */
  bottom: anchor(top);
  
  /* Button ရဲ့ အလယ် (center) နဲ့ အချိုးကျ ညှိမယ် */
  left: anchor(center);
  transform: translateX(-50%);
}
```

### Fallback (နေရာမလောက်ရင် ဘယ်လိုလုပ်မလဲ?)
Screen ရဲ့ အဆုံးကို ရောက်နေလို့ Tooltip ပြစရာ နေရာမရှိတော့ဘူးဆိုရင်၊ အလိုအလျောက် အောက်ဘက်ကို ပြောင်းပြဖို့ (နေရာရွှေ့ဖို့) `@position-try` ကို အသုံးပြုနိုင်ပါတယ်။

```css
.tooltip {
  /* ပုံမှန်ဆိုရင် အပေါ်မှာ ပြမယ်၊ နေရာမလောက်ရင် အောက်ဘက်ကို လှန်ချပြီး (flip-block) ပြမယ် */
  position-try-fallbacks: flip-block;
}
```

<Aside type="tip">
HTML Structure (DOM) အရ Button နဲ့ Tooltip က အဝေးကြီး ဖြစ်နေရင်တောင်မှ (ဥပမာ - Tooltip က `<body>` ရဲ့ အောက်ဆုံးမှာ ရှိနေရင်တောင်) ဒီနည်းလမ်းနဲ့ CSS ကနေ တိုက်ရိုက် လှမ်းချိတ်လို့ ရပါတယ်။
</Aside>
