<Aside title="လုံးဝန်းခြင်း ထက်ပိုသော ပုံစံများ (Beyond Rounded Corners)">
ပုံမှန်သုံးနေကျ `border-radius` က စက်ဝိုင်းခြမ်း (Circle) ပုံစံပဲ ရပါတယ်။ iOS App Icon တွေလိုမျိုး မျက်စိပသာဒဖြစ်ပြီး ပြေပြစ်တဲ့ "Squircle" (Superellipse) ပုံစံမျိုး လိုချင်ရင်တော့ `corner-shape` ဆိုတဲ့ Property အသစ်က ကူညီပေးပါလိမ့်မယ်။
</Aside>

### အသုံးပြုပုံ

```css
.card {
  /* ထောင့်ချိုး ဘယ်လောက်ကွေးမလဲ ဆိုတဲ့ အရွယ်အစား */
  border-radius: 20px;
  
  /* ထောင့်ချိုးကို ဘယ်လိုပုံစံမျိုး ကွေးချင်တာလဲ */
  corner-shape: super-ellipse; /* သို့မဟုတ် squircle လို့ သတ်မှတ်လို့ ရပါတယ် */
}
```

### ရရှိနိုင်သော ပုံစံများ (Available Shapes)
*   `round`: ဒါကတော့ ပုံမှန် `border-radius` ရဲ့ စက်ဝိုင်းခြမ်း ပုံစံပါ။
*   `angle`: ကွေးမသွားဘဲ ထောင့်ချိုး အတိအကျ ဖြတ်ချလိုက်တဲ့ ပုံစံ (Diamond shape ဆန်ဆန် ဖြစ်သွားပါမယ်)။
*   `super-ellipse`: Apple ရဲ့ App Icon တွေလိုမျိုး အရမ်းကို ပြေပြစ်တဲ့ ထောင့်ချိုးပုံစံ။
