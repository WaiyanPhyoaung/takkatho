Database နဲ့ ပတ်သက်တဲ့ နယ်ပယ်ကို ဝင်ရောက်လာတဲ့အခါ၊ အရင်ဆုံး မေးလေ့ရှိတဲ့ မေးခွန်းကတော့ "SQL ဆိုတာ ဘာလဲ?" ဆိုတာပါပဲ။

ဒီသင်ခန်းစာမှာ SQL ရဲ့ အဓိပ္ပာယ်၊ သူဘယ်လို အလုပ်လုပ်တယ်ဆိုတာကို အရိုးရှင်းဆုံး ရှင်းပြပေးသွားမှာ ဖြစ်ပါတယ်။

## SQL ဆိုတာ...

**SQL** ကို အရှည်ကောက်အားဖြင့် **Structured Query Language** လို့ ခေါ်ပါတယ်။

သူက "Database (အချက်အလက်သိုလှောင်ရုံ) စနစ်များနဲ့ စကားပြောဆိုဆက်သွယ်ဖို့ သုံးတဲ့ စံသတ်မှတ်ထားတဲ့ ဘာသာစကားတစ်ခု" ဖြစ်ပါတယ်။ Programming တွေလို ရှုပ်ထွေးတဲ့ Logic တွေ ရေးစရာမလိုဘဲ လူတွေနားလည်နိုင်တဲ့ အင်္ဂလိပ်စာလုံးတွေ (ဥပမာ - `SELECT`, `INSERT`, `DELETE`) နဲ့ Data တွေကို ခိုင်းစေလို့ ရပါတယ်။

## ဘာလို့ SQL ကို သုံးရတာလဲ?

1.  **လွယ်ကူခြင်း:** စကားပြောသလို ရေးရတာဖြစ်လို့ သင်ယူရတာ အရမ်း မြန်ဆန်ပါတယ်။
2.  **စံနှုန်းတစ်ခုတည်းဖြစ်ခြင်း (Standard):** MySQL, PostgreSQL, SQL Server စတဲ့ နာမည်ကြီး Database တွေ အားလုံးက ဒီ SQL ဘာသာစကား တစ်ခုတည်းကိုပဲ အခြေခံပြီး နားလည်ကြပါတယ်။ ခပ်ဆင်ဆင်ပဲမို့ တစ်ခုရရင် ကျန်တာတွေပါ သုံးတတ်သွားပါလိမ့်မယ်။
3.  **မြန်ဆန်လှပခြင်း:** အချက်အလက် (Data) သန်းချီ ရှိနေရင်တောင် လိုချင်တဲ့ အချက်အလက် ကို စက္ကန့်ပိုင်းအတွင်း ရှာဖွေပေးနိုင်ပါတယ်။

## Relational Database ဆိုတာ ဘာလဲ?

SQL က Database သက်သက်ကို စီမံတာတော့ ဟုတ်ပါပြီ။ ဒါပေမယ့် သူ့ကို အများအားဖြင့် **RDBMS (Relational Database Management System)** တွေမှာ သုံးပါတယ်။

ရိုးရိုးရှင်းရှင်း ပြောရရင် Excel ဖိုင်လို **ဇယား (Table)** ပုံစံနဲ့ သိမ်းပြီး၊ အဲဒီ ဇယားတွေကို တစ်ခုနဲ့တစ်ခု ဆက်စပ်ပြီး (ဥပမာ - User ဇယား နဲ့ သူဝယ်ထားတဲ့ Order ဇယား ကို ချိတ်ဆက်တာမျိုး) အလုပ်လုပ်တဲ့ စနစ်ကို ခေါ်တာပါ။

ဒီလောက်ဆို အခြေခံ သဘောတရားကို ရလောက်ပြီလို့ ထင်ပါတယ်။ နောက်သင်ခန်းစာမှာ လက်တွေ့ စတင်ပြီး Query လေးတွေ ရေးကြည့်ကြပါမယ်!
