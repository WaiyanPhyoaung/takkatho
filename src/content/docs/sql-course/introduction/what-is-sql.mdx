Database နဲ့ ပတ်သက်တဲ့ နယ်ပယ်ကို စတင် ဝင်ရောက်လာပြီ ဆိုရင်၊ အရင်ဆုံး မေးလေ့ရှိတဲ့ မေးခွန်းကတော့ "SQL ဆိုတာ ဘာလဲ?" ဆိုတာပါပဲ။

ဒီသင်ခန်းစာမှာ SQL ရဲ့ အဓိပ္ပာယ်၊ သူ ဘယ်လို အလုပ်လုပ်တယ် ဆိုတာကို အရိုးရှင်းဆုံး ရှင်းပြပေးသွားမှာ ဖြစ်ပါတယ် ခင်ဗျာ။

## SQL ဆိုတာ ဘာလဲ?

**SQL** ရဲ့ အရှည်ကောက်ကတော့ **Structured Query Language** ဖြစ်ပါတယ်။

သူက "Database (အချက်အလက် သိုလှောင်ရုံ) တွေနဲ့ စကားပြောဆို ဆက်သွယ်ဖို့ သုံးတဲ့ စံသတ်မှတ်ထားတဲ့ ဘာသာစကား တစ်ခု" ဖြစ်ပါတယ်။ Programming တွေလို ရှုပ်ထွေးတဲ့ Logic တွေ ရေးစရာ မလိုဘဲ၊ လူတွေ နားလည်နိုင်တဲ့ ရိုးရှင်းတဲ့ အင်္ဂလိပ် စကားလုံးတွေ (ဥပမာ - `SELECT`, `INSERT`, `DELETE`) သုံးပြီး Data တွေကို လိုသလို ခိုင်းစေလို့ ရပါတယ်။

## ဘာလို့ SQL ကို သုံးရတာလဲ?

1. **လွယ်ကူခြင်း:** စကားပြောသလို ရေးရတာ ဖြစ်လို့ သင်ယူရတာ အရမ်း မြန်ဆန်ပါတယ်။
2. **စံနှုန်းတစ်ခုတည်း ဖြစ်ခြင်း (Standard):** MySQL, PostgreSQL, SQL Server စတဲ့ နာမည်ကြီး Database တွေ အားလုံးက ဒီ SQL ဘာသာစကား တစ်ခုတည်းကိုပဲ အခြေခံပြီး နားလည်ကြပါတယ်။ ဒါကြောင့် တစ်ခုကို သေချာ ရထားရင် ကျန်တာတွေပါ အလွယ်တကူ သုံးတတ်သွားပါလိမ့်မယ်။
3. **မြန်ဆန်ခြင်း:** Data (အချက်အလက်) တွေ သန်းချီ ရှိနေရင်တောင် ကိုယ်လိုချင်တဲ့ Data ကို စက္ကန့်ပိုင်းအတွင်း မြန်မြန်ဆန်ဆန် ရှာဖွေပေးနိုင်ပါတယ်။

## Relational Database ဆိုတာ ဘာလဲ?

SQL က Database တွေကို စီမံတာတော့ ဟုတ်ပါပြီ။ ဒါပေမဲ့ သူ့ကို အများအားဖြင့် **RDBMS (Relational Database Management System)** တွေမှာ သုံးလေ့ရှိပါတယ်။

ရိုးရိုးရှင်းရှင်း ပြောရရင် Excel ဖိုင်လိုပဲ **Table (ဇယား)** ပုံစံနဲ့ သိမ်းတယ်။ ပြီးရင် အဲ့ဒီ Table တွေကို တစ်ခုနဲ့တစ်ခု ဆက်စပ်ပြီး (ဥပမာ - User Table နဲ့ သူ ဝယ်ထားတဲ့ Order Table ကို ချိတ်ဆက်တာမျိုး) အလုပ်လုပ်တဲ့ စနစ်ကို ခေါ်တာပါ။

ဒီလောက်ဆို အခြေခံ သဘောတရားကို ရလောက်ပြီလို့ ထင်ပါတယ်။ နောက်သင်ခန်းစာမှာတော့ လက်တွေ့ စတင်ပြီး Query တွေ ရေးကြည့်ကြပါမယ် ခင်ဗျာ!
