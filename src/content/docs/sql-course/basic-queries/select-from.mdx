SQL သင်ယူတဲ့အခါ ပထမဆုံး သိထားသင့်တာကတော့ Data (ဒေတာ) တွေကို ဘယ်လို ပြန်ထုတ်ကြည့်မလဲ ဆိုတာပဲ ဖြစ်ပါတယ်။ ဒါကို **Query (ကွာရီ)** ရေးတယ်လို့ ခေါ်လေ့ရှိပါတယ်။

အသုံးအများဆုံး စကားလုံးကတော့ `SELECT` ဖြစ်ပါတယ်။ `SELECT` ရဲ့ အဓိပ္ပာယ်က "ရွေးချယ်ပါ" လို့ ဆိုလိုတာပါ။

## အခြေခံအကျဆုံး `SELECT ... FROM`

စကားပြောသလိုပါပဲ၊ "ဘာတွေကို ... ဘယ်နေရာကနေ ရွေးထုတ်မလဲ" ဆိုတဲ့ ပုံစံပါ။ `FROM` ဆိုတာက "ဘယ် Table (ဇယား) ကနေလဲ" လို့ ညွှန်ပြတာ ဖြစ်ပါတယ်။

ဥပမာ - `users` ဆိုတဲ့ Table ထဲမှာရှိတဲ့ **အချက်အလက် အကုန်လုံး** ကို ပြချင်တယ် ဆိုပါစို့။

```sql
SELECT * FROM users;
```

**ရှင်းလင်းချက်:**
- `SELECT *` : `*` (ခရေပွင့်) လေးက "အရာအားလုံး၊ အကုန်လုံး" လို့ ဆိုလိုပါတယ်။ Table ထဲက Column (ခေါင်းစဉ်) အကုန်လုံးကို ပြပေးမှာပါ။
- `FROM users` : `users` ဆိုတဲ့ နာမည်နဲ့ Table ကနေ ယူမှာ ဖြစ်ပါတယ်။
- `;` (Semicolon) : SQL ဝါကျ တစ်ကြောင်း ပြီးဆုံးတိုင်း အမြဲတမ်း `;` လေးနဲ့ ပိတ်ပေးလေ့ ရှိပါတယ်။

## လိုချင်တဲ့ Column ကိုပဲ ရွေးချယ်ခြင်း

တကယ့် Database အကြီးစားတွေမှာ `*` ကို သုံးလိုက်ရင် Data တွေ အများကြီး တစ်ခါတည်း ထွက်လာပြီး စနစ်ကို အရမ်း နှေးသွားစေနိုင်ပါတယ်။ ဒါကြောင့် ကိုယ်လိုချင်တဲ့ အချက်အလက် (ဥပမာ - နာမည်နဲ့ အီးမေးလ်) သီးသန့်ကိုပဲ ဆွဲထုတ်တာ အကောင်းဆုံး အလေ့အကျင့်ပါ။

```sql
SELECT name, email FROM users;
```

*(အဓိပ္ပာယ်: `users` Table ထဲကနေ `name` Column ရယ်၊ `email` Column ရယ်... အဲ့ဒီ ၂ ခုထဲက Data တွေကိုပဲ ထုတ်ပြပေးပါလို့ ခိုင်းတာပါ)*။

ဒီအဆင့်အထိဆိုရင် `SELECT` နဲ့ ဒေတာတွေ ဘယ်လို ရွေးထုတ်လဲ ဆိုတာ သဘောပေါက်ပြီလို့ ထင်ပါတယ်။ နောက်သင်ခန်းစာမှာတော့ ကိုယ်လိုချင်တဲ့ အခြေအနေနဲ့ ကိုက်ညီတဲ့ Data တွေကို ဘယ်လို စစ်ထုတ် (Filter) မလဲဆိုတာ ဆက်လေ့လာကြပါမယ် ခင်ဗျာ။
