---

`SELECT` နဲ့ Data တွေ အားလုံးကို ဆွဲထုတ်ယူလို့ ရတာ မှန်ပေမဲ့၊ လက်တွေ့ လုပ်ငန်းခွင်မှာတော့ ကိုယ်ရှာချင်တဲ့ Data တွေ၊ သတ်မှတ်ထားတဲ့ စည်းမျဉ်းနဲ့ ကိုက်ညီတဲ့ Data တွေကိုပဲ သီးသန့် စစ်ထုတ် (Filter) ပြီး ယူလေ့ရှိပါတယ်။ အဲ့ဒီလို အခြေအနေမျိုးမှာ **`WHERE`** ကို သုံးရပါတယ်။

## အခြေခံ `WHERE` ရေးနည်း

`WHERE` ဆိုတာ "ဘယ်လို အခြေအနေ (Condition) နဲ့ ကိုက်ညီတဲ့ အခါမှာ" ဆိုတာမျိုး ခိုင်းစေတာပါ။

ဥပမာ - `users` Table ထဲကနေ `age` (အသက်) ၂၀ ထက် ကြီးတဲ့ User တွေကိုပဲ ရွေးပြီး ရှာပြချင်တယ် ဆိုပါစို့။

```sql
SELECT * FROM users
WHERE age > 20;
```
*(အဓိပ္ပာယ်: `users` Table ထဲကနေ အသက် ၂၀ ထက်ကျော်တဲ့ လူတွေရဲ့ အချက်အလက် အားလုံး `*` ကို ထုတ်ပြပေးပါ)*။

## အသုံးများတဲ့ Operators တွေက ဘာတွေလဲ?

`WHERE` နဲ့ တွဲသုံးလေ့ရှိတဲ့ အခြေခံ သင်္ချာ သင်္ကေတလေးတွေ (Operators) ဖြစ်ပါတယ်။

- `=` : ညီမျှတယ် (ဥပမာ - `WHERE city = 'Yangon'`)
- `>` : ပိုကြီးတယ် / ပိုများတယ် (ဥပမာ - `WHERE price > 100`)
- `<` : ပိုငယ်တယ် / ပိုနည်းတယ် (ဥပမာ - `WHERE price < 50`)
- `>=` : ကြီးတယ် (သို့) ညီတယ်
- `<=` : ငယ်တယ် (သို့) ညီတယ်
- `<>` (သို့) `!=` : မညီဘူး / မတူဘူး (ဥပမာ - `WHERE city != 'Mandalay'`)

## `AND`, `OR`, `NOT` အသုံးပြုနည်း

အခြေအနေ (Condition) တစ်ခုထက်မက ပေါင်းစပ်ပြီး ရှာချင်တဲ့ အခါမျိုးမှာ သုံးပါတယ်။

### `AND` (နှင့်)

သတ်မှတ်ထားတဲ့ အခြေအနေ **အားလုံး** မှန်ကန်မှ (ပြည့်စုံမှ) Data ကို ထုတ်ပေးပါမယ်။

```sql
SELECT * FROM users
WHERE age > 18 AND city = 'Yangon';
```
*(အဓိပ္ပာယ်: အသက်လည်း ၁၈ ထက် ကြီးရမယ်၊ နေတဲ့ မြို့ကလည်း Yangon ဖြစ်ရမယ်။ နှစ်ခုလုံး မှန်မှ ပြပေးမယ်)*။

### `OR` (သို့မဟုတ်)

သတ်မှတ်ထားတဲ့ အခြေအနေတွေထဲက **တစ်ခုခု** မှန်သွားရင် (ပြည့်စုံရင်) ရပါပြီ။

```sql
SELECT * FROM users
WHERE city = 'Yangon' OR city = 'Mandalay';
```
*(အဓိပ္ပာယ်: ရန်ကုန်က User ဖြစ်ဖြစ်၊ မန္တလေးက User ဖြစ်ဖြစ် အကုန် ထုတ်ပြပေးမယ်)*။

### `NOT` (မဟုတ်သော)

မိမိ မလိုချင်တဲ့၊ ဖယ်ထုတ်ချင်တဲ့ အခါမျိုးမှာ သုံးပါတယ်။

```sql
SELECT * FROM users
WHERE NOT city = 'Yangon';
```
*(အဓိပ္ပာယ်: ရန်ကုန်မြို့က လွဲပြီး တခြားမြို့က လူတွေ အကုန်လုံးကို ထုတ်ပြပေးမယ်)*။

ဒီလောက်ဆိုရင်တော့ `WHERE` ကို သုံးပြီး Data တွေကို ကိုယ်လိုချင်သလို သေချာ စစ်ထုတ် (Filter) တတ်ပြီလို့ ယုံကြည်ပါတယ်။ နောက်သင်ခန်းစာမှာတော့ ရလာတဲ့ Data တွေကို သေသပ်အောင် အစဉ်လိုက် စီစဉ်တဲ့ အပိုင်းကို ဆက်သွားပါမယ် ခင်ဗျာ။
