Data (ဒေတာ) တွေကို ရှာဖွေ ပြန်ထုတ်ကြည့်တာ (`SELECT`) ပြီးရင်၊ နောက်ထပ် အရေးကြီးဆုံး လုပ်ငန်းစဉ် (၃) ခုကတော့ Data အသစ် ထည့်တာ (Create)၊ ရှိပြီးသားကို ပြင်တာ (Update) နဲ့ မလိုတာကို ဖျက်တာ (Delete) ပဲ ဖြစ်ပါတယ်။ ဒါတွေကို `SELECT` (Read) နဲ့ စုပေါင်းပြီးတော့ **CRUD (Create, Read, Update, Delete)** Operations တွေလို့ ခေါ်လေ့ရှိပါတယ်။

## Data အသစ် ထည့်ခြင်း (`INSERT`)

Table (ဇယား) တစ်ခုခုထဲကို Data အသစ်တွေ ထည့်သွင်းချင်ရင် `INSERT INTO` ကို အသုံးပြုပါတယ်။

**အခြေခံ ရေးသားပုံ:** 
```sql
INSERT INTO ဇယားအမည် (Column နာမည်များ) 
VALUES (ထည့်မည့် တန်ဖိုးများ);
```

ဥပမာ - `users` Table ထဲကို User သစ် တစ်ယောက်ရဲ့ အချက်အလက် ထည့်မယ် ဆိုပါစို့။

```sql
INSERT INTO users (name, email, age)
VALUES ('Aung Aung', 'aung@gmail.com', 25);
```
*(အဓိပ္ပာယ်: `users` Table ရဲ့ `name`, `email`, `age` ဆိုတဲ့ Column နေရာတွေမှာ အောက်က Values ၃ ခုကို အစဉ်လိုက် ကိုက်ညီအောင် ထည့်ပေးပါ)*။

## Data များကို ပြင်ဆင်ခြင်း (`UPDATE`)

ရှိပြီးသား Data တစ်ခုခုကို အပြောင်းအလဲ လုပ်ချင်တဲ့အခါ `UPDATE` နဲ့ `SET` ကို တွဲပြီး သုံးပါတယ်။

> [!WARNING]
> `UPDATE` လုပ်တဲ့အခါ ဘယ် Data (ဘယ်သူ့ကို) ပြင်မှာလဲ ဆိုတာ တိတိကျကျ သတ်မှတ်ပေးဖို့ **`WHERE` ကို အမြဲတမ်း တွဲသုံးဖို့** လိုပါတယ်။ `WHERE` မပါရင် Table ထဲမှာ ရှိသမျှ Data တွေ အကုန်လုံး အပြောင်းအလဲ ဖြစ်သွားပါလိမ့်မယ်။

ဥပမာ - ID `1` ရှိတဲ့ User ရဲ့ ဖုန်းနံပါတ်ကို ပြောင်းချင်တယ် ဆိုပါစို့။

```sql
UPDATE users
SET phone = '0912345678'
WHERE id = 1;
```
*(အဓိပ္ပာယ်: `users` Table ထဲက `id` နံပါတ် 1 ရှိတဲ့သူကို ရှာပြီး၊ သူ့ရဲ့ ဖုန်းနံပါတ် နေရာမှာ `09...` လို့ ပြောင်းလဲ သတ်မှတ်ပေးပါ)*။

## Data များ ဖျက်ပစ်ခြင်း (`DELETE`)

မလိုအပ်တော့တဲ့ Data တွေကို Database ထဲကနေ အပြီး ဖယ်ရှားချင်ရင် `DELETE FROM` ကို သုံးပါတယ်။

> [!CAUTION]
> `DELETE` လုပ်တဲ့ အခါမှာလည်း ဘယ် Data ကို ဖျက်မှာလဲ ဆိုတာ သေချာအောင် **`WHERE` ကို အမြဲတမ်း ထည့်သုံးပေးပါ**။ မသုံးမိရင် Table ထဲက Data တွေ အကုန်လုံး ပျက်သွားနိုင်ပါတယ်။

ဥပမာ - အီးမေးလ် `test@gmail.com` ရှိတဲ့ User ရဲ့ အချက်အလက်တွေကို ဖျက်မယ် ဆိုပါစို့။

```sql
DELETE FROM users
WHERE email = 'test@gmail.com';
```
*(အဓိပ္ပာယ်: `users` Table ထဲမှာ `email` က `test@gmail.com` ဖြစ်နေတဲ့ သူရဲ့ မှတ်တမ်းကို ရှာပြီး ဖျက်ပစ်ပါ)*။

ဒီ `INSERT`, `UPDATE`, `DELETE` သုံးခုက တကယ့် Application တွေ၊ System တွေ ရေးတဲ့အခါ နေ့စဉ်နဲ့အမျှ အများဆုံး အသုံးပြုရမယ့် လုပ်ဆောင်ချက်တွေပဲ ဖြစ်ပါတယ် ခင်ဗျာ။
