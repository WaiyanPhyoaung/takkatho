### ဒီ Course က ဘာအကြောင်းလဲ?
Programmer တစ်ယောက်အနေနဲ့ Code ရေးတတ်ရုံနဲ့ မလုံလောက်ပါဘူး။
- "ငါ့ Code က ဘာလို့ နှေးနေတာလဲ?"
- "Data တွေ အများကြီးကို ဘယ်လို စနစ်တကျ သိမ်းမလဲ?"
- "ပြဿနာတစ်ခုကို အကောင်းဆုံး နည်းလမ်းနဲ့ ဘယ်လို ဖြေရှင်းမလဲ?"
ဆိုတာတွေကို နားလည်ဖို့ DSA လိုအပ်ပါတယ်။

ကျွန်တော်တို့ရဲ့ ရည်မှန်းချက်က **Brute Force (ဇွတ်လုပ်နည်း)** ကနေ **Optimized (ပညာသားပါပါ လုပ်နည်း)** ဘက်ကို ပြောင်းလဲ တွေးခေါ်တတ်စေဖို့ ဖြစ်ပါတယ်။

### ဘာတွေ လေ့လာရမလဲ?

**Module 1: The "Why" & The "How" (Foundations)**
Code တွေရဲ့ မြန်နှုန်း (Speed) နဲ့ နေရာယူမှု (Memory) ကို တိုင်းတာတဲ့ Big O အကြောင်းနဲ့ Memory အလုပ်လုပ်ပုံကို လေ့လာမယ်။

**Module 2: Linear Data Structures (The Basics)**
Array, String နဲ့ Linked List တွေအကြောင်း။ Todo List တွေ၊ Playlist တွေ ဘယ်လိုအလုပ်လုပ်လဲ ဆိုတာ လက်တွေ့မြင်ရမယ်။

**Module 3: The "LIFO" & "FIFO" World**
Stack (Undo/Redo), Queue (Printer jobs) နဲ့ အမြန်ဆုံး ရှာဖွေနိုင်တဲ့ Hash Map အကြောင်း။

**Module 4: The Art of Recursion & Sorting**
Recursion ဆိုတဲ့ သဘောတရားနဲ့ Sorting Algorithms (Merge Sort, Quick Sort) တွေကို လေ့လာမယ်။

**Module 5: Trees & Graphs**
File System တွေ၊ Google Map လမ်းကြောင်းရှာတာတွေရဲ့ နောက်ကွယ်က Tree နဲ့ Graph တွေအကြောင်း။

**Module 6: Advanced Problem Solving**
Dynamic Programming၊ Greedy Algorithms နဲ့ Backtracking လိုမျိုး အဆင့်မြင့် နည်းလမ်းတွေနဲ့ Sudoku ဖြေရှင်းနည်းတွေ။
