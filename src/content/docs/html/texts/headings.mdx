### ခေါင်းစဉ်များ (`<h1>` to `<h6>`)
ခေါင်းစဉ်တွေ (**Headings**) ကို webpage မှာ ခေါင်းစဉ်ကြီးတွေ ဒါမှမဟုတ် အရေးကြီးတဲ့ အကြောင်းအရာတွေအတွက် သုံးပါတယ်။ သူတို့က Page ကို ပုံစံ စနစ်တကျဖြစ်စေပြီး အသုံးပြုသူတွေအတွက် ဖတ်ရလွယ်ကူအောင် ကူညီပေးပါတယ်။

HTML မှာ ခေါင်းစဉ်တွေက `<h1>` ကနေ `<h6>` အထိ ရှိပါတယ်။ ပုံမှန်အားဖြင့်တော့ `<h1>` က အကြီးဆုံးဖြစ်ပြီး `<h6>` ဘက်ကိုသွားလေလေ စာလုံးအရွယ်အစားက ပိုသေးသွားလေလေပါပဲ။ ဒါပေမဲ့ ဒီအရွယ်အစားကို CSS သုံးပြီး ထိန်းချုပ်လို့ရပါတယ်။

- `<h1>` က အဓိကခေါင်းစဉ်ကြီးဖြစ်ပြီး၊ များသောအားဖြင့် စာမျက်နှာရဲ့ ခေါင်းစဉ်အတွက် သုံးလေ့ရှိတယ်။

- `<h2>` to `<h6>` ကိုတော့ ခေါင်းစဉ်ငယ်တွေအတွက် သုံးပါတယ်။ `<h2>` က အဓိကခေါင်းစဉ်ငယ်ဖြစ်ပြီး၊ `<h6>` ကတော့ သိပ်အရေးမကြီးတဲ့ ခေါင်းစဉ်ငယ်ပါ။

ဥပမာ:

<Code title="HTML" lang="html" code={headings} />

> **အကြံပြုချက်**: အကြောင်းအရာတွေကို ရှင်းလင်းပြီး တစုတစည်းတည်း ဖြစ်အောင် ခေါင်းစဉ်တွေကို သုံးပါ။ ဒါက အသုံးပြုသူတွေရော၊ search engine တွေအတွက်ပါ သင့် webpage ကို နားလည်ရလွယ်ကူစေပါတယ်။

### စာပိုဒ်များ (`<p>`)

HTML မှာ စာပိုဒ် (paragraph) တစ်ခုကို `<p>` tag သုံးပြီး ဖန်တီးပါတယ်။ စာသားတွေကို ဖတ်လို့ကောင်းတဲ့ ပမာဏလောက်ပဲ ခွဲထုတ်ဖို့ စာပိုဒ်တွေကို သုံးပါတယ်။ စာပိုဒ်အသစ်တစ်ခု စချင်တိုင်း `<p>` tag ကို သုံးရပါတယ်။

ဥပမာ:

<Code title="HTML" lang='html' code={paragraph} />

> အကြံပြုချက်: စာသားအရှည်ကြီးတွေကို စာပိုဒ်တွေမခွဲဘဲ တစ်တုံးတစ်ခဲတည်း မထည့်ပါနဲ့။ အဲဒါက သင့် webpage ကို ဖတ်ရခက်စေပါတယ်။
