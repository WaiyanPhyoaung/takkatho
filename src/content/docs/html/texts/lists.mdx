<Aside title="ရည်ရွယ်ချက်">
ဒီအခန်းမှာတော့ HTML မှာ list အမျိုးမျိုး ဘယ်လိုဖန်တီးရမလဲဆိုတာ လေ့လာကြပါမယ်။ Bullet point list တွေ၊ နံပါတ်စဉ် list တွေ၊ list ထဲမှာ list ထပ်ထည့်တာ (nested lists) နဲ့ description list (`<dl>`) တွေအကြောင်းကို သင်ယူရမှာပါ။
</Aside>

HTML မှာ list အမျိုးအစား နှစ်မျိုးဖန်တီးနိုင်ပါတယ် - **unordered lists (bullet point list)** နဲ့ **ordered lists (number list)** ပါ။

### ဖော်ပြချက် List များ (Description Lists)
Description list (`<dl>`) ဆိုတာက အသုံးအနှုန်းတွေ (`<dt>`) နဲ့ သူ့ရဲ့ ဖော်ပြချက်တွေ (`<dd>`) ကို တွဲဖော်ပြဖို့ သုံးပါတယ်။ ဝေါဟာရရှင်းလင်းချက်တွေ၊ FAQ တွေအတွက် အသင့်တော်ဆုံးပါ။

- `<dl>` — description list container
- `<dt>` — term/item (Definition Term)
- `<dd>` — ဖော်ပြချက် (Definition Description)

**ဥပမာ**

<Aside type='tip' title='အကျဉ်းချုပ်'>

- Headings (`<h1>` to `<h6>`) တွေက အကြောင်းအရာကို အဆင့်ဆင့် စနစ်တကျဖြစ်အောင် ကူညီပေးပါတယ်။

- Paragraphs (`<p>`) တွေက စာသားတွေကို ဖတ်လို့ကောင်းတဲ့ အပိုင်းလေးတွေအဖြစ် ပုံစံချပေးပါတယ်။

- Line breaks (`<br>`) က စာကြောင်းအသစ်ဆင်းစေပြီး၊ horizontal rules (`<hr>`) က အပိုင်းတွေကို ခွဲခြားပေးပါတယ်။

- `<b>`, `<i>`, `<u>`, `<strong>`, `<em>`, and `<mark>` လိုမျိုး tag တွေသုံးပြီး စာသားကို ပုံစံပြောင်းနိုင်ပါတယ်။

- Lists တွေကို `<ul>` (bullet list) နဲ့ `<ol>` (နံပါတ်စဉ် list) သုံးပြီး ဖန်တီးနိုင်ပါတယ်၊ item တစ်ခုချင်းစီကိုတော့ `<li>` tag ထဲမှာ ထည့်ရပါတယ်။

- **Nested list** တွေကို `<li>` ထဲမှာ `<ul>` ဒါမှမဟုတ် `<ol>` ထပ်ထည့်ပြီး ဖန်တီးနိုင်ပါတယ်။

- **Description list** (`<dl>`) က term (`<dt>`) နဲ့ ဖော်ပြချက် (`<dd>`) တွဲဖော်ပြဖို့ အသုံးဝင်ပါတယ်။
</Aside>
