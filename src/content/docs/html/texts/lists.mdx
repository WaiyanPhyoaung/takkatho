### List ထဲမှာ List ထည့်ခြင်း (Nested Lists)

Nested list ကို `<li>` ထဲမှာပဲ ထည့်ရပါတယ် — `<ul>` ဒါမှမဟုတ် `<ol>` ရဲ့ အောက်မှာ တိုက်ရိုက် မထည့်မိဖို့ သတိပြုပါ ခင်ဗျာ။

### ဖော်ပြချက် List များ (Description Lists)
Description list (`<dl>`) ဆိုတာက အသုံးအနှုန်းတွေ (`<dt>`) နဲ့ သူ့ရဲ့ ဖော်ပြချက်တွေ (`<dd>`) ကို တွဲပြီး ဖော်ပြဖို့အတွက် သုံးပါတယ်။ ဝေါဟာရ ရှင်းလင်းချက်တွေ၊ အမေးအဖြေ (FAQ) တွေ ရေးတဲ့အခါ အသင့်တော်ဆုံးပါပဲ။

- `<dl>` — Description list တစ်ခုလုံးကို ငုံထားတဲ့ Container
- `<dt>` — အသုံးအနှုန်း သို့မဟုတ် Item (Definition Term)
- `<dd>` — ဖော်ပြချက် (Definition Description)

<Aside type='tip' title='အကျဉ်းချုပ်'>

- Headings (`<h1>` ကနေ `<h6>` အထိ) တွေက အကြောင်းအရာကို အဆင့်ဆင့် စနစ်တကျဖြစ်အောင် ကူညီပေးပါတယ်။

- Paragraphs (`<p>`) တွေက စာသားတွေကို ဖတ်လို့ကောင်းတဲ့ အပိုင်းလေးတွေအဖြစ် ပုံစံချပေးပါတယ်။

- Line breaks (`<br>`) က စာကြောင်းအသစ် ဆင်းစေပြီး၊ Horizontal rules (`<hr>`) က အပိုင်းတွေကို မျဉ်းတား ခွဲခြားပေးပါတယ်။

- `<b>`, `<i>`, `<u>`, `<strong>`, `<em>`, နဲ့ `<mark>` လိုမျိုး Tag တွေသုံးပြီး စာသားကို လိုသလို ပုံစံပြောင်းနိုင်ပါတယ်။

- Lists တွေကို `<ul>` (Bullet List) နဲ့ `<ol>` (နံပါတ်စဉ် List) သုံးပြီး ဖန်တီးနိုင်ပါတယ်၊ Item တစ်ခုချင်းစီကိုတော့ `<li>` tag ထဲမှာ ထည့်ရပါတယ်။

- **Nested list** တွေကို `<li>` ထဲမှာ `<ul>` ဒါမှမဟုတ် `<ol>` ထပ်ထည့်ပြီး ဖန်တီးနိုင်ပါတယ်။

- **Description list** (`<dl>`) က Term (`<dt>`) နဲ့ သူ့ရဲ့ ဖော်ပြချက် (`<dd>`) တွဲဖော်ပြဖို့ အရမ်း အသုံးဝင်ပါတယ်။
</Aside>
