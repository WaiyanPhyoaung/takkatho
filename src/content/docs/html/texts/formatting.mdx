<Aside title="ရည်ရွယ်ချက်">
HTML မှာ စာသားတွေကို ပိုထင်ရှားအောင်၊ အလေးပေးအောင်၊ ပိုဖတ်ရလွယ်အောင် ပုံစံပြောင်းနိုင်တဲ့ tag တွေ၊ စာကြောင်းအသစ်ဆင်းနိုင်တဲ့ tag တွေကို ‌လေ့လာရန်။
</Aside>

HTML မှာ စာသားတွေကို ထင်ထင်ရှားရှားဖြစ်အောင် ပုံစံပြောင်းပေးတဲ့ tag တွေ အများကြီးရှိပါတယ်။ အသုံးအများဆုံး tag လေးတွေအကြောင်း ကြည့်လိုက်ရအောင်။

- **Bold**: `<b>` tag က စာလုံးကို Bold (အထူ) ဖြစ်စေတယ်။

- **Strong**: `<strong>` tag ကလည်း စာလုံးကို Bold ဖြစ်စေပေမယ့်၊ ဒီစာသားက အရေးကြီးတယ်ဆိုတဲ့ အဓိပ္ပာယ်ကိုပါ သက်ရောက်စေတယ် **(Accessibility နဲ့ SEO အတွက် ပိုအရေးပါပါတယ်)**။

- **Italic**: `<i>` tag က စာလုံးကို Italic (အစောင်း) ဖြစ်စေတယ်။

- **Emphasize**: `<em>` tag ကလည်း စာလုံးကို Italic ဖြစ်စေပေမယ့်၊ ဒီစာသားကို အလေးပေးချင်တယ်ဆိုတဲ့ အဓိပ္ပာယ် သက်ရောက်စေတယ်။

- **Underline**: `<u>` tag က စာသားအောက်မှာ မျဉ်းတားပေးတယ်။

- **Strikethrough**: `<s>` tag က စာသားအလယ်ကနေ မျဉ်းဖျက်ပေးတယ်။ ဒါက မှားနေတာ ဒါမှမဟုတ် ဖယ်ရှားလိုက်ပြီဆိုတာကို ပြချင်တဲ့အခါသုံးတယ်။

    ဥပမာ:
    <Code title="HTML" lang='html' code={formatting} />


> အကြံပြုချက်: စာသားပုံစံပြောင်းတာတွေကို ချင့်ချိန်ပြီးသုံးပါ။ အရမ်းများသွားရင် စာမျက်နှာက ရှုပ်ပွနေသလိုဖြစ်သွားနိုင်တယ်။

### စာကြောင်းအသစ်ဆင်းခြင်း (`<br>`) နှင့် မျဉ်းတားခြင်း (`<hr>`)

တစ်ခါတလေမှာ၊ စာကြောင်းတွေကြားမှာ နေရာလပ်တစ်ခုခြားချင်တာ ဒါမှမဟုတ် စာမျက်နှာပေါ်က အကြောင်းအရာအပိုင်းတွေကို ခွဲခြားချင်တာမျိုး ရှိနိုင်ပါတယ်။

- **Line Break (`<br>`)**: ဒီ tag က စာပိုဒ်အသစ်မစဘဲ စာကြောင်းတစ်ကြောင်းအောက်ကို ဆင်းချင်တဲ့အခါသုံးတယ်။

    ဥပမာ:

    <Code title='HTML' lang='html' code={brEg} />

- **Horizontal Rule (`<hr>`)**: ဒီ tag က စာမျက်နှာမှာ အကြောင်းအရာ အပိုင်းတွေကိုခွဲခြားဖို့ အလျားလိုက်မျဉ်းတစ်ကြောင်း ဖန်တီးပေးတယ်။

    ဥပမာ:

    <Code title="HTML" lang="html" code={hrEg} />

> အကြံပြုချက်: ဒီ tag တွေကို အကြောင်းအရာကို စနစ်တကျဖြစ်အောင် ကူညီပေးပေမယ့်၊ အလွန်အကျွံမသုံးသင့်ပါဘူး။

<Aside type="tip" title="အကျဉ်းချုပ်" >
- **Headings** တွေက အကြောင်းအရာကို စနစ်တကျဖြစ်အောင် ကူညီပေးပြီး၊ သင့်စာမျက်နှာကို ပုံစံကျစေတဲ့အတွက် အသုံးပြုသူတွေရော search engine တွေအတွက်ပါ ဖတ်ရလွယ်ကူစေပါတယ်။

- **Paragraphs** တွေက သင့်စာသားတွေကို အပိုင်းသေးသေးလေးတွေအဖြစ် ခွဲပေးပြီး၊ ဖတ်ရှုရလွယ်ကူစေပါတယ်။

- **Bold, italic, underline** လိုမျိုး Text formatting တွေက အကြောင်းအရာတွေကို ထင်ရှားစေပေမယ့်၊ ချင့်ချိန်ပြီးသုံးသင့်ပါတယ်။

- **Line breaks နဲ့ horizontal rules** တွေက အကြောင်းအရာကို မြင်သာအောင် ပြုလုပ်ရာမှာ အသုံးဝင်ပေမယ့်၊ ဒါတွေကိုလည်း အတိုင်းအတာတစ်ခုအထိပဲ သုံးသင့်ပါတယ်။
</Aside>
