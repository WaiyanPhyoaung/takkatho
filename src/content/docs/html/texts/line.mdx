#### စာလုံးပုံစံပြောင်းတဲ့ Tags များ
HTML မှာ စာသားတွေကို အလေးပေးဖို့၊ ပုံစံလှအောင်လုပ်ဖို့၊ ဒါမှမဟုတ် အရေးကြီးကြောင်းပြဖို့ ကူညီပေးတဲ့ tag အများအပြားရှိပါတယ်။ သူတို့ထဲက တချို့ကို ကြည့်လိုက်ရအောင်။

- **Bold Text (`<b>`):** စာလုံးကို Bold (အထူ) ဖြစ်စေတယ်။ ပုံစံပဲ ပြောင်းတယ်၊ အဓိပ္ပာယ်မပြောင်းဘူး။

- **Italic Text (`<i>`):** စာလုံးကို Italic (အစောင်း) ဖြစ်စေတယ်။ ပုံစံပဲ ပြောင်းတယ်။

- **Underlined Text (`<u>`):** စာသားအောက်မှာ <u>မျဉ်းသား</u>ပေးတယ်။

- **Important Text (`<strong>`):** စာသားက **တကယ်အရေးကြီးတယ်** ဆိုတဲ့ **အဓိပ္ပာယ်** ကိုပါ ဖော်ပြပေးပြီး၊ ပုံမှန်အားဖြင့် Bold နဲ့ပြတယ်။ Screen reader တွေက ဒီ tag ကို မြင်ရင် "အရေးကြီးသောစကားလုံး" လို့ ဖတ်ပြပေးပါတယ်။

- **Emphasized Text (`<em>`):** စာသားကို **အလေးထားတယ်** ဆိုတဲ့ **အဓိပ္ပာယ်** ဖော်ပြပြီး၊ ပုံမှန်အားဖြင့် Italic နဲ့ပြတယ်။

- **Highlighted Text (`<mark>`):** စာသားကို highlight လုပ်ပေးတယ်၊ ပုံမှန်အားဖြင့် အဝါရောင်နောက်ခံနဲ့ပါ။

- **Strikethrough (`<s>`):** စာသားအလယ်ကနေ မျဉ်းဖျက်ပေးတယ်။ ဒါက မှားနေတာ ဒါမှမဟုတ် ဖယ်ရှားလိုက်ပြီဆိုတာကို ပြချင်တဲ့အခါသုံးတယ်။

<Aside type="caution" title="&lt;b&gt; vs &lt;strong&gt; — မှတ်ထားသင့်တဲ့ ကွာခြားချက်">

| Tag | ပုံစံ | အဓိပ္ပာယ် | ဘယ်အချိန်သုံးမလဲ |
|-----|-------|-----------|-----------------|
| `<b>` | Bold | မပါ (ပုံစံပဲ) | ပုံစံချင်းပဲ Bold လိုတဲ့အခါ |
| `<strong>` | Bold | အရေးကြီးတဲ့ content | content တကယ်အရေးကြီးတဲ့အခါ |
| `<i>` | Italic | မပါ (ပုံစံပဲ) | ပုံစံချင်းပဲ Italic လိုတဲ့အခါ |
| `<em>` | Italic | အလေးပေးတဲ့ content | content ကို အလေးထားပြချင်တဲ့အခါ |

**ဥပမာ:** `သတိပြုပါ — <strong>password</strong> ကို မည်သူ့မျှ မပြောင်ပါနဲ့`
ဒီမှာ `<strong>` ကသုံးတာ "password" ဆိုတဲ့ စကားလုံးက တကယ်အရေးကြီးကြောင်း ဖော်ပြဖို့ ဖြစ်ပါတယ်။
</Aside>

### HTML Entities (အထူးအက္ခရာများ)
တချို့ character တွေဟာ HTML မှာ special meaning ရှိပါတယ် (ဥပမာ `<`, `>` ဟာ tag တွေ)။ ဒီ character တွေကို content အဖြစ် ပြချင်ရင် **HTML entity** တွေသုံးရပါတယ်။

| ပြချင်တဲ့ symbol | Entity code | ဥပမာ |
|-----------------|-------------|-------|
| `<` | `&lt;` | `1 &lt; 2` → 1 < 2 |
| `>` | `&gt;` | `2 &gt; 1` → 2 > 1 |
| `&` | `&amp;` | `HTML &amp; CSS` → HTML & CSS |
| `"` | `&quot;` | attribute value အတွက် |
| `©` | `&copy;` | မူပိုင်ခွင့် သင်္ကေတ |
| `®` | `&reg;` | မှတ်ပုံတင် trademark |
| `→` | `&rarr;` | ညာဘက်မြားခြားသားသင်္ကေတ |
| non-breaking space | `&nbsp;` | အကွာအဝေးကို break မလုပ်ဘဲ ထိန်းဖို့ |

**ဥပမာ -**

```html
<p>HTML &amp; CSS ဟာ web development ရဲ့ အခြေခံပါ။</p>
<p>&copy; 2024 Takkatho. All rights reserved.</p>
<p>5 &lt; 10 ဖြစ်လို့ HTML tag ဖြစ်မသွားပါ။</p>
```

<Aside type="caution" title="ဖြစ်တတ်တဲ့ အမှား">
HTML code ထဲမှာ `<` သို့မဟုတ် `&` ကို တိုက်ရိုက်မရေးပါနဲ့ — browser က tag/entity ဖြစ်သွားပြီး မမြင်ရတော့ ဒါမှမဟုတ် page ပျက်သွားနိုင်ပါတယ်။ `&lt;` နဲ့ `&amp;` ကို သုံးပါ။
</Aside>
