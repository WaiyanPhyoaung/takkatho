<Aside title="ရည်ရွယ်ချက်">
ဒီအခန်းမှာတော့ Webpage တွေထဲကို `<video>` နဲ့ `<audio>` tag တွေသုံးပြီး Media File တွေ ဘယ်လို ထည့်သွင်းရမလဲဆိုတာ လေ့လာကြပါမယ် ခင်ဗျာ။ Browser မျိုးစုံမှာ အဆင်ပြေပြေ အလုပ်လုပ်ဖို့ (Browser Compatibility) အတွက် Source မျိုးစုံ ဘယ်လိုသုံးရမလဲနဲ့ Media တွေအတွက် အကောင်းဆုံး လက်တွေ့အသုံးချနည်း (Best Practices) တွေကိုလည်း သေသေချာချာ သင်ယူရမှာပါ။
</Aside>

### Video များ ထည့်သွင်းခြင်း
`<video>` tag ကိုသုံးပြီး Webpage တွေမှာ Video တွေကို ထည့်သွင်းနိုင်ပါတယ်။ ဒါက User တွေကို Browser ထဲမှာတင် တိုက်ရိုက် Video ကြည့်ခွင့် ပေးပါတယ်။

အခြေခံ Video Tag:

```html
<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  Your browser does not support the video tag.
</video>
```
<br/>
- `<video>`: Video ထည့်သွင်းဖို့ သုံးတဲ့ Tag ပါ။

- **`width` and `height`**: Video player ရဲ့ အရွယ်အစားကို သတ်မှတ်ပေးပါတယ်။

- `controls`: Video အတွက် လိုအပ်တဲ့ အခြေခံ Control တွေ (Play, Pause, Volume) ကို ထည့်ပေးပါတယ်။

- `<source>`: Video file နဲ့ သူ့ရဲ့ Format ကို သတ်မှတ်ပေးပါတယ်။ Format မတူတဲ့ Source မျိုးစုံ (ဥပမာ .mp4, .webm) တွဲပြီး ထည့်ပေးထားနိုင်ပါတယ်။

#### ဘာကြောင့် Source မျိုးစုံ သုံးရတာလဲ?
Browser တစ်ခုနဲ့တစ်ခုက Video format တွေ Support လုပ်တာ မတူညီကြပါဘူး။ Source မျိုးစုံကို ထည့်ပေးထားခြင်းအားဖြင့် Browser အားလုံးမှာ အလုပ်လုပ်တာ သေချာသွားတာပေါ့။

### Audio များ ထည့်သွင်းခြင်း
`<audio>` tag ကို ခင်ဗျားရဲ့ Webpage ထဲမှာ Audio file တွေ (သီချင်း၊ Podcast၊ Sound effect တွေလိုမျိုး) ထည့်သွင်းဖို့ သုံးပါတယ်။

အခြေခံ Audio Tag:
```html
<audio controls>
  <source src="audio.mp3" type="audio/mp3">
  Your browser does not support the audio element.
</audio>
```
- `<audio>`: Audio file ထည့်သွင်းဖို့ သုံးတဲ့ Tag ပါ။

- `controls`: Play, Pause, Volume လိုမျိုး Audio control တွေကို ထည့်ပေးပါတယ်။

- `<source>`: Audio file နဲ့ သူ့ရဲ့ Format (ဥပမာ .mp3, .ogg) ကို သတ်မှတ်ပေးပါတယ်။

### Media File များအတွက် အကောင်းဆုံး လက်တွေ့အသုံးချနည်းများ
ခင်ဗျားရဲ့ Website မှာ ပုံ၊ Video, Audio တွေ သုံးတဲ့အခါ၊ အောက်ပါ အကောင်းဆုံး အလေ့အကျင့်တွေကို ထည့်သွင်း စဉ်းစားသင့်ပါတယ်-

- **ပုံ Size တွေကို Optimize လုပ်ပါ -** ပုံအကြီးကြီးတွေက ခင်ဗျားရဲ့ Website ကို အရမ်း နှေးကွေးသွားစေနိုင်ပါတယ်။ အရည်အသွေးနဲ့ File size မျှတအောင် ပုံတွေကို အမြဲတမ်း Compress လုပ်ပေးပါ။

- **Responsive Media ကို သုံးပါ -** ခင်ဗျားရဲ့ Media file တွေက မတူညီတဲ့ Screen size တွေအလိုက် အလိုက်သင့် ပြောင်းလဲသွားအောင် Responsive CSS တွေကို အသုံးပြုပါ။

- **စာသားဖြင့် အစားထိုး ဖော်ပြချက်တွေ ထည့်ပေးပါ -** Accessibility အတွက် ပုံတွေမှာ alt text ကို အမြဲသုံးပါ။ Video နဲ့ Audio တွေအတွက်တော့ Transcripts ဒါမှမဟုတ် Captions တွေ ထည့်ပေးဖို့ စဉ်းစားပါ။

- **Browser တွေမှာ အဆင်ပြေအောင် လုပ်ပါ -** Video နဲ့ Audio တွေ Browser အားလုံးမှာ အဆင်ပြေပြေ အလုပ်လုပ်တာ သေချာအောင် Format မျိုးစုံကို တွဲသုံးပေးပါ။

<Aside type="tip" title="အကျဉ်းချုပ်">
- **ပုံများ -** `<img>` tag ကို **src နဲ့ alt** တို့နဲ့အတူ အမြဲ သုံးပါတယ်။

- **Video -** `<video>` tag နဲ့ Video တွေကို ထည့်သွင်းနိုင်ပြီး၊ Browser အားလုံး အဆင်ပြေအောင် Source မျိုးစုံ ပေးနိုင်ပါတယ်။

- **Audio -** `<audio>` tag ကိုသုံးပြီး Audio တွေ ထည့်သွင်းနိုင်ပါတယ်၊ Video တွေလိုပဲ Format မျိုးစုံ သုံးနိုင်ပါတယ်။

- **အလေ့အကျင့်‌ကောင်းများ -** Media တွေကို မြန်မြန် Load ဖြစ်အောင် Optimize လုပ်ပါ၊ Accessibility အတွက် စာသားဖော်ပြချက်တွေ ရေးပါ၊ ပြီးတော့ Device မျိုးစုံအတွက် Responsive ဖြစ်အောင် လုပ်ပေးပါ ခင်ဗျာ။
</Aside>
