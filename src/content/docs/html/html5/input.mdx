### HTML5 မှာ ပါဝင်လာတဲ့ Input Type အသစ်များ
HTML5 မှာ Form တွေအတွက် Input type အသစ်တွေ အများကြီး ပါဝင်လာပါတယ်။ ဒါကြောင့် User တွေအတွက် အဆင်ပြေတဲ့ Form တွေကို Code တွေ အများကြီး ရေးနေစရာမလိုဘဲ အလွယ်တကူ ဖန်တီးနိုင်လာပါတယ်။

##### 1. Email (`<input type="email">`)
Email လိပ်စာ ထည့်ခိုင်းဖို့ သုံးပါတယ်။ Browser က ထည့်လိုက်တဲ့ စာသားက Email format အမှန် (ဥပမာ - `user@example.com`) ဟုတ်မဟုတ် အလိုအလျောက် စစ်ဆေးပေးပါတယ်။

**ဥပမာ -**

```html
<label for="email">Email:</label>
<input type="email" id="email" name="email">
```
##### 2. URL (`<input type="url">`)
URL (ဝဘ်ဆိုဒ်လိပ်စာ) တွေ ထည့်ဖို့အတွက် သုံးပါတယ်။ ထည့်လိုက်တဲ့ စာသားက Web address အမှန် (ဥပမာ - `https://www.example.com`) ဟုတ်မဟုတ် စစ်ဆေးပေးပါတယ်။

**ဥပမာ -**

```html
<label for="website">Website:</label>
<input type="url" id="website" name="website">
```
##### 3. Date (`<input type="date">`)
User တွေကို ပြက္ခဒိန် (Calendar) ပုံစံလေး ပေါ်လာပြီး ရက်စွဲတစ်ခု ရွေးခိုင်းနိုင်ပါတယ်။

**ဥပမာ -**

```html
<label for="dob">Date of Birth:</label>
<input type="date" id="dob" name="dob">
```

<ContentImage src={date} alt="date picker"  height={300}/>

##### 4. Range (`<input type="range">`)
User တွေကို သတ်မှတ်ထားတဲ့ Range အတွင်းက တန်ဖိုးတစ်ခုကို Slider ပုံစံလေး ရွှေ့ပြီး ရွေးခိုင်းနိုင်ပါတယ်။

**ဥပမာ -**

```html
<label for="volume">Volume:</label>
<input type="range" id="volume" name="volume" min="0" max="100">
```

<ContentImage src={range} alt="range slider" height={300} width={200} />
##### 5. Color (`<input type="color">`)
အရောင်တစ်ခုကို ရွေးချယ်ဖို့အတွက် Color picker (အရောင်ရွေးတဲ့ Box လေး) တစ်ခု ပေါ်လာစေပါတယ်။

**ဥပမာ -**

```html
<label for="color">Pick a Color:</label>
<input type="color" id="color" name="color">
```
<ContentImage src={color} alt="color picker" height={300} />
