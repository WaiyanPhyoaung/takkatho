### ရိုးရှင်းသော HTML Document တစ်ခု ဖန်တီးခြင်း

ရိုးရှင်းတဲ့ HTML document တစ်ခုမှာ element လို့ခေါ်တဲ့ အစိတ်အပိုင်းလေးတွေ အများကြီးနဲ့ ဖွဲ့စည်းထားပြီး၊ element တစ်ခုချင်းစီမှာ သူ့ရဲ့ သီးသန့်လုပ်ဆောင်ရမယ့် တာဝန်ကိုယ်စီရှိပါတယ်။ အောက်မှာ အခြေခံအကျဆုံး HTML document တစ်ခုရဲ့ ပုံစံကို ကြည့်ရအောင်။

<Code title="HTML" lang='html' code={basicHtml}/>

- `<!DOCTYPE html>`: ဒါက HTML document တိုင်းရဲ့ ပထမဆုံးလိုင်းမှာ အမြဲရှိရပါတယ်။ သူက browser ကို "ဒီစာမျက်နှာက HTML5 နဲ့ ရေးထားတာနော်" လို့ ပြောလိုက်တာနဲ့ အတူတူပါပဲ။

- `<html>`: ဒါက document တစ်ခုလုံးရဲ့ အဓိကအကျဆုံး tag ဖြစ်ပြီး၊ အရာအားလုံးရဲ့ အခွံကြီးလို့ ပြောလို့ရပါတယ်။ သင့် webpage မှာရှိတဲ့ အရာအားလုံးက ဒီ `<html>` tag အဖွင့်အပိတ်ကြားထဲမှာပဲ ရှိနေရပါမယ်။

- `<head>`: ဒီအပိုင်းကတော့ webpage ရဲ့ အချက်အလက်တွေ ထည့်တဲ့နေရာပါ။ ဥပမာ - စာမျက်နှာရဲ့ ခေါင်းစဉ် (`<title>`)၊ CSS stylesheet တွေ၊ ဒါမှမဟုတ် JavaScript file တွေလို ပြင်ပ file တွေကို ချိတ်ဆက်တာမျိုးတွေ လုပ်ပါတယ်။

- `<title>`: ဒီ tag က webpage ရဲ့ ခေါင်းစဉ်ကို သတ်မှတ်ပေးပြီး၊ browser ရဲ့ tab မှာပေါ်တဲ့ စာသားလေးဖြစ်ပါတယ်။

- `<body>`: ဒါကတော့ အဓိကအကျဆုံးနေရာပါပဲ။ သင်က user ကို မြင်စေချင်တဲ့ အကြောင်းအရာအကုန်လုံးဖြစ်တဲ့ စာသားတွေ၊ ပုံတွေ၊ link တွေ အစရှိတာတွေကို ဒီနေရာမှာ ထည့်ရပါတယ်။

### HTML Tags နှင့် Elements
HTML မှာ tag ဆိုတာက သင်ဘယ်လို content အမျိုးအစားကို သုံးနေလဲဆိုတာ browser ကို ပြောပြတဲ့ အထူး keyword တစ်ခုပါ။ Tag တွေကိုသုံးပြီး element တွေကို ဖန်တီးရပြီး၊ အဲ့ဒီ element တွေက webpage တစ်ခုရဲ့ အုတ်ချပ်လေးတွေလိုပါပဲ။

Element တစ်ခုတိုင်းကို အဖွင့် tag (opening tag) နဲ့ အပိတ် tag (closing tag) ကြားမှာ ထည့်ရေးရပါတယ်။ ဥပမာ -

- Opening tag: `<p>`

- Closing tag: `</p>`



အဲဒီ အဖွင့်နဲ့ အပိတ် tag ကြားထဲမှာရှိတဲ့ အရာအားလုံးက အဲဒီ element ရဲ့ အစိတ်အပိုင်းဖြစ်ပါတယ်။

ဥပမာ:

<Code title="HTML" lang='html' code={pTag} />

ဒီဥပမာမှာ၊ `<p>` tag က စာပိုဒ် (paragraph) တစ်ခုကို ဖန်တီးပေးပါတယ်။

တချို့ element တွေကတော့ **self-closing** ဖြစ်ပါတယ်၊ ဆိုလိုတာက သူတို့အတွက် အပိတ် tag သီးသန့် ပြန်ထည့်စရာမလိုပါဘူး။ ဥပမာ -

- `<img>`: ဒါက ပုံတွေပြဖို့သုံးပြီး၊ သူ့မှာ အပိတ် `</img>` tag ပြန်ထည့်စရာမလိုတဲ့အတွက် self-closing element တစ်ခုဖြစ်ပါတယ်။

ဥပမာ:

<Code title='HTML' lang='html' code={imgTag} />
