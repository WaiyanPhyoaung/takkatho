### HTML Document ရဲ့ တည်ဆောက်ပုံ
HTML Page တိုင်းမှာ အခြေခံကျတဲ့ တည်ဆောက်ပုံတစ်ခု ရှိပါတယ်။ အဲ့ဒီတည်ဆောက်ပုံက ခင်ဗျား တည်ဆောက်မယ့် Webpage တိုင်းရဲ့ အုတ်မြစ်ပါပဲ။ သူက အထဲက အကြောင်းအရာတွေကို ဘယ်လို စီစဉ်မလဲ၊ ဘယ်လို ပြသမလဲဆိုတာကို သတ်မှတ်ပေးပါတယ်။

ဒါကတော့ HTML document တစ်ခုရဲ့ အခြေခံ တည်ဆောက်ပုံ နမူနာ ဖြစ်ပါတယ်-

<Code title="HTML" lang='html' code={basicHtml}/>


- `<!DOCTYPE html>`:
ဒါကတော့ Document အမျိုးအစားကို ကြေညာတာပါ။ ဒီစာမျက်နှာကို နောက်ဆုံး Version ဖြစ်တဲ့ HTML5 နဲ့ ရေးထားတယ်ဆိုတာကို Browser ကို ပြောပြတာဖြစ်ပါတယ်။

- `<html>`:
ဒီ tag က document တစ်ခုလုံးကို အုပ်ထားပြီး၊ သူ့အတွင်းမှာရှိတဲ့ အရာအားလုံးက HTML document ဖြစ်တယ်ဆိုတာကို browser ကို သိစေပါတယ်။

- `<head>`:
head အပိုင်းမှာ webpage နဲ့ပတ်သက်တဲ့ အချက်အလက် (metadata) တွေ ပါပါတယ်။ Metadata ဆိုတာက user တွေ တိုက်ရိုက်မမြင်ရပေမယ့် browser အတွက် အရေးကြီးတဲ့ အချက်အလက်တွေပါ။ ဥပမာ -

    - `<meta charset="UTF-8">`: ဒီ Tag လေးကတော့ မြန်မာစာလိုမျိုး ဘာသာစကားအစုံနဲ့ အထူးစာလုံးတွေ (é, ©, 💻) ကို Webpage ပေါ်မှာ ကောင်းကောင်းမွန်မွန် ပေါ်စေဖို့အတွက်ပါ။

    - `<meta name="viewport" content="width=device-width, initial-scale=1.0">`: ဒီ Tag ကတော့ Webpage ကို ဖုန်းလိုမျိုး Mobile Device တွေမှာ ကြည့်တဲ့အခါ Device ရဲ့ Screen အကျယ်အလိုက် အဆင်ပြေပြေ ပေါ်အောင် ချိန်ညှိပေးပါတယ်။

    - `<title>`: Browser ရဲ့ tab မှာ ပေါ်မယ့် စာမျက်နှာရဲ့ ခေါင်းစဉ်ကို သတ်မှတ်ပေးပါတယ်။

- `<body>`:
ဒါကတော့ webpage ရဲ့ အဓိက အပိုင်းဖြစ်ပြီး၊ user တွေ မြင်ရမယ့် အကြောင်းအရာအားလုံးကို ဒီထဲမှာ ထည့်ရပါတယ်။ ခေါင်းစဉ်တွေ၊ စာပိုဒ်တွေ၊ ပုံတွေ၊ link တွေ အားလုံး ဒီထဲမှာပဲ ပါပါတယ်။

### Tag တွေနဲ့ Element တွေ
HTML မှာ tag တွေက အဓိက အုတ်မြစ်တွေပါပဲ။ Tag တွေကို စာသား၊ ပုံ၊ link လိုမျိုး မတူညီတဲ့ အကြောင်းအရာအမျိုးအစားတွေကို သတ်မှတ်ဖို့ သုံးပါတယ်။ HTML element တစ်ခုမှာ အဖွင့် tag (opening tag) နဲ့ အပိတ် tag (closing tag) ဆိုပြီး ပါလေ့ရှိပါတယ်။

**ဥပမာ -**

<Code title="HTML" lang='html' code={h1Tag}/>


- `<h1>`: ဒါက ခေါင်းစဉ်ရဲ့ အစကို သတ်မှတ်ပေးတဲ့ အဖွင့် tag ပါ။

- `</h1>`: ဒါက ခေါင်းစဉ်ရဲ့ အဆုံးကို သတ်မှတ်ပေးတဲ့ အပိတ် tag ပါ။

အဖွင့် tag နဲ့ အပိတ် tag ကြားထဲမှာရှိတဲ့ အရာအားလုံးက အကြောင်းအရာ (content) ဖြစ်ပါတယ်။

### HTML Tag အမျိုးအစားများ
HTML tag တွေမှာ အဓိကအားဖြင့် အမျိုးအစား နှစ်မျိုးရှိပါတယ်။

#### Paired Tags (အတွဲလိုက် Tag များ):
ဒီ tag တွေက အဖွင့် tag နဲ့ အပိတ် tag ဆိုပြီး အတွဲလိုက်လာပါတယ်။

ဥပမာ:
<Code title="HTML" lang='html' code={pTag}/>

#### Self-closing Tags (အပိတ်မလိုသော Tag များ):
ဒီ tag တွေကတော့ အပိတ် tag ပြန်ပိတ်စရာမလိုပါဘူး။ ပုံတွေ ဒါမှမဟုတ် စာကြောင်းဆင်းတာတွေအတွက် သုံးပါတယ်။

ဥပမာ:
<Code title="HTML" lang='html' code={selfClosingTag}/>

`<img>` ဆိုတာ ပုံထည့်ဖို့သုံးတဲ့ self-closing tag တစ်ခု ဖြစ်ပါတယ်။

### DOM Tree — HTML Document ရဲ့ မိသားစုပင်
Browser က HTML ကိုဖတ်ပြီး **DOM (Document Object Model)** ဆိုတဲ့ မိသားစုပင်ပုံစံ တည်ဆောက်ပါတယ်။ `<html>` က အပေါ်ဆုံးမှာ ရပ်နေပြီး၊ `<head>` နဲ့ `<body>` က သူ့ children ဖြစ်ပါတယ်။ ထို element တွေ အောက်မှာ ထပ်ဆင့် children တွေ ဆင်းသွားပါတယ်။

မိသားစုပင်တစ်ပင်မှာ root ကနေ ကိုင်းခွဲတွေ ပေါက်သွားသလိုမျိုး HTML element တွေကလည်း parent → children ဆက်ဆံရေး ရှိပါတယ်။
