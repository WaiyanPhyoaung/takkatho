`<body>` အပိုင်းမှာတော့ သုံးစွဲသူတွေ မြင်ရမယ့် webpage ရဲ့ အကြောင်းအရာ အားလုံး ပါဝင်ပါတယ်။ ခေါင်းစဉ်တွေ၊ စာပိုဒ်တွေ၊ ပုံတွေနဲ့ link တွေလိုမျိုး အရာအားလုံးကို ဒီနေရာမှာပဲ ထည့်ရမှာ ဖြစ်ပါတယ်။

### `<body>` ထဲမှာ အသုံးများတဲ့ element တွေ:

-   #### Headings (`<h1>` to `<h6>`):
    ခေါင်းစဉ်တွေက စာမျက်နှာရဲ့ တည်ဆောက်ပုံကို သတ်မှတ်ပေးပြီး အကြောင်းအရာတွေကို စနစ်တကျ ဖြစ်အောင် ကူညီပေးပါတယ်။ `<h1>` က အရေးအကြီးဆုံး ခေါင်းစဉ်ဖြစ်ပြီး `<h6>` ကတော့  ခေါင်းစဉ်ငယ် ဖြစ်ပါတယ်။

    ဥပမာ:

    <Code title='HTML' lang='html' code={heading} />

-   #### Paragraphs (`<p>`):
    စာပိုဒ်တွေ သတ်မှတ်ဖို့အတွက် `<p>` tag ကို သုံးပါတယ်။

    ဥပမာ:

    <Code title='HTML' lang='html' code={paragraph} />


-   #### Images (`<img>`):
    `<img>` tag ကို စာမျက်နှာပေါ်မှာ ပုံတွေဖော်ပြဖို့ သုံးပါတယ်။

    ဥပမာ:

    <Code title='HTML' lang='html' code={imgTag} />


-   #### Links (`<a>`):
    `<a>` tag ကို တခြားစာမျက်နှာတွေကို ချိတ်ဆက်ပေးတဲ့ hyperlink တွေ ဖန်တီးဖို့ သုံးပါတယ်။

    ဥပမာ:

    <Code title='HTML' lang='html' code={linkTag} />


### နမူနာ - အခြေခံ HTML Page တစ်ခု
ခုနက ပြောခဲ့တာတွေအားလုံးကို ပေါင်းစပ်ထားတဲ့ နမူနာတစ်ခုကို ကြည့်လိုက်ရအောင်။

<Code title='HTML' lang='html' code={basicHtml} />

*ရှင်းလင်းချက် -**

- `<!DOCTYPE html>`: Browser ကို HTML5 သုံးထားကြောင်း ပြောပြပါတယ်။
- `<html>` **tag**: စာမျက်နှာတစ်ခုလုံးကို ခြုံငုံထားပါတယ်။
- `<head>` **tag**: စာမျက်နှာ ခေါင်းစဉ်နဲ့ meta tag တွေလို metadata တွေ ပါဝင်ပါတယ်။
- `<body>` **tag**: မြင်ရတဲ့ အကြောင်းအရာတွေ (ခေါင်းစဉ်၊ စာပိုဒ်၊ ပုံ၊ link) တွေ ပါဝင်ပါတယ်။
