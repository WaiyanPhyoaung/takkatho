### HTML ဆိုတာဘာလဲ?

HTML(HyperText Markup Language) ဆိုတာ web ပေါ်မှာ အကြောင်းအရာတွေကို တည်ဆောက်ဖို့နဲ့ ပုံစံချဖို့သုံးတဲ့ language တစ်ခုပါ။

### HTML က ဘာလို့လိုအပ်တာလဲ?

- HTML က webpage တစ်ခုရဲ့ ဖွဲ့စည်းပုံကို သတ်မှတ်ပေးပါတယ်။

- သူက browser ကို screen ပေါ်မှာ ဘာတွေပြရမလဲဆိုတာ (စာတွေ၊ ပုံတွေ၊ video တွေ) ပြောပြတယ်။

- HTML က ပုံစံလှအောင်လုပ်တာ (styling) အတွက် တာဝန်မရှိပါဘူး (အဲဒါက CSS ရဲ့အလုပ်ပါ)၊ ပြီးတော့ အပြန်အလှန်တုံ့ပြန်မှုတွေ (interactivity) ထည့်ပေးတာလည်း မဟုတ်ပါဘူး (အဲဒါက JavaScript ရဲ့အလုပ်ပါ)။

### HTML က ဘယ်လိုအလုပ်လုပ်သလဲ?
HTML က "tags" လို့ခေါ်တဲ့ စနစ်ကိုသုံးပါတယ်။ Tag တွေ ရဲ့ ကြားထဲမှာ အကြောင်းအရာတွေကိုရေးပြီး browser ကို ဘယ်လိုပြရမလဲဆိုတာ ပြောပြတယ်။

- `<p>` ဆိုတာက စာပိုဒ်တစ်ခုကို သတ်မှတ်ပေးတဲ့ tag ပါ။

### Web Development မှာ HTML ရဲ့ အခန်းကဏ္ဍ
Web development မှာ HTML က အဓိကကျတဲ့ အခန်းကဏ္ဍကနေ ပါဝင်ပါတယ်။ သင်အင်တာနက်ပေါ်မှာ မြင်တွေ့ရတဲ့ webpage တိုင်းရဲ့ အခြေခံအုတ်မြစ်က HTML ပါပဲ။ သူက ဖွဲ့စည်းပုံနဲ့ အကြောင်းအရာအတွက် ဖြစ်ပါတယ်။

### Web Development ရဲ့ အဓိကအစိတ်အပိုင်းများ:

- **HTML:** ဖွဲ့စည်းပုံကို သတ်မှတ်တယ် (အိမ်အလွတ်တစ်လုံးလို)။
- **CSS (Cascading Style Sheets):** ဖွဲ့စည်းပုံကို ပုံစံလှအောင်လုပ်တယ် (အိမ်ကို ဆေးသုတ်တာ၊ အလှဆင်တာလို)။
- **JavaScript:** အပြန်အလှန်တုံ့ပြန်မှုတွေ ထည့်ပေးတယ် (မီးခလုတ်ဖွင့်တာ၊ တံခါးဖွင့်တာပိတ်တာတွေလိုမျိုး)။
