### HTML ဆိုတာ ဘာလဲ။
**HTML (Hypertext Markup Language)** ဆိုတာ Website တွေနဲ့ Webpage တွေရဲ့ အခြေခံအကျဆုံး အရိုးစု ကို တည်ဆောက်တဲ့ ဘာသာစကားတစ်ခု ဖြစ်ပါတယ်။ သူက Website တစ်ခုမှာ ဘယ်အကြောင်းအရာကို ဘယ်လိုပုံစံနဲ့ပြမလဲဆိုတာကို Browser (ဥပမာ - Chrome, Firefox) ကို ပြောပြပေးပါတယ်။

#### ဥပမာအနေနဲ့ အိမ်တစ်လုံး နဲ့ နှိုင်းယှဉ်ကြည့်ရအောင်။
HTML ဆိုတာ အဲ့ဒီအိမ်ရဲ့ **အုတ်မြစ်နဲ့ အဓိကတိုင်တွေ** လိုပါပဲ။ အိမ်ရဲ့ အလှအပ (Design) တွေနဲ့ အခြားလုပ်ဆောင်ချက် (Function) တွေကိုတော့ CSS နဲ့ JavaScript လို တခြားအရာတွေက တာဝန်ယူပေမယ့်၊ အဓိကတည်ဆောက်ပုံကိုတော့ HTML ကပဲ ပြုလုပ်တာပါ။

- **Hypertext** - Website စာမျက်နှာတွေကို တစ်ခုနဲ့တစ်ခု ချိတ်ဆက်ပေးတဲ့ Link တွေကို ရည်ညွှန်းပါတယ်။ ဒီ Link တွေကနေတစ်ဆင့် စာမျက်နှာတွေကြား ကူးလူးသွားလာနိုင်တာပါ။

- **Markup** - အကြောင်းအရာတွေကို ဘယ်လိုပြသရမလဲဆိုတာ Browser ကို ညွှန်ကြားတဲ့ သင်္ကေတ (Tag & Element) တွေကို ဆိုလိုပါတယ်။

> အတိုချုပ်ပြောရရင်၊ HTML ဆိုတာ Website တိုင်းရဲ့ အခြေခံအုတ်မြစ် ပါပဲ။

### Web Development အတွက် HTML က ဘာလို့ အရေးကြီးတာလဲ။
သင်မြင်တွေ့နေရတဲ့ Website တိုင်းရဲ့ အခြေခံအုတ်မြစ်ကို HTML နဲ့ တည်ဆောက်ထားတာပါ။ HTML မရှိရင် Website တွေမှာ စာတွေ၊ ပုံတွေ၊ ဘာအကြောင်းအရာမှ ရှိမှာမဟုတ်ပါဘူး။

- **Browser တွေက Website ကို ကောင်းကောင်းမွန်မွန်ပြသနိုင်ဖို့** HTML ကို လိုအပ်ပါတယ်။

- **SEO (Search Engine Optimization)** အတွက်လည်း HTML က အရေးပါပါတယ်။ Google လို Search Engine တွေက သင့် Website က အကြောင်းအရာတွေကို နားလည်ပြီး လူတွေရှာဖွေတဲ့အခါ အလွယ်တကူတွေ့ရှိနိုင်အောင် HTML က ကူညီပေးပါတယ်။

- HTML ကို တတ်မြောက်ထားရင် ကိုယ့် Website ရဲ့ အကြောင်းအရာတွေကို **ကိုယ်တိုင် ဖန်တီးနိုင်၊ ပြင်ဆင်နိုင်၊ ထိန်းချုပ်နိုင်မှာ** ဖြစ်ပါတယ်။
