### HTML Document တစ်ခုရဲ့ တည်ဆောက်ပုံ
HTML Document တိုင်းမှာ အကြောင်းအရာတွေကို စနစ်တကျဖြစ်အောင် သတ်မှတ်ထားတဲ့ တည်ဆောက်ပုံတစ်ခု ရှိပါတယ်။

#### DOCTYPE ကြေညာချက်
ဒါက Browser ကို ဒီစာမျက်နှာက HTML ရဲ့ ဘယ် version နဲ့ ရေးထားတာလဲဆိုတာ ပြောပြတာပါ။ အသုံးအများဆုံးကတော့ အောက်ပါအတိုင်းဖြစ်ပါတယ်။

<Code title="HTML" code="<!DOCTYPE html>">
</Code>

ဒီတစ်ကြောင်းကို HTML ဖိုင်ရဲ့ **အပေါ်ဆုံးမှာ** အမြဲတမ်းရေးရပါတယ်။ ဒါက ဒီစာမျက်နှာကို HTML5 နဲ့ ရေးထားကြောင်း Browser ကို အသိပေးလိုက်တာပါ။

#### HTML Element (`<html>` tag)

Website ရဲ့ အကြောင်းအရာအားလုံးကို ဒီ `<html>` tag ကြားမှာ ထည့်ရေးရပါတယ်။ ဒါက Document တစ်ခုလုံးရဲ့ အဓိက အခွံကြီးပါပဲ။

<Code title="HTML" code="<html>
    </html>" />

#### Head Section (`<head>` tag)


`<head>` အပိုင်းထဲမှာ Website နဲ့ ပတ်သက်တဲ့ **တိုက်ရိုက်မမြင်ရတဲ့** အချက်အလက်တွေ ကို ထည့်ပါတယ်။ ဥပမာ - စာမျက်နှာရဲ့ခေါင်းစဉ်၊ အခြားဖိုင်တွေကို ချိတ်ဆက်တာမျိုးတွေပေါ့။

<Code title="HTML" code={headTag}/>



`<meta charset="UTF-8">` - မြန်မာစာလို ဘာသာစကားအမျိုးမျိုးကို ကောင်းကောင်းမွန်မွန် ပေါ်စေဖို့ ထည့်ပေးရတာပါ။

`<title>` - Browser ရဲ့ Tab မှာပေါ်မယ့် နာမည်ကို ဒီမှာ သတ်မှတ်ပေးရပါတယ်။

#### Body Section (`<body>` tag)

`<body>` အပိုင်းကတော့ Website မှာ တကယ်မြင်နေရတဲ့ အကြောင်းအရာတွေ ကို ထည့်တဲ့နေရာပါ။ စာတွေ၊ ပုံတွေ၊ ခေါင်းစဉ်တွေ၊ လင့်ခ်တွေ အားလုံးကို ဒီ `<body>` tag ကြားမှာပဲ ရေးရပါတယ်။

<Code code={bodyTag} title='HTML'/>

ဒီ `<body>` ထဲမှာ သင့် Website မှာပြချင်တဲ့ ခေါင်းစဉ်ကြီး (Heading)၊ စာပိုဒ် (Paragraph)၊ ပုံ (Image) နဲ့ လင့်ခ် (Link) တွေကို ထည့်သွင်းရမှာ ဖြစ်ပါတယ်။
