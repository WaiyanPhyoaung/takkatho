### Browser တွေက HTML ကို ဘယ်လိုနားလည်လဲ
သင်က browser မှာ website ရဲ့ URL ကို ရိုက်ထည့်လိုက်တဲ့အခါ၊ webpage တစ်ခုကို ပြသဖို့အတွက် browser က အောက်ပါအလုပ်တွေကို အဆင့်ဆင့်လုပ်ဆောင်ပါတယ်-

    1. **HTML ဖိုင်ကို သွားယူတယ်** - Browser က server ဆီကနေ HTML code ကို သွားယူ (load) ပါတယ်။

    2. **HTML ကို ဖတ်ပြီး နားလည်အောင်လုပ်တယ်** - သူက HTML တည်ဆောက်ပုံကို ဖတ်ပြီး အကြောင်းအရာတွေကို ဘယ်လိုပြသရမလဲဆိုတာကို နားလည်အောင်လုပ်ပါတယ်။

    3. **စာမျက်နှာကို မြင်ရအောင် ပြပေးတယ်** - သူက အကြောင်းအရာတွေ (စာသား၊ ပုံ၊ link) ကို ဖတ်လို့ရပြီး တုံ့ပြန်နိုင်တဲ့ ပုံစံနဲ့ ပြသပေးပါတယ်။

Browser တွေမှာ HTML code တွေကို ဖတ်ပြီး သင့် screen ပေါ်မှာ မြင်ရတဲ့ ပုံစံအဖြစ် ပြောင်းပေးတဲ့ **HTML rendering engine** ဆိုတာ ပါလာတဲ့အတွက် HTML ကို ဘယ်လိုပြရမလဲဆိုတာကို သူတို့က သိတာဖြစ်ပါတယ်။

### Browser က စာမျက်နှာကို ပြသပုံ အဆင့်ဆင့်
Browser က HTML ကို ဘယ်လိုသွားယူပြီး ပြသတယ်ဆိုတဲ့ လုပ်ငန်းစဉ်ကို ရှင်းလင်းအောင် ပုံကြမ်း သို့မဟုတ် infographic တစ်ခုနဲ့ ပြနိုင်ပါတယ်။ ဥပမာ-

User က URL ကို ရိုက်ထည့်တယ် → Browser က HTML ဖိုင်ကို တောင်းဆိုတယ် → Browser က HTML ကို load လုပ်ပြီး နားလည်အောင်လုပ်တယ် → စာမျက်နှာကို screen ပေါ်မှာ ပြသပေးတယ်။
