### HTML Attributes ဆိုတာ ဘာလဲ?
Attribute ဆိုတာက HTML tag တွေရဲ့ အပြုအမူ (Behavior) ကို ထိန်းချုပ်ဖို့ ဒါမှမဟုတ် Element တစ်ခုအကြောင်း အသေးစိတ် အချက်အလက်တွေ ထပ်ထည့်ပေးဖို့ သုံးတဲ့ **အပို အချက်အလက်တွေ** ဖြစ်ပါတယ်။ သူတို့ကို အမြဲတမ်း Element တစ်ခုရဲ့ အဖွင့် Tag (Opening tag) ထဲမှာပဲ ထည့်ရေးရပါတယ်။

**Attribute တစ်ခုရဲ့ တည်ဆောက်ပုံ**<br/>
`Attribute Name` = `"Attribute Value"`

**ဥပမာ -**

```html
<a href="https://www.example.com">Visit Example</a>
```

- `href` က **Attribute Name** ဖြစ်ပါတယ်။
- `"https://www.example.com"` က **Attribute Value** ဖြစ်ပါတယ်။

### အသုံးများတဲ့ HTML Attribute များ
HTML မှာ အသုံးများတဲ့ Attribute တချို့ကို အောက်မှာ ဖော်ပြပေးထားပါတယ်။

##### 1. `id` Attribute
Element တစ်ခုကို သီးသန့် အမှတ်အသား (Unique Identifier) ပေးဖို့ သုံးပါတယ်။ CSS ဒါမှမဟုတ် JavaScript နဲ့ အဲ့ဒီ Element ကို သီးသန့် လှမ်းခေါ်သုံးဖို့ အလွန် အသုံးဝင်ပါတယ်။

**ဥပမာ -**

```html
<h1 id="main-title">Welcome to My Website</h1>
```

##### 2. `class` Attribute
Element တစ်ခုကို CSS Class တစ်ခု သို့မဟုတ် တစ်ခုထက်ပိုပြီး သတ်မှတ်ပေးဖို့ သုံးပါတယ်။ CSS Styling တွေ ပေးတဲ့အခါ အသုံးအများဆုံး Attribute ပါ။

**ဥပမာ -**

```html
<p class="highlighted-text">This is a highlighted paragraph.</p>
```

##### 3. `src` Attribute
ပုံတွေ၊ Video တွေလိုမျိုး ပြင်ပက File တွေရဲ့ Source ကို သတ်မှတ်ပေးဖို့ သုံးပါတယ်။

**ဥပမာ -**

```html
<img src="image.jpg" alt="A beautiful scenery">
```
##### 4. `alt` Attribute
ပုံတွေနဲ့ တွဲသုံးပြီး၊ Accessibility အတွက် ပုံရဲ့ အကြောင်းအရာကို စာသားနဲ့ အစားထိုး ဖော်ပြပေးဖို့ သုံးပါတယ်။

**ဥပမာ -**

```html
<img src="logo.png" alt="Website Logo">
```
#### 5. `href` Attribute
`<a>` (anchor) tag မှာသုံးပြီး၊ အဲ့ဒီ Link ကို နှိပ်လိုက်ရင် ဘယ် URL ကို သွားရမလဲဆိုတာ သတ်မှတ်ပေးပါတယ်။

**ဥပမာ -**

```html
<a href="https://www.example.com">Visit Example</a>
```

##### 6. `target` Attribute
`<a>` tag နဲ့ တွဲသုံးပြီး နှိပ်လိုက်တဲ့ Link ကို ဘယ်နေရာမှာ ဖွင့်မလဲဆိုတာ သတ်မှတ်ပေးပါတယ်။

- `"_blank"` က Link ကို Browser Tab အသစ်မှာ သွားဖွင့်ပေးပါတယ်။

- `"_self"` က Link ကို လက်ရှိ Tab မှာပဲ ဖွင့်ပေးပါတယ် (Default ပါ)။

**ဥပမာ -**

```html
<a href="https://www.example.com" target="_blank">Open in new tab</a>
```
