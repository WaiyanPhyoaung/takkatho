### `<a>` Tag ဖြင့် Link များ ဖန်တီးခြင်း

Link တွေက user တွေကို တခြားစာမျက်နှာတွေဆီ၊ တခြား website တွေဆီ၊ ဒါမှမဟုတ် စာမျက်နှာတစ်ခုတည်းရဲ့ မတူညီတဲ့ အပိုင်းတွေဆီကို သွားလာနိုင်အောင် ကူညီပေးပါတယ်။ HTML မှာ link တွေကို `<a>` tag (anchor လို့ အတိုကောက်ခေါ်ပါတယ်) ကိုသုံးပြီး ဖန်တီးပါတယ်။

**အခြေခံ Syntax:**

<Code title="HTML" code={aTag} lang="html" />

    - `<a>`: Link ဖန်တီးဖို့သုံးတဲ့ anchor tag ပါ။

    - `href`: Link က ဘယ် URL ကို ညွှန်းသလဲဆိုတာ သတ်မှတ်ပေးတဲ့ attribute ပါ။

    - Link text: User တွေမြင်ရပြီး နှိပ်ရမယ့် မြင်သာတဲ့ စာသားပါ။

#### Internal Link ဥပမာ (ကိုယ့် website ထဲက စာမျက်နှာတစ်ခုကို ချိတ်ဆက်ခြင်း):

<Code title="HTML" code={aboutUsLink} lang="html" />

### External Link ဥပမာ (ပြင်ပ website တစ်ခုကို ချိတ်ဆက်ခြင်း):

<Code title="HTML" code={googleLink} lang="html" />

##### အကြံပြုချက်များ -

    - Link ကို tab အသစ်မှာ ဖွင့်ချင်ရင် `target="_blank"` attribute ကို ထည့်ပေးပါ-
    <Code title="HTML" code={targetBlank} lang="html" />


    - User တွေရော SEO အတွက်ပါ ပိုအထောက်အကူဖြစ်အောင် "click here" ဆိုတာမျိုးအစား၊ link က ဘာအကြောင်းလဲဆိုတာကို ဖော်ပြတဲ့ စာသားကို သုံးပါ။

### စာမျက်နှာတစ်ခုတည်းမှ တခြားအပိုင်းများသို့ Link ချိတ်ခြင်း (Anchors)

`id` attribute တစ်ခုကိုသုံးပြီး စာမျက်နှာတစ်ခုတည်းပေါ်က သီးသန့်နေရာတစ်ခုကို link ချိတ်လို့ရပါတယ်။

    - **အဆင့် ၁ - သင်သွားချင်တဲ့ အပိုင်းကို ID တစ်ခုပေးပါ။**
    <Code title="HTML" lang="html" code={headingTag} />

    - **အဆင့် ၂ - အဲဒီ section ကို link ချိတ်ပါ။**
    <Code title="HTML" lang="html" code={contactLink} />

ID ကို link လုပ်တဲ့အခါ ရှေ့မှာ `#` သင်္ကေတ ထည့်ဖို့လိုပါတယ်။ User က link ကို နှိပ်လိုက်တဲ့အခါ၊ စာမျက်နှာက ID တူတဲ့ section ဆီကို ရောက်သွားပါလိမ့်မယ်။
