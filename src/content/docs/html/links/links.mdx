### Link ဆိုတာ ဘာလဲ
link (Hyperlink လို့လည်း ခေါ်တယ်) ဆိုတာ Website တစ်ခုမှာ စာမျက်နှာတစ်ခုကနေ နောက်တစ်ခုကို၊ ဒါမှမဟုတ် အပိုင်းတစ်ခုကနေ နောက်တစ်ပိုင်းကို ကူးပြောင်းသွားလာနိုင်အောင် လုပ်ပေးတဲ့အရာ ဖြစ်ပါတယ်။ လင့်ခ်တွေကို `<a>` (anchor) tag သုံးပြီး ဖန်တီးပါတယ်။

#### link တစ်ခုရဲ့ အခြေခံတည်ဆောက်ပုံ

<Code code={aTag} title="HTML" lang='html' />

- `<a>` tag ကို link တစ်ခု ပြုလုပ်ဖို့ သုံးပါတယ်။

- href attribute က link ကို နှိပ်လိုက်ရင် ဘယ်နေရာကို ရောက်သွားမလဲဆိုတဲ့ လိပ်စာ (URL) ကို သတ်မှတ်ပေးပါတယ်။

- "Visit Example Website" ဆိုတဲ့ စာသားကတော့ သုံးစွဲသူ မြင်ရမယ့်၊ နှိပ်ရမယ့် စာသားဖြစ်ပါတယ်။

#### links တွေ ဘယ်လိုအလုပ်လုပ်သလဲ

သုံးစွဲသူတစ်ယောက်က link တစ်ခုကို နှိပ်လိုက်တဲ့အခါ သူ့ရဲ့ browser ဟာ href ထဲမှာ သတ်မှတ်ထားတဲ့ နေရာ (တခြား Website၊ တခြားစာမျက်နှာ၊ ဒါမှမဟုတ် လက်ရှိစာမျက်နှာရဲ့ အခြားတစ်နေရာ) ကို ရောက်သွားပါတယ်။

### Absolute URL နဲ့ Relative URL
Links အမျိုးအစား နှစ်မျိုးရှိပါတယ်။ အဲဒါတွေကတော့ Absolute URL နဲ့ Relative URL တို့ ဖြစ်ပါတယ်။

#### Absolute URL (လိပ်စာ အပြည့်အစုံ)

Absolute URL ဆိုတာ domain name အပါအဝင် Website လိပ်စာ အပြည့်အစုံကို ဆိုလိုပါတယ်။

ဥပမာ:

HTML

<a href="https://www.example.com">Visit Example</a>

#### Relative URL (ကိုယ့် Website အတွင်းက လိပ်စာ)

Relative URL ကတော့ ပိုတိုပြီး ကိုယ့် Website ထဲမှာရှိတဲ့ တခြားစာမျက်နှာတွေကို ချိတ်ဆက်ဖို့ သုံးပါတယ်။

ဥပမာ:

HTML

<a href="about.html">About Us</a>
ဒီလင့်ခ်က လက်ရှိ folder ထဲမှာရှိတဲ့ about.html ဆိုတဲ့ file ကို ညွှန်းပါတယ်။

### links တွေကို Tab အသစ်မှာ ဖွင့်ခြင်း
ပုံမှန်အားဖြင့် လင့်ခ်တစ်ခုကို နှိပ်လိုက်ရင် လက်ရှိ browser tab မှာပဲ ပွင့်ပါတယ်။ ဒါပေမဲ့ တစ်ခါတလေမှာ လင့်ခ်ကို tab အသစ်မှာ သီးသန့်သွားပွင့်စေချင်တာမျိုး ရှိနိုင်ပါတယ်။ အဲဒီလိုလုပ်ဖို့ target="_blank" attribute ကို သုံးနိုင်ပါတယ်။

ဥပမာ:

HTML

<a href="https://www.example.com" target="_blank">Example ကို tab အသစ်မှာ ဖွင့်ပါ</a>
target="_blank" ဆိုတာက browser ကို လက်ရှိစာမျက်နှာကနေ မထွက်သွားဘဲ လင့်ခ်ကို tab အသစ်မှာ ဖွင့်ပေးဖို့ ပြောလိုက်တာပါ။

ဒါကို ဘာကြောင့်သုံးသင့်လဲ။

ကိုယ့် Website ကနေ တခြား Website တွေကို လင့်ခ်ချိတ်ပေးတဲ့အခါမျိုးမှာ သုံးရတာ အဆင်ပြေပါတယ်။ ဒါမှ သုံးစွဲသူက ကိုယ့် Website ကနေ လုံးဝထွက်သွားမှာ မဟုတ်ပါဘူး။

၄.၄ စာမျက်နှာတစ်ခုတည်းမှာ အပိုင်းအလိုက် လင့်ခ်ချိတ်ခြင်း
တစ်ခါတလေမှာ စာမျက်နှာက ရှည်နေတဲ့အခါ၊ ဒါမှမဟုတ် single-page website တွေမှာ စာမျက်နှာရဲ့ အပေါ်ပိုင်းက လင့်ခ်ကို နှိပ်လိုက်ရင် အောက်ဘက်က သက်ဆိုင်ရာ အပိုင်းကို တန်းရောက်သွားစေချင်တာမျိုး ရှိပါတယ်။

ဘယ်လိုလုပ်ရမလဲ။

အရင်ဆုံး ကိုယ်ရောက်စေချင်တဲ့ အပိုင်းကို id တစ်ခု ပေးထားရပါမယ်။

ပြီးရင် အဲဒီ id ကို ရည်ညွှန်းတဲ့ လင့်ခ်တစ်ခု ဖန်တီးရပါမယ်။

ဥပမာ:

HTML

<a href="#contact">ဆက်သွယ်ရန် အပိုင်းသို့ သွားရန်</a>

<h2 id="contact">ဆက်သွယ်ရန်</h2>
<p>Email us at info@example.com</p>
ဒီမှာ href="#contact" ဆိုတာက ဒီစာမျက်နှာထဲမှာ id="contact" လို့ နာမည်ပေးထားတဲ့ အပိုင်းကို သွားဖို့ ချိတ်ဆက်ပေးလိုက်တာပါ။

ဒါကို ဘာကြောင့်သုံးသင့်လဲ။

သုံးစွဲသူတွေက သူတို့ စိတ်ဝင်စားတဲ့ အပိုင်းကို တန်းပြီး ခုန်ကြည့်နိုင်တဲ့အတွက် Website ကို သုံးရတာ ပိုအဆင်ပြေစေပါတယ်။
