<img src="flower.jpg" alt="A beautiful red flower">

<a href="https://www.example.com">
    <img src="button.jpg" alt="Click to visit Example">

<Aside title="ရည်ရွယ်ချက်">
   Webpage ထဲကို ပုံတွေ ဘယ်လိုထည့်မလဲ၊ ပြီးတော့ မရှိမဖြစ်လိုအပ်တဲ့ src နဲ့ alt attribute တွေကို မှန်မှန်ကန်ကန် ဘယ်လိုသုံးရမလဲဆိုတာ လေ့လာဖို့ ဖြစ်ပါတယ်။
</Aside>

ပုံတွေက ခေတ်မီ web design ရဲ့ အရေးအကြီးဆုံး အစိတ်အပိုင်းတွေထဲမှာ ပါပါတယ်။ သူတို့က စာသားတွေချည်းပဲထက်  ပြောချင်တဲ့အချက်တွေကို ပုံနဲ့မြင်သာအောင် ပြသပေးနိုင်ပါတယ်။

ပုံတွေက webpage တစ်ခုကို ပိုပြီး စိတ်ဝင်စားဖို့ကောင်းအောင်၊ ကြည့်လို့လှအောင် လုပ်ပေးပါတယ်။ HTML မှာ ပုံတွေကို `<img>` tag သုံးပြီး ထည့်သွင်းရပါတယ်။ ဒီ tag က **self-closing** ဖြစ်တယ်၊ ဆိုလိုတာက သူ့မှာ အပိတ် tag `</img>` ပြန်ထည့်စရာမလိုပါဘူး။


**အခြေခံ Syntax:**

<Code title="HTML" lang="html" code={imgTag} />

    - **src:** ပုံရဲ့ source (ဖိုင်တည်နေရာ ဒါမှမဟုတ် URL)။

    - **alt:** ပုံက မပေါ်လာခဲ့ရင် ပုံအကြောင်း ရှင်းပြပေးတဲ့ စာသား (alternative text) ပါ။ ဒါက အမြင်အာရုံ ချို့ယွင်းနေတဲ့ user တွေအတွက်ရော SEO အတွက်ပါ အထောက်အကူဖြစ်ပါတယ်။

**ဥပမာ -**

<Code title="HTML" lang="html" code={altText} />

**အကြံပြုချက်များ -**

    - alt text ကို အဓိပ္ပာယ်ရှိတဲ့ စာသားအဖြစ် အမြဲသုံးပါ။

    - သင့် webpage load ဖြစ်တာမြန်အောင် ပုံတွေကို size သေးတာ သုံးပါ။

    - အကြောင်းအရာနဲ့ သက်ဆိုင်တဲ့ ပုံတွေကိုပဲ သုံးပါ။

### ပုံများကို Link ချိတ်ခြင်း
ပုံတစ်ပုံကို `<a>` tag တစ်ခုအတွင်းမှာ ထည့်ရေးခြင်းအားဖြင့် နှိပ်လို့ရအောင် လုပ်နိုင်ပါတယ်-

<Code title="HTML" lang="html" code={imgLink} />

အခုဆိုရင် user တွေက ပုံကို နှိပ်လိုက်တဲ့အခါ၊ သတ်မှတ်ထားတဲ့ URL ဆီကို ရောက်သွားပါလိမ့်မယ်။

### အသုံးများသော Attributes များ
    - `width` and `height`: ပုံရဲ့ အရွယ်အစား ကို ထိန်းချုပ်‌ပေးတယ်။
    <Code title="HTML" lang="html" code={widthAndHeight} />

    - `title`: User က ပုံပေါ်မှာ mouse တင်လိုက်တဲ့အခါ အချက်အလက်ကို ပြပေးတယ်။
    <Code title="HTML" lang="html" code={titleAttr} />

> **အကြံပြုချက်:** ပိုပြီး ပြောင်းလွယ်ပြင်လွယ်ရှိအောင်၊ အဆင့်မြင့်တဲ့ styling တွေအတွက် HTML attribute တွေအစား CSS ကို သုံးသင့်ပါတယ်။

<Aside title="အကျဉ်းချုပ်" type="tip">
- **Links** တွေက user တွေကို စာမျက်နှာတွေ ဒါမှမဟုတ် အပိုင်းတွေကြားမှာ သွားလာနိုင်အောင် ကူညီပေးပြီး အသုံးပြုရလွယ်ကူစေပါတယ်။

- **Images** တွေက webpages တွေကို ကြည့်လို့လှပြီး စိတ်ဝင်စားစရာကောင်းအောင် လုပ်ပေးပါတယ်။

- **Alt text** က accessibility (လူတိုင်းအသုံးပြုနိုင်မှု) နဲ့ SEO အတွက် အရေးကြီးပါတယ်။

- **Clickable images** တွေက links နဲ့ images တွေကို ပေါင်းစပ်ပြီး interactive element တွေ ဖန်တီးပေးပါတယ်။

-  ပိုကောင်းတဲ့ user experience အတွက် အဓိပ္ပာယ်ရှိတဲ့ စာသားနဲ့ ပုံဖော်ပြချက်တွေကို အမြဲသုံးပါ။

</Aside>
