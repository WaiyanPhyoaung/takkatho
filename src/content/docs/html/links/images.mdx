ဒီအခန်းမှာတော့ ပုံတွေကို နှိပ်လို့ရတဲ့ Link တွေအဖြစ် ဘယ်လိုသုံးရမလဲ၊ ပြီးတော့ Link တွေ ရေးတဲ့အခါ လိုက်နာသင့်တဲ့ အကောင်းဆုံး လက်တွေ့အသုံးချနည်း (Best Practices) တွေကို သင်ယူရမှာပါ ခင်ဗျာ။
</Aside>

Website တွေမှာ Link လုပ်တဲ့အခါ စာသားတွေကိုချည်းပဲ သုံးရမယ်လို့ မဆိုလိုပါဘူး။ ပုံတွေကိုလည်း နှိပ်လို့ရတဲ့ Link တွေအဖြစ် လှလှပပ သုံးနိုင်ပါတယ်။ အထူးသဖြင့် Logo တွေ၊ ကြော်ငြာပုံတွေမှာ အရမ်းကို အသုံးဝင်ပါတယ်။

ဥပမာ -

<Code title="HTML" lang="html" code={imgLink} />

ဒီမှာတော့ `<img>` tag ကို `<a>` tag နဲ့ အုပ်ထားတဲ့အတွက် ပုံကို နှိပ်လို့ရတဲ့ Link တစ်ခု ဖြစ်သွားပါတယ်။ `alt attribute` ကတော့ ပုံက ဘာပုံလဲဆိုတာကို ရှင်းပြထားတာဖြစ်လို့ မဖြစ်မနေ ထည့်ပေးသင့်ပါတယ်။

#### ဒါကို ဘာကြောင့်သုံးသင့်လဲ။

Website ရဲ့ Logo ကို နှိပ်လိုက်ရင် ပင်မစာမျက်နှာ (Homepage) ကို ပြန်ရောက်သွားတာမျိုး Link လုပ်ပေးခြင်း၊ Content တွေကို ပုံတွေနဲ့ ပြထားပြီး နှိပ်လိုက်ရင် Details page ကိုရောက်သွားတာမျိုးတွေ လုပ်နိုင်ပါတယ်။

### Link တွေအတွက် လိုက်နာသင့်တဲ့ အချက်များ
ခင်ဗျားရဲ့ Website က Link တွေကို ပိုထိရောက်ပြီး လူတိုင်းအဆင်ပြေအောင် သုံးနိုင်ဖို့အတွက် လိုက်နာသင့်တဲ့ အချက်တစ်ချို့ ရှိပါတယ်။

- ##### Link စာသားကို အဓိပ္ပာယ်ရှိအောင် ရေးပါ
    "Click here" လို့ ရေးမယ့်အစား "About Us" ဆိုတာမျိုး ရှင်းလင်းတဲ့ စာသားကို သုံးပါ။
    ဥပမာ: `<a href="about.html">About Us</a>`

- ##### ပုံတွေမှာ alt Text သုံးပါ
    အထူးသဖြင့် Link လုပ်ထားတဲ့ ပုံတွေမှာ **alt attribute** ကို အမြဲသုံးပေးပါ။ ဒါမှ အမြင်အာရုံ မသန်စွမ်းသူတွေအတွက် အထောက်အကူ ဖြစ်စေပါတယ်။

- ##### Link တိုင်းကို Tab အသစ်မှာ မဖွင့်ပါနဲ့
    `target="_blank"` ကို လိုအပ်မှပဲ သုံးပါ။ ခင်ဗျားရဲ့ Website ကနေ တခြား Website ကို သွားတဲ့ Link တွေလောက်မှာပဲ သုံးတာ ပိုကောင်းပါတယ်။

- ##### Link တွေကို ပုံမှန်စစ်ဆေးပါ
    Link တွေက အလုပ်မလုပ်တော့ရင် (Broken Links) သုံးစွဲသူအတွက် အဆင်မပြေဖြစ်စေသလို၊ ခင်ဗျားရဲ့ Website ရဲ့ SEO ကိုလည်း ထိခိုက်စေတဲ့အတွက် အလုပ်လုပ်/မလုပ် အမြဲတမ်း စစ်ဆေးပေးသင့်ပါတယ်။


<Aside title="အကျဉ်းချုပ်" type='tip'>
- Link တွေကို `<a>` tag နဲ့ `href attribute` သုံးပြီး ဖန်တီးပါတယ်။

- Absolute URL တွေက တခြား Website တွေကို ချိတ်ပြီး၊ Relative URL တွေက ခင်ဗျားရဲ့ Website ထဲက Page တွေကို ချိတ်ပါတယ်။

- `target="_blank"` ကို သုံးပြီး Link တွေကို Tab အသစ်မှာ ဖွင့်နိုင်ပါတယ်။

- `href="#id"` ကို သုံးပြီး စာမျက်နှာတစ်ခုတည်းမှာပဲ အပိုင်းအလိုက် ခုန်ကူးနိုင်ပါတယ်။

- ပုံတွေကို `<a>` tag နဲ့ အုပ်ပြီး နှိပ်လို့ရတဲ့ Link တွေအဖြစ် လုပ်နိုင်ပါတယ်။

- Link တွေအတွက် အကောင်းဆုံးကတော့ အဓိပ္ပာယ်ရှိတဲ့ စာသားသုံးတာ၊ ပုံတွေမှာ **alt text** ထည့်တာ၊ အလုပ်မလုပ်တဲ့ Link တွေ မရှိအောင် ပုံမှန်စစ်ဆေးတာတွေပဲ ဖြစ်ပါတယ်။
</Aside>
