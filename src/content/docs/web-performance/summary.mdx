- **ဘာကြောင့်အရေးကြီးသလဲ -** Performance သည် နည်းပညာဆိုင်ရာ metric တစ်ခုမျှသာမဟုတ်ဘဲ၊ **user happiness, business success, SEO, နှင့် accessibility** တို့ကို တိုက်ရိုက်သက်ရောက်မှုရှိသည်။
- **ဘယ်လိုတိုင်းတာမလဲ -** အဓိကကျသော metric များ (LCP, INP, CLS) နှင့် Debug လုပ်ရန် tool များ (Lighthouse, PageSpeed Insights, DevTools) ကို lab နှင့် field data နှစ်မျိုးလုံးဖြင့် အသုံးပြုတတ်သွားပြီဖြစ်သည်။
- **ဘယ်လို Optimize လုပ်မလဲ -** asset များ (Image၊ Fonts)၊ code (JS, CSS)၊ နှင့် server/delivery နည်းလမ်းများ (caching, CDNs) ကို ကောင်းအောင်လုပ်ရန် အစွမ်းထက်သော နည်းဗျူဟာများစွာကို သိရှိသွားပြီးဖြစ်သည်။

### ရှေ့ဆက်ရမည့်ခရီး

> ကျွမ်းကျင်သူတစ်ဦးဖြစ်လာရန် အကောင်းဆုံးနည်းလမ်းမှာ လက်တွေ့လေ့ကျင့်ခြင်းဖြစ်သည်။

 - **အားလုံးကို စစ်ဆေးလေ့လာပါ -** သင်၏ကိုယ်ပိုင် project များ၊ သင်အကြိုက်ဆုံး website များ၊ နှင့် ပြိုင်ဘက်များ၏ website များကို performance audit လုပ်ကြည့်ပါ။ သင် ဘာတွေရှာဖွေတွေ့ရှိနိုင်သည်ကို ကြည့်ပါ။
- **အချက်တစ်ချက်ကို လက်တွေ့အသုံးချကြည့်ပါ -** ဒီ Course မှ optimization အသေးစားတစ်ခုကို ရွေးချယ်ပြီး တစ်ပတ်အတွင်း project တစ်ခုတွင် လက်တွေ့အသုံးချကြည့်ပါ။
- **အမြဲတမ်း စူးစမ်းလေ့လာပါ -** ဖော်ပြခဲ့သော resource များကို follow လုပ်ပါ။ web performance လောကသည် အမြဲတမ်းပြောင်းလဲနေသောကြောင့် စိတ်လှုပ်ရှားဖွယ်ကောင်းသော နယ်ပယ်တစ်ခုဖြစ်သည်။
- **သင်ယူသိရှိလာသည်များကို ပြန်လည်မျှဝေပါ -** သင်၏ team သို့မဟုတ် community တွင် performance ကောင်းအောင် ပြောဆိုခြင်းက web ကို ပိုကောင်းသောနေရာတစ်ခုဖြစ်အောင် ကူညီပေးပါသည်။
