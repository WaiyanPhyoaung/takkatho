<Aside title="ရည်ရွယ်ချက်">
 Web Performance က ငွေကြေးနဲ့ လုပ်ငန်းအတွက် ဘယ်လောက် အကျိုးရှိလဲဆိုတဲ့ အချက်အလက်တွေ သိရှိစေဖို့ ဖြစ်ပါတယ်။ 
</Aside>

### မြန်နှုန်းရဲ့ ကြီးမားတဲ့ သက်ရောက်မှု - ကုမ္ပဏီကြီးများရဲ့ ဖြစ်ရပ်မှန်များ 🏆
- **Amazon:** 100 ms (0.1 second) လောက် နှေးတာနဲ့ ရောင်းအား 1% လျော့ကျနိုင်တယ်ဆိုတာ သူတို့ တွေ့ရှိခဲ့ပါတယ်။
- **Google:** search results တက်တာ 0.5 second နှေးသွားရင် လာရောက်လေ့လာသူ (traffic) 20% လျော့ကျသွားတယ်လို့ တွေ့ရှိခဲ့ပါတယ်။
- **Walmart:** website တက်လာချိန် 1 second မြန်လာတိုင်း ရောင်းအား (conversions) 2% အထိ တိုးတက်လာတယ်လို့ ဖော်ပြခဲ့ပါတယ်။
- **BBC:** သူတို့ website တက်ဖို့ 1 second ပိုကြာလေ၊ သုံးစွဲသူ 10% ထပ်မံ ဆုံးရှုံးလေပဲလို့ တွေ့ရှိခဲ့ပါတယ်။

> ဒီဥပမာတွေက Website စွမ်းဆောင်ရည်မှာ စက္ကန့်ရဲ့ အပိုင်းအစလေးတွေ တိုးတက်တာကတောင် ဝင်ငွေ၊ သုံးစွဲသူတွေရဲ့ အသုံးပြုမှုနဲ့ လာရောက်လေ့လာမှု (traffic) တွေအပေါ် သိသိသာသာ အပြုသဘောဆောင်တဲ့ သက်ရောက်မှုတွေ ရှိနိုင်တယ်ဆိုတာ ပြသနေပါတယ်။

<Aside title='အနှစ်ချုပ်'>
### Website မြန်ဆန်စေခြင်းက ငွေကြေးအမြတ်အစွန်း ရစေပုံ

Website မြန်ဆန်အောင် လုပ်တာဟာ ကုန်ကျစရိတ်တစ်ခု မဟုတ်ပါဘူး။ ဒါက ပိုက်ဆံပြန်ရတဲ့ ရင်းနှီးမြှုပ်နှံမှုတစ်ခုပါ။

- Website မြန်ရင် လူတွေ ထွက်မသွားတော့ဘူး။ ဒါကြောင့် Users တွေ ပိုများလာတယ်။
- ဆိုဒ်က မြန်ပြီး သုံးရလွယ်ရင် လူတွေက ပိုပြီး ဝယ်ကြတယ်၊ အကောင့်ဖွင့်ကြတယ်၊ ဒေါင်းလုဒ်လုပ်ကြတယ်။ ဒါက ဝင်ငွေတိုက်ရိုက် တိုးစေတယ်။
- ဆိုက်မြန်ရင် Google မှာ အဆင့်ကောင်းလာပြီး၊ ကြော်ငြာစရိတ် သက်သာစေတယ်။
</Aside>
