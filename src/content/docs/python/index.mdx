### Python ဆိုတာ ဘာလဲ?

Python ဆိုတာ ဖတ်ရလွယ်ကူပြီး ရေးရလွယ်ကူတဲ့ Programming Language တစ်ခုပါ။ လူတွေ နားလည်တဲ့ အင်္ဂလိပ်စာနဲ့ အတော်လေး ဆင်တူတာမို့လို့ ခင်ဗျားတို့အတွက် လေ့လာရတာ အဆင်ပြေစေမှာပါ။

### ဘာလို့ Python ကို လေ့လာသင့်တာလဲ?

- **လွယ်ကူတယ်:** Code တွေက ရှင်းလင်းပြီး ဖတ်ရလွယ်ပါတယ်။ Beginner တွေအတွက် အသင့်တော်ဆုံး Language တစ်ခုပါပဲ။
- **နေရာတိုင်းမှာ သုံးတယ်:** Web Development, Data Science, AI (Artificial Intelligence), နဲ့ Automation စတဲ့ နေရာပေါင်းစုံမှာ သုံးလို့ရပါတယ်။
- **Community ကြီးတယ်:** လေ့လာရင်း အခက်အခဲရှိရင် မေးစရာ နေရာတွေ၊ အဖြေရှာစရာ နေရာတွေ အများကြီး ရှိပါတယ်။

### ဒီ Course မှာ ဘာတွေ သင်ရမလဲ?

1. **Variables & Data Types:** Data တွေကို ဘယ်လို သိမ်းမလဲ ဆိုတာကို လေ့လာပါမယ်။
2. **Control Flow:** အခြေအနေပေါ်မူတည်ပြီး အလုပ်လုပ်ခိုင်းတဲ့ If/Else တွေ၊ ထပ်ခါထပ်ခါ လုပ်ခိုင်းတဲ့ Loops တွေကို လေ့လာပါမယ်။
3. **Functions:** Code တွေကို ပြန်သုံးလို့ရအောင် ဘယ်လို စုစည်းမလဲ ဆိုတာကို ကြည့်သွားပါမယ်။
4. **Data Structures:** Data အများကြီးကို စနစ်တကျ ဘယ်လို သိမ်းမလဲ ဆိုတာကို Lists တွေ၊ Dictionaries တွေနဲ့ လေ့လာပါမယ်။
5. **Object-Oriented Programming (OOP):** Code တွေကို ပိုမို စနစ်ကျအောင် ဘယ်လို တည်ဆောက်မလဲ ဆိုတာကို လေ့လာပါမယ်။
6. **File Handling & Errors:** File တွေ ဖတ်တာ၊ ရေးတာနဲ့ Error တွေကို ဘယ်လို ဖြေရှင်းမလဲ ဆိုတာတွေကို လေ့လာသွားမှာပါ။
