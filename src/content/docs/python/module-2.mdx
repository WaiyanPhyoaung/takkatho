Program တစ်ခု ရေးတဲ့အခါ အပေါ်ကနေ အောက်ကို တစ်ကြောင်းချင်း အလုပ်လုပ်သွားတာချည်းပဲ မဟုတ်ပါဘူး။ တချို့နေရာတွေမှာ **"ဒီလိုဆိုရင် ဒါလုပ်မယ်"** ဆိုပြီး ဆုံးဖြတ်ချက်ချတာတွေ၊ **"ဒီအလုပ်ကို ၅ ကြိမ် ထပ်လုပ်မယ်"** ဆိုပြီး ခိုင်းတာတွေ လိုအပ်လာပါတယ်။ ဒါကို Control Flow လို့ ခေါ်ပါတယ်။

### Indentation (ရှေ့ကွက်လပ် ချန်ခြင်း)

Python မှာ အရေးအကြီးဆုံး အချက်တစ်ခုက **Indentation** ပါ။ တခြား Language တွေမှာ `{}` ကို သုံးပြီး Code Block ဖွဲ့ပေမယ့်၊ Python မှာတော့ **Space 4 ချက်** (သို့) **1 Tab** ချန်ပြီး ဖွဲ့ပါတယ်:

```python
if True:
    print("ဒါက if အထဲမှာ ရှိပါတယ်")  # ← 4 Space
    print("ဒါလည်း if အထဲ ပါဝင်")     # ← 4 Space
print("ဒါကတော့ if အပြင်ပြောင်းသွားပြီ")  # ← 0 Space
```

Space မချန်ရင် (သို့) မညီညာရင် `IndentationError` တက်ပါမည်။

### If / Elif / Else (အခြေအနေ စစ်ဆေးခြင်း)

<ContentImage src={controlFlowImg} alt="Control Flow Diagram" />

ဆိုင်မှာ ဈေးနှုန်းသတ်မှတ်တဲ့ပုံစံနဲ့ တူပါတယ် — "ကျပ် ၅၀၀ အောက်ဆိုရင် သေးငယ်သောထုတ်ပိုးမှုပေး၊ ၅၀၀ ကနေ ၁၀၀၀ ကြားဆိုရင် အလတ်ထုတ်ပိုး၊ ၁၀၀၀ ကျော်ဆိုရင် ကြီးမားသောထုတ်ပိုး" ဆိုတဲ့ ဆုံးဖြတ်ချက်မျိုးပါ:

```python
age = int(input("အသက် ဘယ်လောက်လဲ? "))

if age >= 18:
    print("ကားမောင်းလို့ ရပါပြီ")
elif age >= 16:
    print("ကားမောင်းသင်လို့ ရပါပြီ")
else:
    print("ကားမောင်းဖို့ အချိန်ရသေးဘူး")
```

**`if` က ဘယ်နဲ့ စစ်ဆေးသလဲ?** — Comparison Operators တွေနဲ့ စစ်ပြီး `True/False` ပြန်ပေးတာပါ:

```python
score = 75

if score >= 80:
    grade = "A"
elif score >= 70:
    grade = "B"
elif score >= 60:
    grade = "C"
else:
    grade = "F"

print(f"Grade: {grade}")  # Grade: B
```

#### Logical Operators (`and`, `or`, `not`)

```python
# and — နှစ်ခုလုံး မှန်မှ True
age = 20
has_license = True
if age >= 18 and has_license:
    print("ကားမောင်းနိုင်ပါပြီ")

# or — တစ်ခုမှန်ရုံနဲ့ True
is_admin = False
is_manager = True
if is_admin or is_manager:
    print("Dashboard ဝင်ခွင့် ရှိသည်")

# not — ပြောင်းပြန်
is_logged_in = False
if not is_logged_in:
    print("ဦးစွာ Login လုပ်ပါ")
```

#### in — Membership စစ်ဆေးခြင်း

`in` ကို သုံးပြီး List (သို့) String ထဲမှာ ရှိမရှိ စစ်ဆေးနိုင်ပါတယ်:

```python
fruits = ["Apple", "Banana", "Orange"]

if "Apple" in fruits:
    print("Apple ရှိပါတယ်")    # ← ဒါ ပြပါမည်

if "Mango" not in fruits:
    print("Mango မရှိဘူး")     # ← ဒါ ပြပါမည်

# String ထဲမှာ စစ်ဆေးလည်း ရသည်
email = "user@gmail.com"
if "@" in email:
    print("Valid email format")
```

#### Ternary Expression (တစ်ကြောင်းတည်း if)

တိုတောင်းသော if/else ကို တစ်ကြောင်းတည်းနဲ့ ရေးနိုင်ပါတယ်:

```python
# ပုံစံ: value_if_true if condition else value_if_false

age = 20
status = "လူကြီး" if age >= 18 else "ကလေး"
print(status)   # လူကြီး

# ပိုကြည်လင်သော ဥပမာ
score = 75
result = "အောင်" if score >= 60 else "ကျ"
print(f"Score {score} — {result}")  # Score 75 — အောင်
```

### Loops (ထပ်ခါထပ်ခါ အလုပ်လုပ်ခြင်း)

#### For Loop — ကိန်းနံပါတ် (Range)

```python
# range(n) → 0, 1, 2, ..., n-1
for i in range(5):
    print(f"အကြိမ် {i}")
# အကြိမ် 0
# အကြိမ် 1
# ...
# အကြိမ် 4

# range(start, end, step)
for i in range(1, 10, 2):
    print(i, end=" ")   # 1 3 5 7 9

# ဂဏန်း ၁ မှ ၁၀ ပေါင်းခြင်း
total = 0
for n in range(1, 11):
    total += n
print(f"1-10 ပေါင်းရင် {total}")   # 55
```

#### For Loop — List ကို ရွှေ့ပတ်ခြင်း

```python
fruits = ["Apple", "Banana", "Orange"]

# တန်ဖိုးသာ ယူ
for fruit in fruits:
    print(fruit)

# enumerate() — Index ပါ ယူချင်ရင်
for i, fruit in enumerate(fruits, start=1):
    print(f"{i}. {fruit}")
# 1. Apple
# 2. Banana
# 3. Orange
```

<Aside type="tip" title="enumerate() ကို မမေ့ပါနဲ့">
Loop မှာ Index (နံပါတ်) ပါ လိုချင်ရင် `enumerate()` ကိုသာ သုံးပါ။ `range(len(fruits))` ဟာ Pythonic မဟုတ်ပြီး ဖတ်ရ ခက်ပါတယ်။
</Aside>

#### Nested Loops — Loop ထဲ Loop

```python
# မြှောက်ဇယား (Multiplication Table)
for i in range(1, 4):
    for j in range(1, 4):
        print(f"{i} x {j} = {i*j}", end="    ")
    print()   # ကြောင်းအသစ် ဆင်း

# 1 x 1 = 1    1 x 2 = 2    1 x 3 = 3
# 2 x 1 = 2    2 x 2 = 4    2 x 3 = 6
# 3 x 1 = 3    3 x 2 = 6    3 x 3 = 9
```

#### While Loop — အခြေအနေ မှန်သရွေ့ ဆက်လုပ်ခြင်း

```python
count = 1

while count <= 5:
    print(f"ကြိမ် {count}")
    count += 1    # ← ဒါကို မမေ့ပါနဲ့! ရှိမရင် အဆုံးမရှိ Loop ဖြစ်မည်

# ကြိမ် 1
# ကြိမ် 2
# ...
# ကြိမ် 5
```

<Aside type="caution" title="Infinite Loop ကို ကြောက်ပါ">
`while` Loop မှာ ထွက်နိုင်တဲ့ နည်းလမ်း မပါရင် **Infinite Loop** (အဆုံးမရှိ Loop) ဖြစ်တတ်ပါတယ်:

```python
# ❌ ဒါမဆို — count မပြောင်းဘူး ဆိုရင်
count = 1
while count <= 5:
    print(count)
    # count += 1   ← ဒါ မပါလို့ 1, 1, 1, ... မဆုံးဘဲ ဆက်ထုတ်နေမည်
```

Infinite Loop ကို ရပ်ဖို့ **Ctrl + C** နှိပ်ပါ။
</Aside>

### Break, Continue, Pass

#### break — Loop ရပ်ခြင်း

```python
# ပထမဆုံး စုံဂဏန်း ရှာပြီး ရပ်ခြင်း
for n in range(1, 20):
    if n % 2 == 0:
        print(f"ပထမဆုံး စုံဂဏန်း: {n}")
        break   # ← Loop ကြီးတစ်ခုလုံး ရပ်သွားမည်
# ပထမဆုံး စုံဂဏန်း: 2
```

#### continue — ကျော်ပြီး ဆက်ခြင်း

```python
# 1-10 မှ မပုံကိန်းများပဲ ထုတ်ဖော်ပြခြင်း
for n in range(1, 11):
    if n % 2 == 0:
        continue   # ← စုံဂဏန်းကို ကျော်သွား
    print(n, end=" ")
# 1 3 5 7 9
```

#### pass — ထည့်ပေးထားဖို့ နောက်မှ ဖြည့်မည်

```python
# Syntax error မဖြစ်ဖို့ ခေတ္တ ထည့်ပေးထားသော placeholder
for item in range(5):
    pass   # ← ဘာမှ မလုပ်ဘဲ ကျော်သွားသည် (Syntax Error မတက်)

if score >= 60:
    pass   # TODO: Handle passing case later
```

### ဖြစ်တတ်သော အမှားများ

```python
# ၁. if ရဲ့ Condition မှာ = (Assign) ကို == (Compare) နဲ့ မရောဆိုပါ
x = 5
if x == 5:     # ← Compare (မှန်)
    print("Five")

# ၂. Indentation ညီညာမှ ဖြစ်မည်
if True:
    print("Line 1")
   print("Line 2")  # IndentationError! (3 Space ဖြင့် ရေးထား)

# ၃. elif ကို else if လို့ မရေးနိုင်
# if condition:
# else if condition:   # ← SyntaxError!
# elif condition:      # ← မှန်သည်
```
