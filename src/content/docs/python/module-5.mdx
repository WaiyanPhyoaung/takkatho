Object-Oriented Programming (OOP) ဆိုတာ Code တွေကို **"အရာဝတ္ထု (Objects)"** တွေ အနေနဲ့ မြင်ပြီး ရေးတဲ့ ပုံစံပါ။ လက်တွေ့ကမ္ဘာက အရာတွေ (ကား၊ ဘဏ်အကောင့်၊ User Account) ကို Code အဖြစ် ပြောင်းလဲရေးသားတဲ့အခါ OOP ဟာ Code ကို ပိုမို **ကြည်လင်** ပြီး **ပြန်သုံးနိုင်** စေပါတယ်။

### Class ဘယ်လို တည်ဆောက်မလဲ?

```python
class Dog:
    # __init__ ← Constructor: Object ဆောက်တဲ့အခါ အလိုအလျောက် Run မည်
    # self ← "ကိုယ်ကိုယ်တိုင်" ကို ကိုယ်စားပြု — ဒါကို မမေ့ပါနဲ့
    def __init__(self, name, breed):
        self.name  = name     # Instance Attribute
        self.breed = breed    # Instance Attribute
    
    def bark(self):
        print(f"{self.name} က ဝုတ် ဝုတ် လို့ ဟောင်နေပါတယ်")
```

<Aside type="caution" title="self ကို မမေ့ပါနဲ့">
Method Parameter မှာ `self` မပါရင် Python Error တက်ပါမည်:

```python
class Dog:
    def bark():           # ❌ self မပါ
        print("bark!")

d = Dog()
d.bark()    # TypeError: bark() takes 0 positional arguments but 1 was given
```

`self` ဟာ Python က Object ကိုယ်တိုင်ကို Argument အဖြစ် Pass ပေးနေသောကြောင့် Method ဖော်ပြချက်မှာ `self` ဦးစွာ ထည့်ရပါတယ်။
</Aside>

### Object ဖန်တီးခြင်းနှင့် အသုံးပြုခြင်း

```python
# Object ဆောက်ခြင်း
dog1 = Dog("Aung Net", "Golden Retriever")
dog2 = Dog("Phoe Wa", "Bulldog")

# Attributes ကြည့်ခြင်း
print(dog1.name)    # Aung Net
print(dog2.breed)   # Bulldog

# Methods ခေါ်ခြင်း
dog1.bark()    # Aung Net က ဝုတ် ဝုတ် လို့ ဟောင်နေပါတယ်
dog2.bark()    # Phoe Wa က ဝုတ် ဝုတ် လို့ ဟောင်နေပါတယ်

# Attribute ပြင်ခြင်း
dog1.name = "Aung Net Jr."
print(dog1.name)    # Aung Net Jr.
```

### __str__ — Object ကို ကြည်လင်စွာ ပြသခြင်း

`__str__` Method မပါရင် `print(dog1)` ဟာ ဒီလိုမျိုး ပြပါမည်:

```
<__main__.Dog object at 0x0000020B4A7A6E90>
```

`__str__` Method ထည့်လိုက်ရင် ကြည်လင်သော Output ပြနိုင်ပါမည်:

```python
class Dog:
    def __init__(self, name, breed, age):
        self.name  = name
        self.breed = breed
        self.age   = age
    
    def __str__(self):
        return f"Dog(name='{self.name}', breed='{self.breed}', age={self.age})"
    
    def bark(self):
        print(f"{self.name} က ဝုတ် ဝုတ် ဟောင်သည်!")

dog = Dog("Lucky", "Shih Tzu", 3)
print(dog)   # Dog(name='Lucky', breed='Shih Tzu', age=3)
```

### Class Attributes vs Instance Attributes

**Instance Attribute** — Object တစ်ခုချင်းစီ ကိုယ်ပိုင် Value ရှိသည် (`self.name`)

**Class Attribute** — Class ကနေ ဆောက်သော Object **အားလုံး မျှဝေ** သောတန်ဖိုး:

```python
class Dog:
    # Class Attribute — Object တိုင်းမှာ တူညီ
    species = "Canis lupus familiaris"
    count   = 0
    
    def __init__(self, name):
        self.name = name          # Instance Attribute — တစ်ကောင်ချင်းစီ ကွဲ
        Dog.count += 1            # Class Attribute ကို Class နာမည်နဲ့ ဝင်ကြည့်
    
    @classmethod
    def get_count(cls):
        return f"ခွေး {cls.count} ကောင် ဖန်တီးပြီးပါပြီ"

d1 = Dog("Lucky")
d2 = Dog("Max")
d3 = Dog("Buddy")

print(Dog.species)         # Canis lupus familiaris
print(d1.species)          # Canis lupus familiaris  (Object ကနေ ကြည့်လည်း ရ)
print(Dog.get_count())     # ခွေး 3 ကောင် ဖန်တီးပြီးပါပြီ
```

### Inheritance (အမွေဆက်ခံခြင်း)

ရှိပြီးသား Class ကို အခြေခံပြီး Child Class ဆောက်နိုင်ပါတယ်:

```python
# Parent Class
class Animal:
    def __init__(self, name, age):
        self.name = name
        self.age  = age
    
    def __str__(self):
        return f"{type(self).__name__}({self.name}, {self.age} နှစ်)"
    
    def eat(self):
        print(f"{self.name} က အစာ စားနေပါတယ်")
    
    def sleep(self):
        print(f"{self.name} က အိပ်နေပါတယ်")


# Child Class — Animal ကနေ inherit
class Cat(Animal):
    def __init__(self, name, age, indoor):
        super().__init__(name, age)   # ← Parent __init__ ခေါ်ခြင်း
        self.indoor = indoor           # Cat ကိုယ်ပိုင် Attribute
    
    def meow(self):
        print(f"{self.name} က မြောင် လို့ ဆိုနေပါတယ်")


class Dog(Animal):
    def __init__(self, name, age, breed):
        super().__init__(name, age)
        self.breed = breed
    
    def bark(self):
        print(f"{self.name} ({self.breed}) က ဟောင်နေပါတယ်!")


# Object ဆောက်ပြီး Test
cat = Cat("ကြောင်ဝါ", 2, indoor=True)
dog = Dog("ကျား", 4, "Labrador")

# Parent Methods — Child မှာ သုံးနိုင်
cat.eat()    # ကြောင်ဝါ က အစာ စားနေပါတယ်
dog.eat()    # ကျား က အစာ စားနေပါတယ်

# ကိုယ်ပိုင် Methods
cat.meow()   # ကြောင်ဝါ က မြောင် လို့ ဆိုနေပါတယ်
dog.bark()   # ကျား (Labrador) က ဟောင်နေပါတယ်!

print(cat)   # Cat(ကြောင်ဝါ, 2 နှစ်)
print(dog)   # Dog(ကျား, 4 နှစ်)
```

<Aside type="tip" title="super() ကို ဘာကြောင့် သုံးမလဲ?">
`super().__init__(name, age)` ဟာ **Parent Class ရဲ့ `__init__`** ကို ခေါ်ပေးပါတယ်။ ဒါမပါရင် Parent Class ဆောက်ပေးထားသော Attribute တွေ (name, age) ကို Child Object မှာ ရှိမနေပါဘူး:

```python
# super() မပါသောနည်း (မကောင်း)
class Cat(Animal):
    def __init__(self, name, age, indoor):
        self.name = name    # ← ဒါကို ထပ်ရေးနေရသည် (DRY မဟုတ်ဘူး)
        self.age  = age
        self.indoor = indoor

# super() ပါသောနည်း (ကောင်း)
class Cat(Animal):
    def __init__(self, name, age, indoor):
        super().__init__(name, age)   # Parent က handle လုပ်မည်
        self.indoor = indoor
```
</Aside>

### Encapsulation — Private Attributes

Python မှာ Convention အားဖြင့် ထိတွေ့ မသင့်တဲ့ Attribute တွေကို `_` (underscore) တစ်ခုနဲ့ ကြိုဘောင်ရေးပြီး ညွှန်ပြပါတယ်:

```python
class BankAccount:
    def __init__(self, owner, balance):
        self.owner    = owner       # Public  — ပြင်ကနေ ဝင်ကြည့်လို့ ရ
        self._balance = balance     # Private — ဒါကို တိုက်ရိုက် မပြင်ပါနဲ့
    
    def deposit(self, amount):
        if amount <= 0:
            raise ValueError("ထည့်ငွေ ၀ ကျော်ရမည်")
        self._balance += amount
        print(f"+{amount} ကျပ် ထည့်ပြီး — Remaining: {self._balance}")
    
    def withdraw(self, amount):
        if amount > self._balance:
            raise ValueError("လက်ကျန်ငွေ မလုံလောက်ပါ")
        self._balance -= amount
        print(f"-{amount} ကျပ် ထုတ်ပြီး — Remaining: {self._balance}")
    
    def get_balance(self):
        return self._balance

acc = BankAccount("Aung Aung", 10000)
acc.deposit(5000)    # +5000 ကျပ် ထည့်ပြီး — Remaining: 15000
acc.withdraw(3000)   # -3000 ကျပ် ထုတ်ပြီး — Remaining: 12000
print(acc.get_balance())  # 12000
```

### Myanmar Context ဥပမာ — ကျောင်းသား စီမံခန့်ခွဲမှု

```python
class Student:
    """ကျောင်းသားတစ်ယောက်ကို ကိုယ်စားပြုသော Class"""
    
    school_name = "ရန်ကုန် Technical University"  # Class Attribute
    
    def __init__(self, name, roll_no, scores):
        self.name     = name        # ကျောင်းသားနာမည်
        self.roll_no  = roll_no     # အမှတ်
        self.scores   = scores      # စာမေးပွဲ မှတ်များ
    
    def __str__(self):
        return f"Student({self.name}, Roll #{self.roll_no})"
    
    def average(self):
        """ပျမ်းမျှ Score တွက်ပေးသော Method"""
        if not self.scores:
            return 0
        return sum(self.scores) / len(self.scores)
    
    def grade(self):
        """Grade ပြန်ပေးသော Method"""
        avg = self.average()
        if avg >= 80:
            return "A"
        elif avg >= 70:
            return "B"
        elif avg >= 60:
            return "C"
        else:
            return "F"

# Object ဆောက်ခြင်း
mg_mg = Student("မောင်မောင်", 1001, [85, 72, 90])
su_su = Student("စုစု",      1002, [92, 88, 95])

print(mg_mg)                           # Student(မောင်မောင်, Roll #1001)
print(f"ပျမ်းမျှ: {mg_mg.average():.1f}")  # ပျမ်းမျှ: 82.3
print(f"Grade: {mg_mg.grade()}")       # Grade: A
print(f"ကျောင်း: {Student.school_name}")
```
