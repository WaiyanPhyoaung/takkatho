### File Handling

Python မှာ File တွေကို `open()` Function ကို သုံးပြီး ဝင်ရောက်ပါတယ်။ Mode (၃) မျိုး ရှိပါတယ်:

- `"r"` — ဖတ်ဖို့ (Read)
- `"w"` — ရေးဖို့ (Write — ရှိပြီးသားဆိုရင် ဖျက်ပြီး အသစ်ရေးမည်)
- `"a"` — ဆက်ထည့်ဖို့ (Append — အနောက်ကနေ ဆက်ရေးမည်)

<Aside type="caution" title="encoding='utf-8' — မမေ့ပါနဲ့">
Myanmar Unicode နဲ့ ရေးထားသော Text တွေဆိုရင် `encoding='utf-8'` မပါရင် ဖတ်ရတဲ့အခါ Error တက်နိုင်ပါတယ် (Windows မှာ အထူးသဖြင့်)။ **File တိုင်းမှာ encoding ကို ဖော်ပြပါ**:

```python
open("file.txt", "r", encoding="utf-8")
```
</Aside>

#### File ထဲကို ရေးခြင်း

```python
# "w" Mode — ဖိုင်ကို ဆောက်ပြီး ရေးသည် (ရှိပြီးသားဆိုရင် ဖျက်ပြီး ပြန်ရေး)
with open("student.txt", "w", encoding="utf-8") as file:
    file.write("ကျောင်းသားစာရင်း\n")
    file.write("Aung Aung — 85 မှတ်\n")
    file.write("Su Su — 92 မှတ်\n")

print("ရေးပြီးပါပြီ!")
```

**`with open(...)` ကို ဘာကြောင့် သုံးမလဲ?**
`with` Block ပြီးသွားတာနဲ့ File ကို **အလိုအလျောက် ပိတ်** (Close) ပေးပါတယ်။ ကိုယ်တိုင် `file.close()` ရိုက်စရာ မလိုပါဘူး — ပိတ်မပြီးဘဲ မနေနိုင်ပါ။

#### File ကနေ ဖတ်ခြင်း

```python
import os

# ဖိုင်ရှိမရှိ စစ်ဆေးပြီးမှ ဖတ်ပါ
if not os.path.exists("student.txt"):
    print("ဖိုင် မတွေ့ပါ!")
else:
    with open("student.txt", "r", encoding="utf-8") as file:
        # နည်း ၁: အကုန်တစ်ကြိမ်တည်း ဖတ်
        content = file.read()
        print(content)

    with open("student.txt", "r", encoding="utf-8") as file:
        # နည်း ၂: တစ်ကြောင်းချင်းစီ ဖတ်
        for line in file:
            print(line.strip())   # strip() ဖြင့် နောက်ဆုံး \n ဖြုတ်
```

#### Append — ဆက်ထည့်ခြင်း

```python
# "a" Mode — ရှိပြီးသားနောက်မှာ ဆက်ရေးပါ (မဖျက်ဘဲ)
with open("student.txt", "a", encoding="utf-8") as file:
    file.write("Kyaw Kyaw — 78 မှတ်\n")
```

### JSON File — Data ကို Structure နဲ့ သိမ်းခြင်း

JSON (JavaScript Object Notation) ဆိုတာ Data ကို **ဘာသာစကားမဆိုဖတ်နိုင်သော** Format နဲ့ သိမ်းနိုင်သော Standard ပါ — Python Dictionary ပုံစံနဲ့ တူပြီး Program ၂ ခုကြားနဲ့ Data ပြောင်းဖို့ API Response Format အဖြစ် ကမ္ဘာတစ်ဝှမ်း သုံးကြပါတယ်:

```python
import json

# Data (Python Dictionary)
student_data = {
    "name": "Aung Aung",
    "age": 20,
    "scores": [85, 92, 78],
    "active": True
}

# Dictionary → JSON File ထဲ သိမ်းခြင်း
with open("student.json", "w", encoding="utf-8") as f:
    json.dump(student_data, f, ensure_ascii=False, indent=2)
```

```json
{
  "name": "Aung Aung",
  "age": 20,
  "scores": [85, 92, 78],
  "active": true
}
```

```python
# JSON File → Python Dictionary ဖြင့် ပြန်ဖတ်ခြင်း
with open("student.json", "r", encoding="utf-8") as f:
    loaded = json.load(f)

print(loaded["name"])     # Aung Aung
print(loaded["scores"])   # [85, 92, 78]
print(type(loaded))       # <class 'dict'>
```

<Aside type="tip" title="json.dump() Params">
- `ensure_ascii=False` — Myanmar Unicode ကို မပြောင်းဘဲ ရေးနိုင်မည်
- `indent=2` — ကြည်လင်ရန် ဖော်မတ်ထည့်မည် (မပါရင် တစ်ကြောင်းတည်း ရေးမည်)
</Aside>

### Error Handling (try / except)

Code ရေးတဲ့အခါ မထင်မှတ်ဘဲ Error တွေ တက်တတ်ပါတယ်။ ဆိုင်မှာ အချိုရည်မှာတဲ့ Customer ကို "ဒီ Flavor မရှိဘူးရင် ဒါပေး" ဆိုပြီး ကြိုပြင်ဆင်ထားသလိုပဲ — try/except နဲ့ Error ဖြစ်ရင် ဘာလုပ်မလဲ ကြိုသတ်မှတ်ထားနိုင်ပါတယ်။ `try / except` ကို သုံးပြီး ထိန်းချုပ်ပါ:

```python
try:
    # Error တက်နိုင်သော Code
    result = 10 / 0
except ZeroDivisionError:
    # Error တက်ခဲ့ရင် ဒီမှာ Handle လုပ်
    print("သုညနဲ့ မစားနိုင်ပါဘူး")
```

#### Error မျိုးစုံ ကိုင်တွယ်ခြင်း

```python
def get_age():
    try:
        age = int(input("အသက် ဘယ်လောက်လဲ? "))
        result = 100 / age
        print(f"သင်ဟာ {100 - age} နှစ်အကြာမှာ ၁၀၀ ဖြည့်မှာပါ")

    except ValueError:
        # int("hello") ကဲ့သို့ ဆောင်ရွက်မရသောအခါ
        print("ကျေးဇူးပြု၍ ဂဏန်းသာ ထည့်ပါ")

    except ZeroDivisionError:
        # age = 0 ဖြင့် 100/0 ဆောင်ရွက်ချိန်
        print("အသက် 0 ဆိုတာ မဖြစ်နိုင်ပါဘူး")

    except Exception as e:
        # မထင်မဆင် တခြား Error တွေ
        print(f"အမှား ဖြစ်ပွားသည်: {e}")

    else:
        # Error မတက်ဘဲ အောင်မြင်ခဲ့ရင် ဒါ Run မည်
        print("အောင်မြင်သည်!")

    finally:
        # Error တက်တက် မတက်တက် ဒါ အမြဲ Run မည်
        print("စစ်ဆေးမှု ပြီးဆုံးပါပြီ")

get_age()
```

### raise — ကိုယ်ကိုင် Error မြှောက်ခြင်း

`raise` ကို သုံးပြီး Input မှားနေတဲ့အခါ ကိုယ်တိုင် Error တက်ခိုင်းနိုင်ပါတယ်:

```python
def set_age(age):
    if not isinstance(age, int):
        raise TypeError(f"Age ဟာ Integer ဖြစ်ရမည် — {type(age)} ပေးထားသည်")
    if age < 0:
        raise ValueError(f"Age ဟာ 0 အောက် မဖြစ်နိုင်ပါ — {age} ပေးထားသည်")
    if age > 150:
        raise ValueError(f"Age ဟာ 150 ကျော်ရင် ဖြစ်မနိုင်ပါ — {age} ပေးထားသည်")
    print(f"Age သတ်မှတ်ပြီးပါပြီ: {age}")

# မှန်ကန်သော Input
set_age(20)      # Age သတ်မှတ်ပြီးပါပြီ: 20

# မှားသော Input
try:
    set_age(-5)  # ValueError!
except ValueError as e:
    print(f"Error: {e}")
```

### ပေါင်းစပ် ဥပမာ — Student Record System

```python
import json
import os

FILE = "records.json"

def load_records():
    if not os.path.exists(FILE):
        return []
    with open(FILE, "r", encoding="utf-8") as f:
        return json.load(f)

def save_records(records):
    with open(FILE, "w", encoding="utf-8") as f:
        json.dump(records, f, ensure_ascii=False, indent=2)

def add_student(name, score):
    if not name.strip():
        raise ValueError("နာမည် ထည့်ပေးပါ")
    if not 0 <= score <= 100:
        raise ValueError("Score ဟာ 0-100 ကြားဖြစ်ရမည်")
    
    records = load_records()
    records.append({"name": name, "score": score})
    save_records(records)
    print(f"✅ {name} ကို ထည့်ပြီးပါပြီ")

try:
    add_student("Aung Aung", 85)
    add_student("Su Su", 92)
    add_student("", 70)         # ValueError!
except ValueError as e:
    print(f"❌ {e}")
```
