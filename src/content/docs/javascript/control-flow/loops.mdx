---

အလုပ်တစ်ခုတည်းကို အကြိမ်အရေအတွက် အများကြီး ထပ်ခါထပ်ခါ လုပ်ဆောင်စေချင်တဲ့အခါ **Loops** ကို အသုံးပြုရပါတယ်။ ဥပမာ - မျက်နှာပြင်ပေါ်မှာ "မင်္ဂလာပါ" ဆိုတဲ့ စာသားကို အကြိမ် ၁၀၀ လောက် ထုတ်ပြခိုင်းတာမျိုး ဖြစ်ပါတယ်။

## ၁။ For Loop (အသုံးအများဆုံး Loop)

အလုပ်တစ်ခုကို ဘယ်နှကြိမ် လုပ်ရမလဲ ဆိုတဲ့ အကြိမ်အရေအတွက် အတိအကျကို ကြိုသိထားတဲ့ အခါမျိုးမှာ အသုံးပြုပါတယ်။

```javascript
// ၁ ကနေ ၅ အထိ ထုတ်ပြပါမယ်
for (let i = 1; i <= 5; i++) {
  console.log("Number: " + i);
}
```
* `let i = 1` : စတင်မယ့် ကနဦးမှတ် (Start)
* `i <= 5` : ဘယ်အခြေအနေရောက်ရင် ရပ်တန့်မလဲ ဆိုတဲ့ သတ်မှတ်ချက် (Stop/Condition)
* `i++` : တစ်ကြိမ်ပြီးတိုင်း တစ်ဆင့်ချင်း တိုးသွားမယ် (Step)

> `i++` ဆိုတာ `i = i + 1` လို့ ရေးတာနဲ့ အတူတူပါပဲ။ လက်ရှိ `i` ရဲ့ တန်ဖိုးကို (၁) ထပ်ပေါင်းပြီး တိုးလိုက်တာ ဖြစ်ပါတယ်။

## ၂။ While Loop

ပေးထားတဲ့ အခြေအနေ (Condition) တစ်ခု မှန်ကန်နေသရွေ့ (True ဖြစ်နေသရွေ့) ဆက်တိုက် အလုပ်လုပ်နေပါမယ်။ ဘယ်အချိန်မှာ ရပ်မလဲ ဆိုတာကို အကြိမ်အရေအတွက် အတိအကျ မသိနိုင်တဲ့ အခါမျိုးမှာ အသုံးပြုပါတယ်။

```javascript
let count = 1;

while (count <= 5) {
  console.log(count);
  count++; // ဒါကိုရေးဖို့ မေ့သွားရင် Loop က ဘယ်တော့မှ မရပ်တန့်တော့ဘဲ (Infinite Loop) ဖြစ်သွားပြီး Browser ဟန်းသွားပါလိမ့်မယ်
}
```

## ၃။ Do...While Loop

Condition မှန်သည်ဖြစ်စေ၊ မှားသည်ဖြစ်စေ အနည်းဆုံး (၁) ကြိမ်တော့ သေချာပေါက် အလုပ်လုပ်စေချင်တဲ့ အခါမျိုးမှာ သုံးပါတယ်။ သူက အလုပ်ကို အရင်လုပ်ပြီးမှ Condition ကို နောက်ဆုံးမှ စစ်ဆေးတာ ဖြစ်ပါတယ်။

```javascript
let i = 100;

do {
  console.log("I run at least once!");
  i++;
} while (i < 5); // i က 5 ထက် မငယ်ပေမယ့် အပေါ်က Code က အရင်အလုပ်လုပ်သွားတဲ့အတွက် တစ်ခါတော့ သေချာပေါက် အဖြေထွက်လာပါမယ်
```

### 💡 လက်တွေ့ အသုံးချမှု ဥပမာ
လက်တွေ့ လုပ်ငန်းခွင်မှာ Loop တွေကို Data တွေ အများကြီးပါတဲ့ Array တွေနဲ့ တွဲဖက် အသုံးပြုလေ့ ရှိကြပါတယ်။

```javascript
let fruits = ["Apple", "Orange", "Mango"];

// Array ရဲ့ အရှည် (length) အတိုင်း Loop ပတ်ပြီး ထုတ်ပြခြင်း
for (let i = 0; i < fruits.length; i++) {
    console.log("I like " + fruits[i]);
}
```
