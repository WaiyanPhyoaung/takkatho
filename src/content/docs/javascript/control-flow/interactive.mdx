## လေ့ကျင့်ခန်း ၁ (Exercise 1): Number Checker

Number (ကိန်းဂဏန်း) တစ်ခု ပေးလိုက်ရင် အဲ့ဒီဂဏန်းဟာ Positive (အပေါင်းကိန်း)၊ Negative (အနှုတ်ကိန်း)၊ ဒါမှမဟုတ် Zero (သုည) ဟုတ်မဟုတ် စစ်ဆေးပေးတဲ့ Code ကို ရေးသားကြည့်ပါ။

```javascript
let number = -5;

// သင်၏ Code များကို ဒီအောက်တွင် ဆက်လက်ရေးသားပါ
if (number > 0) {
    console.log("Positive");
} // continue...
```

## လေ့ကျင့်ခန်း ၂ (Exercise 2): Even or Odd? (စုံ လား၊ မ လား)

Loop ပတ်ပြီး ၁ ကနေ ၁၀ အထိ ဂဏန်းတွေကို ထုတ်ပြပေးပါ။ ဒါပေမဲ့ ထူးခြားချက် အနေနဲ့ -
* စုံကိန်း (Even) ဖြစ်ရင် "2 is Even"
* မကိန်း (Odd) ဖြစ်ရင် "1 is Odd" ဆိုပြီး ထွက်ပေါ်လာအောင် ရေးသားရပါမယ်။

*သဲလွန်စ (Hint): `%` (Remainder operator) ကို အသုံးပြုပါ။ ဥပမာ - `num % 2 === 0` လို့ စစ်ရင် အကြွင်းသုည ဖြစ်တဲ့အတွက် စုံကိန်း (Even) ဖြစ်ပါတယ်။*

```javascript
for (let i = 1; i <= 10; i++) {
    // i သည် စုံကိန်းလား မကိန်းလား စစ်ဆေးသည့် Code ကို ဤနေရာတွင် ရေးပါ
}
```

## အထူးစိန်ခေါ်မှု (Mini Challenge): Password Checker 

`password` လို့ အမည်ရတဲ့ Variable တစ်ခုကို ကြေညာပါ။
* အကယ်၍ စာလုံးရေ ၆ လုံးထက် နည်းနေရင် "Password too short" လို့ ပြပေးပါ။
* အကယ်၍ စာလုံးရေ ၂၀ ထက် များနေရင် "Password too long" လို့ ပြပေးပါ။
* အဲ့ဒီ ကြားထဲက စာလုံးရေ ဆိုရင်တော့ "Password valid" လို့ ပြပေးပါ။

*သဲလွန်စ (Hint): `password.length` ကို အသုံးပြုပြီး စာလုံးရေ ဘယ်လောက်ရှိလဲ ဆိုတာကို ရယူနိုင်ပါတယ်။*
