---

လူ့ဘဝမှာ အခြေအနေအရ ဆုံးဖြတ်ချက်တွေ ချမှတ်ရသလိုပဲ၊ Programming မှာလည်း အခြေအနေ (Condition) တစ်ခုအပေါ် မူတည်ပြီး ဘယ်အလုပ်ကို ဆက်လုပ်ရမလဲ ဆိုတာကို ဆုံးဖြတ်ချက် ချခိုင်းလို့ ရပါတယ်။ ဒါကို **Conditional Statements** လို့ ခေါ်ပါတယ်။

## If Statement (အကယ်၍)

"အကယ်၍" လို့ အဓိပ္ပာယ်ရပါတယ်။ ပေးထားတဲ့ အခြေအနေဟာ မှန်ကန်နေရင် (True ဖြစ်ရင်) သတ်မှတ်ထားတဲ့ အလုပ်ကို လုပ်ဆောင်ပါမယ်။

```javascript
let isRaining = true;

if (isRaining) {
  console.log("ထီးယူသွားပါ! ☔️");
}
```

## If...Else Statement (မဟုတ်လျှင်)

"ဒီအခြေအနေ မှန်ရင် (True) ဒါကိုလုပ်မယ်၊ မမှန်ဘူး (False) ဆိုရင်တော့ ဟိုဟာကိုလုပ်မယ်" ဆိုတဲ့ ပုံစံမျိုး ဖြစ်ပါတယ်။

```javascript
let age = 15;

if (age >= 18) {
  console.log("မဲပေးခွင့် ရှိပါတယ်။");
} else {
  console.log("အသက်မပြည့်သေးသဖြင့် မဲပေးခွင့် မရှိသေးပါ။");
}
```

## Else If Statement (ရွေးချယ်စရာ များပြားသောအခါ)

ရွေးချယ်စရာ အခြေအနေ (Conditions) တွေ အများကြီး ရှိနေတဲ့အခါမျိုးမှာ ဆက်တိုက် စစ်ဆေးဖို့ အသုံးပြုပါတယ်။

```javascript
let score = 85;

if (score >= 90) {
  console.log("Grade A");
} else if (score >= 80) {
  console.log("Grade B");
} else if (score >= 40) {
  console.log("Passed");
} else {
  console.log("Failed");
}
```

### 💡 အရေးကြီးသော မှတ်သားဖွယ်ရာများ
* JavaScript ဟာ အပေါ်ဆုံး `if` ကနေ စတင်ပြီး အောက်ကို တစ်ဆင့်ချင်း စစ်ဆေးသွားပါတယ်။
* အခြေအနေ တစ်ခု မှန်ကန်သွားတာ (True ဖြစ်သွားတာ) နဲ့ အဲ့ဒီအထဲက Code ကို လုပ်ဆောင်ပြီး၊ ကျန်တဲ့ အောက်က `else if` တွေ `else` တွေကို ဆက်မစစ်တော့ဘဲ ချက်ချင်း ရပ်တန့်သွားမှာ ဖြစ်ပါတယ်။
