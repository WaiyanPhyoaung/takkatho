## Programming Language ဆိုတာ ဘာလဲ?

ကွန်ပျူတာတွေဟာ လူတွေ ပြောတဲ့ စကားကို နားမလည်ပါဘူး။ သူတို့ကို တစ်ခုခု ခိုင်းစေချင်တယ်ဆိုရင် သူတို့ နားလည်တဲ့ ဘာသာစကား (Language) နဲ့ ပြောပြဖို့ လိုအပ်ပါတယ်။ **JavaScript** ဆိုတာဟာ ကွန်ပျူတာ (အထူးသဖြင့် Browser) ကို လိုအပ်တဲ့ အလုပ်တွေ လုပ်ဆောင်ပေးဖို့ ခိုင်းစေတဲ့ Programming Language တစ်ခု ဖြစ်ပါတယ်။

## Browser က JavaScript ကို ဘယ်လို Run သလဲ?

သင့်ရဲ့ Browser (ဥပမာ - Chrome, Firefox, Safari) တွေထဲမှာ **JavaScript Engine** လို့ခေါ်တဲ့ အင်ဂျင်စက်လေးတစ်ခု ပါဝင်ပါတယ်။ အဲ့ဒီ Engine လေးက သင်ရေးလိုက်တဲ့ JavaScript Code တွေကို ဖတ်ရှုပြီး၊ ကွန်ပျူတာ နားလည်နိုင်တဲ့ ပုံစံမျိုး ပြောင်းလဲကာ (ဘာသာပြန်ပေးပြီး) အလုပ်လုပ်ဆောင်ပေးတာ ဖြစ်ပါတယ်။

* မှတ်သားရန် - Chrome Browser မှာဆိုရင် **V8 Engine** ဆိုတဲ့ နာမည်ကြီး Engine ကို အသုံးပြုထားပါတယ်။

## HTML, CSS နှင့် JavaScript တို့၏ ကွာခြားချက်

ဒီနည်းပညာ သုံးခုရဲ့ ကွာခြားချက်ကို အလွယ်ကူဆုံး ဥပမာပေးရရင် **လူတစ်ယောက်** နဲ့ တူပါတယ်။

| နည်းပညာ | ခိုင်းနှိုင်းချက် | လုပ်ဆောင်ချက် (Functionality) |
| :--- | :--- | :--- |
| **HTML** | အရိုးစု (Skeleton) | ဝက်ဘ်ဆိုက်ရဲ့ ဖွဲ့စည်းပုံကို တည်ဆောက်ပေးပါတယ်။ (ဥပမာ - လူတစ်ယောက်ရဲ့ ခေါင်း၊ ခြေ၊ လက်) |
| **CSS** | အဝတ်အစား (Clothes) | မျက်စိပသာဒ ဖြစ်အောင်၊ လှပသပ်ရပ်အောင် ဒီဇိုင်းဆင်ပေးပါတယ်။ |
| **JavaScript**| ဦးနှောက်/ကြွက်သား (Brain/Muscles) | လှုပ်ရှားမှုတွေ လုပ်မယ်၊ စဉ်းစားဆင်ခြင်မယ်၊ တုံ့ပြန်မှုတွေ လုပ်ဆောင်ပေးပါတယ်။ |

### ဥပမာ Code ဖြင့် ရှင်းလင်းချက်

* **HTML:** အရင်ဆုံး ခလုတ် (Button) လေးတစ်ခုကို ဖန်တီးလိုက်ပါမယ်။
* **CSS:** အဲ့ဒီ Button လေးကို အနီရောင်လေး ပြောင်းပြီး လှအောင် ပြင်ဆင်လိုက်ပါမယ်။
* **JavaScript:** အဲ့ဒီ Button လေးကို လူတစ်ယောက်က နှိပ်လိုက်တာနဲ့ Alert Box (သတိပေးစာတမ်း) လေး ပေါ်လာအောင် အသက်သွင်းပေးပါမယ်။

JavaScript ကို သေချာ နားလည်သွားပြီ ဆိုရင်တော့၊ သင်ဟာ ရိုးရှင်းတဲ့ Webpage တစ်ခုကို စွမ်းဆောင်ရည်မြင့်မားတဲ့ Application (Web App) တစ်ခုအဖြစ် ဖန်တီးပြောင်းလဲနိုင်မှာ သေချာပါတယ်။
