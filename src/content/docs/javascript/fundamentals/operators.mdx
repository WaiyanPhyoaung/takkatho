Operators ဆိုတာ **Data တွေကို သင်္ချာ တွက်ချက်ဖို့၊ တန်ဖိုး သတ်မှတ်ဖို့နဲ့၊ တစ်ခုနဲ့ တစ်ခု နှိုင်းယှဉ်ဖို့ (Compare) အသုံးပြုတဲ့ သင်္ကေတများ** ဖြစ်ပါတယ်။

## ၁။ Arithmetic Operators (သင်္ချာ တွက်ချက်မှု သင်္ကေတများ)

အခြေခံ ပေါင်း၊ နှုတ်၊ မြှောက်၊ စား လုပ်ဆောင်ချက်တွေ ဖြစ်ပါတယ်။

```javascript
let a = 10;
let b = 5;

console.log(a + b); // ရလဒ်: 15 (ပေါင်း)
console.log(a - b); // ရလဒ်: 5 (နှုတ်)
console.log(a * b); // ရလဒ်: 50 (မြှောက်)
console.log(a / b); // ရလဒ်: 2 (စားလဒ်)
console.log(a % b); // ရလဒ်: 0 (စားကြွင်း - Remainder လို့ ခေါ်ပါတယ်)
```

## ၂။ Assignment Operators (တန်ဖိုး သတ်မှတ်ခြင်း သင်္ကေတများ)

Variable တစ်ခုထဲကို တန်ဖိုး (Value) ထည့်သွင်းတဲ့ နေရာမှာ အသုံးပြုပါတယ်။ အခြေခံအကျဆုံး သင်္ကေတကတော့ `=` (Equal) ဖြစ်ပါတယ်။

```javascript
let x = 10; 
x += 5; // x = x + 5 လို့ ရေးတာနဲ့ အတူတူပါပဲ (x ရဲ့ တန်ဖိုး အသစ်က 15 ဖြစ်သွားပါမယ်)
x -= 2; // x = x - 2 လို့ ရေးတာနဲ့ အတူတူပါပဲ
```

## ၃။ Comparison Operators (နှိုင်းယှဉ်ခြင်း သင်္ကေတများ)

တန်ဖိုး နှစ်ခုဟာ တူသလား၊ မတူဘူးလား၊ ကြီးသလား၊ ငယ်သလား ဆိုတာတွေကို စစ်ဆေးတဲ့ နေရာမှာ သုံးပါတယ်။ နှိုင်းယှဉ်မှု ရလဒ်က အမြဲတမ်း **Boolean (`true` သို့မဟုတ် `false`)** အနေနဲ့ပဲ ထွက်လာမှာ ဖြစ်ပါတယ်။

```javascript
let num = 10;

console.log(num > 5);  // true (10 က 5 ထက် ကြီးပါသလား? ကြီးလို့ မှန်ပါတယ်)
console.log(num < 5);  // false (10 က 5 ထက် ငယ်ပါသလား? မငယ်လို့ မှားပါတယ်)
console.log(num === 10); // true (10 ဟာ 10 နဲ့ အတိအကျ တူညီပါသလား? တူပါတယ်)
console.log(num !== 10); // false (10 ဟာ 10 နဲ့ မတူဘူးလား? မတူဘူးဆိုတာ မှားပါတယ်)
```

> **သတိပြုရန် (အလွန် အရေးကြီးသည်):** `==` နဲ့ `===` ဟာ အလုပ်လုပ်ပုံ လုံးဝ မတူညီပါဘူး။
> * `==` (Double Equal) ဟာ တန်ဖိုး (Value) ကိုပဲ စစ်ဆေးပြီး Type ကို မစစ်ဆေးပါဘူး။ ဥပမာ - `"5" == 5` လို့ စစ်ရင် Type မတူပေမယ့် တန်ဖိုးတူလို့ `true` လို့ အဖြေထွက်ပါလိမ့်မယ်။
> * `===` (Triple Equal) ဟာ တန်ဖိုးရော၊ Type ကိုပါ တင်းတင်းကျပ်ကျပ် စစ်ဆေးပါတယ်။ ဥပမာ - `"5" === 5` လို့ စစ်ရင် Type မတူတဲ့အတွက် `false` လို့ အဖြေထွက်ပါမယ်။
> **အကောင်းဆုံး အလေ့အကျင့်အနေနဲ့ အမြဲတမ်း `===` ကိုသာ အသုံးပြုပါ။**

## ၄။ Logical Operators (ယုတ္တိဗေဒ သင်္ကေတများ)

အခြေအနေ (Condition) တစ်ခုထက်ပိုပြီး တွဲဖက် စစ်ဆေးချင်တဲ့ အခါတွေမှာ အသုံးပြုပါတယ်။

| Operator | အမည် | ရှင်းလင်းချက် |
| :--- | :--- | :--- |
| `&&` | AND (နှစ်ခုလုံး မှန်မှ) | ဘယ်ဘက် Condition ရော၊ ညာဘက် Condition ပါ နှစ်ခုစလုံး မှန်ကန်မှသာ ရလဒ် `true` ထွက်ပါမယ်။ |
| `\|\|` | OR (တစ်ခု မှန်တာနဲ့) | Condition နှစ်ခုထဲက တစ်ခုခု မှန်ကန်တာနဲ့ ရလဒ် `true` ထွက်ပါမယ်။ |
| `!` | NOT (ပြောင်းပြန်) | မှန်နေရင် `false`၊ မှားနေရင် `true` ဆိုပြီး ပြောင်းပြန်လှန်ပစ်ပါတယ်။ |

```javascript
let age = 20;
let hasID = true;

// အသက် ၁၈ နှစ်ကျော်ပြီး (AND) ID ကတ်လည်း ရှိမှသာ ဝင်ခွင့်ပြုမယ် ဆိုတဲ့ Logic မျိုး ဖြစ်ပါတယ်
if (age > 18 && hasID) {
    console.log("Welcome!");
}
```
