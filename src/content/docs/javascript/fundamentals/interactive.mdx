သင်ခန်းစာတွေမှာ လေ့လာခဲ့တဲ့ အကြောင်းအရာတွေကို လက်တွေ့ စမ်းသပ်ကြည့်ရအောင်။ အောက်ပါ Code တွေကို အရင်ဆုံး ခန့်မှန်းကြည့်ပါ။ ပြီးရင် Browser ရဲ့ Console မှာ လက်တွေ့ ရိုက်ထည့်ပြီး ကိုယ်ခန့်မှန်းထားတဲ့ အဖြေနဲ့ တိုက်ဆိုင် စစ်ဆေးကြည့်ပါ။

## လေ့ကျင့်ခန်း ၁ (Exercise 1): Variables

```javascript
let a = 10;
a = 20;
const b = 30;
// b = 40; 
console.log(a + b);
```
👉 **စဉ်းစားရန် မေးခွန်း:** Console မှာ ဘယ်လို အဖြေ ပေါ်လာမလဲ? `b = 40` ရဲ့ ရှေ့က Comment သင်္ကေတ `//` ကို ဖြုတ်လိုက်ရင်ရော ဘာဆက်ဖြစ်မလဲ?

## လေ့ကျင့်ခန်း ၂ (Exercise 2): Strings & Numbers

```javascript
let x = "10";
let y = 20;
console.log(x + y);
```
👉 **စဉ်းစားရန် မေးခွန်း:** အဖြေက `30` ထွက်မလား၊ `1020` ထွက်မလား? ဘာကြောင့် အဲ့ဒီလို ထွက်ရတာလဲ ဆိုတာကို ရှင်းပြနိုင်မလား?

## လေ့ကျင့်ခန်း ၃ (Exercise 3): Comparison

```javascript
let score = 100;
console.log(score === "100");
```
👉 **စဉ်းစားရန် မေးခွန်း:** ရလဒ်က `true` ဖြစ်မလား၊ `false` ဖြစ်မလား? `===` အစား `==` ကို ပြောင်းသုံးလိုက်ရင်ရော အဖြေက ဘာဖြစ်သွားမလဲ?
