## Variables (ကိန်းရှင်များ)

Variable ဆိုတာ **Data တွေကို ခေတ္တခဏ သိမ်းဆည်းထားပေးတဲ့ သေတ္တာ (Box) လေးတစ်ခု** လို့ မှတ်ယူနိုင်ပါတယ်။
ဥပမာ - ရေထည့်ထားတဲ့ ဖန်ခွက်တစ်ခွက်လိုပါပဲ။ အဲ့ဒီ ဖန်ခွက်ထဲကို ရေသန့် ထည့်မလား၊ ကော်ဖီ ထည့်မလား၊ ကိုယ်ကြိုက်တာ ထည့်လို့ရသလိုမျိုးပါ။

JavaScript မှာ Variable ကြေညာဖို့အတွက် အဓိက နည်းလမ်း (၃) မျိုး ရှိပါတယ်။

### ၁။ `var` (ရှေးဟောင်း နည်းလမ်း)
ဒါကတော့ JavaScript ရဲ့ အစောပိုင်း ကာလတွေကတည်းက အသုံးပြုခဲ့တဲ့ နည်းလမ်းဟောင်းပါ။ လောလောဆယ်မှာတော့ ပြဿနာ တချို့ရှိတတ်လို့ သိပ်မသုံးကြတော့ပါဘူး။ သိထားရုံလောက်ပဲ လုံလောက်ပါတယ်။

### ၂။ `let` (စံသတ်မှတ်ချက်)
နောက်ပိုင်းမှာ ပြန်လည် ပြောင်းလဲနိုင်တဲ့ တန်ဖိုး (Value) တွေအတွက် အသုံးပြုပါတယ်။ (ဥပမာ - ဂိမ်းဆော့တဲ့အခါ ရရှိတဲ့ အမှတ်စာရင်း (Score)၊ လူတစ်ယောက်ရဲ့ အသက် စသဖြင့်)
```javascript
let score = 10;
score = 20; // နောက်မှ တန်ဖိုးကို ၂၀ လို့ ပြန်ပြောင်းလိုက်တာပါ (ပြောင်းလို့ ရပါတယ်)
```

### ၃။ `const` (မပြောင်းလဲသော တန်ဖိုး)
Constant လို့ အဓိပ္ပာယ်ရပြီး၊ လုံးဝ ပြောင်းလဲခွင့် မရှိတဲ့ တန်ဖိုးတွေအတွက် အသုံးပြုပါတယ်။ (ဥပမာ - မွေးသက္ကရာဇ်၊ သင်္ချာမှာသုံးတဲ့ `PI` တန်ဖိုး)
```javascript
const pi = 3.14;
// pi = 3.15; // Error တက်ပါမယ်! const နဲ့ ကြေညာထားရင် လုံးဝ ပြောင်းလို့ မရတော့ပါဘူး။
```

<Aside type="tip">
**Best Practice (အကောင်းဆုံး အလေ့အကျင့်):** ပုံမှန်အားဖြင့် `const` ကို အရင် စဉ်းစားပြီး သုံးပါ။ တန်ဖိုးပြောင်းဖို့ သေချာပေါက် လိုအပ်လာမှသာ `let` ကို သုံးပါ။ `var` ကိုတော့ လုံးဝ (လုံးဝ) မသုံးပါနဲ့။
</Aside>

## Data Types (Data အမျိုးအစားများ)

JavaScript မှာ အသုံးအများဆုံး Data Type (၅) မျိုး ရှိပါတယ်။ ဒီ (၅) မျိုးကို သိထားရင် လုံလောက်ပါပြီ။

၁။ **String (စာသား):** စာသားတွေကို ရေးတဲ့အခါ Double Quotes `""` (သို့မဟုတ်) Single Quotes `''` ထဲမှာ အမြဲတမ်း ထည့်ရေးရပါတယ်။
    ```javascript
    let name = "Mg Mg";
    ```
၂။ **Number (ကိန်းဂဏန်း):** သင်္ချာ ဂဏန်းတွေ ဖြစ်ပါတယ်။ `""` ထည့်စရာ မလိုပါဘူး။
    ```javascript
    let age = 25;
    let price = 10.5; // ဒသမကိန်းတွေလည်း ရပါတယ်
    ```
၃။ **Boolean (အမှား/အမှန်):** `true` (အမှန်) သို့မဟုတ် `false` (အမှား) ဆိုတဲ့ တန်ဖိုး နှစ်ခုတည်းသာ ရှိပါတယ်။
    ```javascript
    let isStudent = true;
    ```
၄။ **Undefined (သတ်မှတ်မထားခြင်း):** Variable တစ်ခုကို ကြေညာထားပေမယ့်၊ ဘာတန်ဖိုးမှ မထည့်ပေးရသေးရင် အဲ့ဒီ Variable ရဲ့ တန်ဖိုးက `undefined` ဖြစ်နေပါလိမ့်မယ်။
    ```javascript
    let x;
    console.log(x); // ဘာမှ မထည့်ထားလို့ undefined လို့ ပြပါမယ်
    ```
၅။ **Null (ဘာမှမရှိခြင်း):** ဒါကတော့ "ဒီအထဲမှာ ဘာတန်ဖိုးမှ မရှိဘူး" ဆိုတာကို Programmer ကိုယ်တိုင်က တိတိကျကျ သတ်မှတ်ပေးလိုက်တာ ဖြစ်ပါတယ်။
    ```javascript
    let y = null;
    ```

**မှတ်သားရန်:** String ဆိုရင် အမြဲတမ်း Quote (`" "`) ပါရပါမယ်၊ Number ဆိုရင်တော့ မပါပါဘူး။ ဥပမာ - `let "20"` ဆိုရင် စာသား (String) အဖြစ် သတ်မှတ်ပြီး၊ `let 20` လို့ ရေးမှသာ ဂဏန်း (Number) အဖြစ် သတ်မှတ်မှာ ဖြစ်ပါတယ်။
