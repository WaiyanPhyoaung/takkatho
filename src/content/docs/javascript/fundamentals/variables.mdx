## Variables (ကိန်းရှင်များ)

Variable ဆိုတာ **Data တွေကို ခဏတာ သိမ်းဆည်းထားတဲ့ နေရာ (Box)** လေးတစ်ခုပါ။
ဥပမာ - ရေထည့်ထားတဲ့ ခွက်လိုပေါ့။ ခွက်ထဲကို ရေထည့်မလား၊ ဖျော်ရည်ထည့်မလား၊ ကြိုက်တာ ထည့်လို့ရတယ်။

JavaScript မှာ Variable ကြေညာဖို့ နည်းလမ်း (၃) မျိုး ရှိပါတယ်။

### 1. `var` (Old School)
ရှေးဟောင်း နည်းလမ်းပါ။ လောလောဆယ်မှာ သိပ်မသုံးတော့ပါဘူး။ သိထားရုံလောက်ပါပဲ။

### 2. `let` (Standard)
ပြောင်းလဲလို့ရတဲ့ တန်ဖိုးတွေအတွက် သုံးပါတယ်။ (ဥပမာ - အမှတ်စာရင်း၊ အသက်)
```javascript
let score = 10;
score = 20; // နောက်မှ ပြောင်းလို့ရတယ်
```

### 3. `const` (Constant)
ပြောင်းလဲလို့ မရတဲ့ တန်ဖိုးတွေအတွက် သုံးပါတယ်။ (ဥပမာ - မွေးနေ့၊ `PI` တန်ဖိုး)
```javascript
const pi = 3.14;
// pi = 3.15; // Error တက်မယ်! ပြောင်းလို့မရဘူး။
```

<Aside type="tip">
**Best Practice:**  ပုံမှန်အားဖြင့် `const` ကို အရင်သုံးပါ။ တန်ဖိုးပြောင်းဖို့ လိုအပ်မှ `let` ကို သုံးပါ။ `var` ကို လုံးဝ မသုံးပါနဲ့။
</Aside>

## Data Types (Data အမျိုးအစားများ)

JavaScript မှာ အဓိက Data Type (၅) မျိုးကို သိထားရင် လုံလောက်ပါပြီ။

1.  **String (စာသား):** စာလုံးတွေကို `""` (သို့) `''` ထဲမှာ ထည့်ရေးရပါတယ်။
    ```javascript
    let name = "Mg Mg";
    ```
2.  **Number (ကိန်းဂဏန်း):** ဂဏန်းတွေပါ။ `""` ထည့်စရာ မလိုပါဘူး။
    ```javascript
    let age = 25;
    let price = 10.5;
    ```
3.  **Boolean (အမှား/အမှန်):** `true` သို့မဟုတ် `false` နှစ်ခုပဲ ရှိပါတယ်။
    ```javascript
    let isStudent = true;
    ```
4.  **Undefined:** Variable တစ်ခုကို ကြေညာပြီး တန်ဖိုးမထည့်ရသေးရင် `undefined` ဖြစ်နေပါမယ်။
    ```javascript
    let x;
    console.log(x); // undefined
    ```
5.  **Null:** "ဘာမှ မရှိဘူး" ဆိုတာကို တိတိကျကျ သတ်မှတ်လိုက်တာပါ။
    ```javascript
    let y = null;
    ```

မှတ်ထားဖို့က - String ဆိုရင် Quote (" ") ပါမယ်၊ Number ဆိုရင် မပါဘူး။ `let "20"` က စာသား၊ `let 20` က ဂဏန်းပါ။
