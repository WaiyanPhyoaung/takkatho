### Arithmetic Operators (သင်္ချာ တွက်ချက်ခြင်း)
ပေါင်း၊ နှုတ်၊ မြှောက်၊ စား လုပ်တာတွေပါ။

```javascript
let x = 10;
let y = 5;

console.log(x + y); // 15 (ပေါင်း)
console.log(x - y); // 5 (နှုတ်)
console.log(x * y); // 50 (မြှောက်)
console.log(x / y); // 2 (စား)
console.log(x % y); // 0 (အကြွင်းရှာ)
```

### String Operators (စာသား ဆက်ခြင်း)
စာသားနှစ်ခုကို ပေါင်းချင်ရင် `+` ကို သုံးလို့ရသလို၊ **Template Literals** (Backticks `` ` ``) ကို သုံးတာ ပိုခေတ်မီပါတယ်။

```javascript
let firstName = "Kyaw";
let lastName = "Kyaw";

// Old way
// let fullName = firstName + " " + lastName;

// Modern way (Recommended)
let fullName = `${firstName} ${lastName}`; // "Kyaw Kyaw"
```

### Comparison Operators (နှိုင်းယှဉ်ခြင်း)
တန်ဖိုးတွေကို ယှဉ်ကြည့်ပြီး `true` သို့မဟုတ် `false` ပြန်ပေးပါတယ်။

*   `>` (ထက်ကြီး)
*   `<` (ထက်ငယ်)
*   `>=` (ကြီး သို့မဟုတ် ညီ)
*   `<=` (ငယ် သို့မဟုတ် ညီ)
*   `===` (ညီသလား?)
*   `!==` (မညီဘူးလား?)

```javascript
let age = 18;

console.log(age > 20);  // false
console.log(age === 18); // true
```

<Aside type="danger">
`==` နဲ့ `===` မတူပါဘူး။
*   `5 == "5"` က **true** (Type မစစ်ဘူး)
*   `5 === "5"` က **false** (Type ပါ စစ်တယ် - Recommended)
အမြဲတမ်း `===` ကိုပဲ သုံးပါ။
</Aside>
