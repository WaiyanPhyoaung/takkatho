<Aside title="ပုံးလေးများ">
Variable ဆိုတာ အချက်အလက်တွေကို သိမ်းထားတဲ့ **"ပုံး"** လေးတွေနဲ့ တူပါတယ်။ ပုံးပေါ်မှာ နာမည်ကပ်ထားပြီး၊ အထဲမှာ ပစ္စည်း (Data) ထည့်ထားတာပါ။
</Aside>

### Variable ကြေညာခြင်း
ခေတ်မီ JavaScript (ES6) မှာ `let` နဲ့ `const` ကို အဓိက သုံးပါတယ်။

#### 1. let (ပြောင်းလဲနိုင်သော ပုံး)
အထဲက Data ကို နောက်မှ ပြန်ပြောင်းချင်ရင် သုံးပါတယ်။

```javascript
let name = "Mg Mg";
name = "Aung Aung"; // ပြောင်းလို့ ရတယ်
```

#### 2. const (မပြောင်းလဲနိုင်သော ပုံး)
တစ်ခါထည့်ပြီးရင် ပြန်ပြင်လို့ မရတော့ပါဘူး။ (Constant)

```javascript
const pi = 3.14;
pi = 3.15; // Error တက်မယ်!
```

### Data Types (အချက်အလက် အမျိုးအစားများ)
ပုံးထဲမှာ ဘာတွေ ထည့်လို့ရလဲ?

1.  **String (စာသား):** မျက်တောင်အဖွင့်အပိတ် `""` သို့မဟုတ် `''` ထဲမှာ ရေးရတယ်။
    ```javascript
    let message = "Hello World";
    ```

2.  **Number (ကိန်းဂဏန်း):** ဂဏန်းအတိုင်း ရေးရတယ်။
    ```javascript
    let age = 25;
    let price = 9.99;
    ```

3.  **Boolean (အမှား/အမှန်):** `true` (မှန်) သို့မဟုတ် `false` (မှား)။
    ```javascript
    let isOnline = true;
    ```

<Aside type="tip">
`var` ဆိုတာ ဟိုးအရင်က သုံးခဲ့တဲ့ နည်းလမ်းပါ။ အခုခေတ်မှာတော့ `let` နဲ့ `const` ကိုပဲ သုံးသင့်ပါတယ်။
</Aside>
