JavaScript ရေးဖို့အတွက် ကွန်ပျူတာ အကောင်းစားကြီးတွေ မလိုအပ်ပါဘူး။ သင့်ကွန်ပျူတာမှာ ရှိပြီးသား Browser လေးတစ်ခု ရှိနေရင်ပဲ လုံလောက်ပါတယ်။ ဒါပေမဲ့ ပိုမိုကောင်းမွန်တဲ့ အတွေ့အကြုံ (Experience) ရရှိဖို့အတွက် အောက်ပါ Tools တွေကို ပြင်ဆင်ထားရင်တော့ အကောင်းဆုံးပါပဲ။

## ၁။ Web Browser (Chrome Recommended)

JavaScript ဟာ မည်သည့် Browser မှာမဆို Run လို့ ရပါတယ်။ ဒါပေမဲ့ Web Developer တွေ အများဆုံး အသုံးပြုပြီး၊ မှားယွင်းမှုတွေကို ရှာဖွေပြင်ဆင်ရ (Debug လုပ်ရ) အဆင်ပြေဆုံး ဖြစ်တဲ့ **Google Chrome** ကို အသုံးပြုဖို့ အကြံပြုလိုပါတယ်။

## ၂။ Code Editor (VS Code Recommended)

Code တွေ ရေးသားဖို့အတွက် **Visual Studio Code (VS Code)** ကို အသုံးပြုဖို့ အထူး တိုက်တွန်းပါတယ်။ သူက အခမဲ့ (Free) အသုံးပြုနိုင်တဲ့အပြင်၊ အလုပ်လုပ်ရာမှာ မြန်ဆန်စေမယ့် Extension တွေ အများကြီး ထည့်သွင်းနိုင်တဲ့အတွက် အရမ်းကို အသုံးဝင်ပါတယ်။

## JavaScript ကို ဘယ်လို Run မလဲ?

JavaScript ကို အဓိကအားဖြင့် နည်းလမ်း (၃) မျိုးနဲ့ Run လို့ ရပါတယ်။

### ၁။ Inline Script (HTML ထဲတွင် တိုက်ရိုက်ရေးခြင်း)

HTML file ထဲမှာ `<script>` ဆိုတဲ့ tag လေးဖွင့်ပြီး တိုက်ရိုက် ရေးသားတာ ဖြစ်ပါတယ်။ ဒါပေမဲ့ ဒီနည်းလမ်းကို စမ်းသပ်ရုံလောက်ပဲ သုံးသင့်ပြီး၊ တကယ့် Project တွေမှာတော့ မသုံးသင့်ပါဘူး။

```html
<button onclick="alert('Hello!')">Click Me</button>
```

### ၂။ External JS File (အသုံးအများဆုံး နည်းလမ်း)

JavaScript code တွေကို `script.js` လိုမျိုး သီးသန့် ဖိုင်ခွဲထုတ်ပြီးမှ HTML နဲ့ ပြန်လည် ချိတ်ဆက် အသုံးပြုတာ ဖြစ်ပါတယ်။ ဒါဟာ Code တွေကို ရှင်းလင်းသပ်ရပ်စေပြီး လက်တွေ့လုပ်ငန်းခွင်မှာ အသုံးအများဆုံး နည်းလမ်း ဖြစ်ပါတယ်။

```html
<!-- index.html ထဲတွင် ချိတ်ဆက်ပုံ -->
<script src="script.js"></script>
```

### ၃။ Browser Console (အမြန်စမ်းသပ်ရန်)

Browser ပေါ်မှာ Code လေးတွေကို ချက်ချင်း စမ်းသပ်ကြည့်ချင်ရင်တော့ Console ကို အသုံးပြုနိုင်ပါတယ်။
1. Browser (Chrome) ပေါ်မှာ Right Click နှိပ်ပြီး **Inspect** ကို ရွေးချယ်ပါ။
2. ပေါ်လာတဲ့ မျက်နှာပြင်ထဲက **Console** ဆိုတဲ့ tab ကို နှိပ်ပါ။
3. အဲ့ဒီမှာ `alert("Hello World")` လို့ ရိုက်ထည့်ပြီး Enter ခေါက်ကြည့်ပါ။
