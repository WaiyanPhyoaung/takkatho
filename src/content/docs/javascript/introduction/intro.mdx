## JavaScript ဆိုတာ ဘာလဲ?

JavaScript ဆိုတာ **Webpage တွေကို အသက်ဝင်စေတဲ့ Programming Language** တစ်ခုဖြစ်ပါတယ်။

လွယ်လွယ်ပြောရရင်:
*   **HTML** က အိမ်ရဲ့ **အရိုးစု (Structure)** နဲ့ တူတယ်။ (ခေါင်းစဉ်တွေ၊ စာပိုဒ်တွေ၊ ပုံတွေ)
*   **CSS** က အိမ်ရဲ့ **အလှဆင်မှု (Decorations)** နဲ့ တူတယ်။ (အရောင်တွေ၊ ဒီဇိုင်းတွေ)
*   **JavaScript** ကတော့ အိမ်ရဲ့ **လျှပ်စစ်မီးနဲ့ ရေပိုက်စနစ် (Functionality)** နဲ့ တူတယ်။ (မီးခလုပ်ဖွင့်ရင် မီးလင်းမယ်၊ ဘုံဘိုင်ဖွင့်ရင် ရေလာမယ်)

<Aside type="tip">
JavaScript ရှိမှသာ Website တစ်ခုက User နဲ့ တုံ့ပြန်ဆောင်ရွက် (Interact) နိုင်မှာ ဖြစ်ပါတယ်။ ဥပမာ - Button နှိပ်လိုက်ရင် ပုံပြောင်းသွားတာ၊ Form ဖြည့်ပြီး Submit လုပ်တာတွေပေါ့။
</Aside>

## DOM Manipulation ဆိုတာ ဘာလဲ?

ဒီ Course ရဲ့ အဓိက Focus က **DOM Manipulation** ပါ။

"DOM Manipulation" ဆိုတာ နည်းနည်းလေး နားလည်ရခက်တဲ့ စကားလုံး ဖြစ်နိုင်တယ်။ ဒါပေမဲ့ ရိုးရှင်းပါတယ်။
**JavaScript ကိုသုံးပြီး HTML လို့ခေါ်တဲ့ Webpage ပေါ်က အရာတွေကို လှမ်းပြောင်းလဲခြင်း** လို့ မှတ်လိုက်ပါ။

ဥပမာ -
*   စာသားတွေကို ပြောင်းမယ်။
*   အရောင်တွေကို ပြောင်းမယ်။
*   Element အသစ်တွေ ထပ်ထည့်မယ်။

မပူပါနဲ့။ နောက်ပိုင်းမှာ ဒါကို လက်တွေ့စမ်းသပ်ရင်း သေချာပေါက် နားလည်လာမှာပါ။

ဒီ Course က စာဖတ်ရုံ သက်သက်မဟုတ်ပါဘူး။ **Interactive** ဖြစ်ပါတယ်။
