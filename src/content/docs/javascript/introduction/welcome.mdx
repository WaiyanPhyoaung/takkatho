<Aside title="Web ၏ ဦးနှောက်">
HTML က **အရိုးစု**၊ CSS က **အလှဆင်ပစ္စည်း** ဆိုရင်၊ JavaScript (JS) ကတော့ Website ရဲ့ **ဦးနှောက်** ပါပဲ။
</Aside>

### JavaScript ဆိုတာ ဘာလဲ?
JavaScript ဆိုတာ Website တွေကို "အသက်ဝင်" လာအောင် လုပ်ပေးတဲ့ Programming Language ဖြစ်ပါတယ်။
User က ခလုတ်နှိပ်လိုက်ရင် ဘာဖြစ်မလဲ? Form ဖြည့်ပြီးရင် ဘာလုပ်မလဲ? ဒါတွေကို JS နဲ့ ခိုင်းစေရပါတယ်။

### ဘာတွေ လုပ်လို့ရလဲ?
*   **Interactive ဖြစ်စေတယ်:** Slide တွေ၊ Pop-up တွေ၊ Menu တွေ ဖွင့်ပိတ်လုပ်တာ။
*   **Data တွက်ချက်တယ်:** ဈေးခြင်း (Cart) ထဲက ပစ္စည်းဈေးနှုန်း ပေါင်းတာ။
*   **Server နဲ့ ချိတ်ဆက်တယ်:** Page ကို Refresh မလုပ်ဘဲ Data အသစ်တွေ ဆွဲယူတာ (ဥပမာ - Facebook Newsfeed)။

### ဘယ်လို ရေးမလဲ?
HTML ဖိုင်ထဲမှာ `<script>` tag ဖွင့်ပြီး ရေးလို့ရသလို၊ `.js` ဖိုင် သီးသန့်ခွဲပြီးလည်း ရေးလို့ရပါတယ်။

```html
<!-- HTML ထဲမှာ ရေးခြင်း -->
<script>
  alert("မင်္ဂလာပါ!");
</script>
```

```html
<!-- သီးသန့်ဖိုင် ချိတ်ခြင်း (Recommended) -->
<script src="script.js" defer></script>
```

<Aside type="tip">
`defer` attribute ထည့်ထားရင် HTML တွေ အကုန် load ပြီးမှ JavaScript အလုပ်လုပ်မှာမို့ Error တက်တာတွေ သက်သာစေပါတယ်။
JavaScript ကို Browser တိုင်းမှာ Run လို့ရပါတယ်။ ဘာ Software မှ ထပ်သွင်းစရာ မလိုပါဘူး။
</Aside>
