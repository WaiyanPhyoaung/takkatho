Element ကို ဖမ်းမိပြီဆိုရင် သူ့ရဲ့ စာသား၊ အရောင်၊ ပုံစံ အကုန် လိုက်ပြောင်းလို့ ရပါပြီ။
</Aside>

### Content ပြောင်းခြင်း
စာသားတွေကို ပြောင်းချင်ရင် `textContent` (ပိုမြန်တယ်) သို့မဟုတ် `innerHTML` ကို သုံးပါတယ်။

```javascript
const title = document.querySelector("h1");

// စာသား သက်သက် ပြောင်းမယ် (Recommended)
title.textContent = "မင်္ဂလာပါ JavaScript";

// HTML tag တွေပါ ထည့်ချင်ရင်
title.innerHTML = "<span style='color: red'>Hello</span> World";
```

### Style ပြောင်းခြင်း
CSS style တွေကို တိုက်ရိုက် ပြောင်းလို့ရပါတယ်။

```javascript
const box = document.querySelector(".box");

box.style.backgroundColor = "blue";
box.style.fontSize = "20px"; // CSS မှာ font-size, JS မှာ fontSize (CamelCase)
box.style.display = "none"; // ဖျောက်လိုက်မယ်
```

### Class ပြောင်းခြင်း
Style တွေကို တစ်ခုချင်း လိုက်ပြင်တာထက် CSS Class ကို အဖွင့်အပိတ် လုပ်တာက ပိုကောင်းပါတယ်။

```javascript
const btn = document.querySelector("button");

// Class ထပ်ထည့်မယ်
btn.classList.add("active");

// Class ဖြုတ်မယ်
btn.classList.remove("active");

// အဖွင့်အပိတ် လုပ်မယ် (Toggle)
btn.classList.toggle("dark-mode");
```
