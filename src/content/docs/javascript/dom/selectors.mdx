<Aside title="DOM ဆိုတာ?">
**Document Object Model (DOM)** ဆိုတာ HTML စာမျက်နှာကို JavaScript က နားလည်တဲ့ ပုံစံနဲ့ ဖွဲ့စည်းထားတာပါ။
JS က DOM ကို သုံးပြီး HTML element တွေကို လှမ်းကိုင်၊ လှမ်းပြင်တာ လုပ်ပါတယ်။
</Aside>

### Element ရှာဖွေခြင်း
HTML ထဲက Element တစ်ခုခုကို JS နဲ့ ပြင်ချင်ရင် အရင်ဆုံး သူ့ကို "ဖမ်း" ရပါတယ်။

#### 1. getElementById (ID နဲ့ ရှာမယ်)
အမြန်ဆုံးနဲ့ အတိကျဆုံး နည်းလမ်းပါ။

```html
<h1 id="title">Hello</h1>
```

```javascript
const title = document.getElementById("title");
```

#### 2. querySelector (CSS Selector နဲ့ ရှာမယ်)
CSS ရေးသလိုမျိုး ရှာလို့ရလို့ အရမ်း အဆင်ပြေပါတယ်။

```javascript
// Class နဲ့ ရှာမယ်
const btn = document.querySelector(".btn-primary");

// Tag နဲ့ ရှာမယ်
const text = document.querySelector("p");

// ID နဲ့ ရှာမယ်
const header = document.querySelector("#header");
```

#### 3. querySelectorAll (အများကြီး ရှာမယ်)
တူညီတဲ့ Element တွေ အကုန်လုံးကို စာရင်း (NodeList) အနေနဲ့ ပြန်ပေးပါတယ်။

```javascript
const items = document.querySelectorAll(".list-item");
```

<Aside type="tip">
အများအားဖြင့် `querySelector` နဲ့ `getElementById` ကို အသုံးအများဆုံး ဖြစ်ပါတယ်။
</Aside>
