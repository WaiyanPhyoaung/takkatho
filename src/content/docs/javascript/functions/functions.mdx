<Aside title="ဟင်းချက်နည်း">
Function ဆိုတာ ဟင်းချက်နည်း (Recipe) တစ်ခုလိုပါပဲ။
ပါဝင်ပစ္စည်း (Input) ထည့်ပေးလိုက်ရင်၊ ချက်ပြုတ်ပြီး ဟင်းပွဲ (Output) ပြန်ထွက်လာပါတယ်။
</Aside>

### Function တည်ဆောက်ခြင်း
`function` keyword ကို သုံးပြီး နာမည်ပေးရပါတယ်။

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}
```

### Function ခေါ်သုံးခြင်း (Calling)
နာမည်ခေါ်ပြီး ကွင်းစကွင်းပိတ် `()` နဲ့ သုံးရပါတယ်။

```javascript
let message = greet("Mg Mg");
console.log(message); // "Hello, Mg Mg!"
```

### Arrow Functions (ခေတ်မီရေးနည်း)
ES6 မှာ Function ရေးနည်း ပိုတိုသွားပါတယ်။

```javascript
const add = (a, b) => {
  return a + b;
};

// တစ်ကြောင်းတည်းဆို return တောင် မလိုတော့ဘူး
const multiply = (a, b) => a * b;

console.log(add(5, 3)); // 8
```

<Aside type="tip">
Function တွေက Code ကို ထပ်ခါထပ်ခါ မရေးရအောင် ကူညီပေးပါတယ်။ "Don't Repeat Yourself (DRY)" ဆိုတဲ့ Rule အတွက် အဓိကပါပဲ။
</Aside>
