---

**Function** ဆိုတာ လုပ်ငန်းစဉ် (Task) တစ်ခုကို လုပ်ဆောင်ပေးတဲ့ Code အစုအဝေး တစ်ခုဖြစ်ပါတယ်။
Function တွေရဲ့ အဓိက ရည်ရွယ်ချက်က **"Write once, use many times"** (တစ်ခါရေး၊ ခဏခဏ ပြန်သုံး) ဖြစ်ပါတယ်။

## Function Syntax

Function တစ်ခု တည်ဆောက်မယ်ဆိုရင် `function` ဆိုတဲ့ keyword နဲ့ စရပါတယ်။

```javascript
// Function ကြေညာခြင်း (Defining)
function sayHello() {
  console.log("မင်္ဂလာပါ! 🙏");
}
```

## Calling Functions (Function ခေါ်သုံးခြင်း)

Function ကြေညာရုံနဲ့ အလုပ်မလုပ်ပါဘူး။ သူ့ကို ခေါ်သုံး (Call/Invoke) မှ အလုပ်လုပ်ပါတယ်။

```javascript
sayHello(); // "မင်္ဂလာပါ! 🙏"
sayHello(); // "မင်္ဂလာပါ! 🙏"
```

တွေ့လား? နှစ်ခါခေါ်ရင် နှစ်ခါ အလုပ်လုပ်တယ်။ Code တွေ အများကြီး ထပ်ကာထပ်ကာ ရေးစရာ မလိုတော့ဘူး။
