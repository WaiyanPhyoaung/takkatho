---

**Function** ဆိုတာ သီးသန့် လုပ်ငန်းစဉ် (Task) တစ်ခုတည်းကိုပဲ တိတိကျကျ လုပ်ဆောင်ပေးမယ့် Code အစုအဝေး တစ်ခု ဖြစ်ပါတယ်။
Function တွေရဲ့ အဓိက ရည်ရွယ်ချက်ကတော့ **"Write once, use many times"** (တစ်ခါပဲ အပင်ပန်းခံ ရေးထားမယ်၊ လိုအပ်တဲ့ အခါတိုင်း ခဏခဏ ပြန်ခေါ်သုံးမယ်) ဆိုတဲ့ သဘောတရားပဲ ဖြစ်ပါတယ်။

## Function Syntax (ရေးသားပုံ)

Function တစ်ခုကို စတင် တည်ဆောက်မယ်ဆိုရင် `function` ဆိုတဲ့ Keyword လေးကို အရင်ဆုံး ရေးပေးရပါတယ်။

```javascript
// Function ကို တည်ဆောက် ကြေညာခြင်း (Defining a Function)
function sayHello() {
  console.log("မင်္ဂလာပါ! 🙏");
}
```

## Calling Functions (Function ခေါ်ယူ အသုံးပြုခြင်း)

Function ကို ကြေညာထားရုံ သက်သက်နဲ့တော့ သူက အလုပ်မလုပ်သေးပါဘူး။ သူ့ကို နာမည်တပ်ပြီး ခေါ်သုံး (Call သို့မဟုတ် Invoke လုပ်) မှသာ အထဲက Code တွေက စတင် အလုပ်လုပ်မှာ ဖြစ်ပါတယ်။

```javascript
sayHello(); // ရလဒ်: "မင်္ဂလာပါ! 🙏"
sayHello(); // ရလဒ်: "မင်္ဂလာပါ! 🙏"
```

တွေ့လား ခင်ဗျာ? နှစ်ခါ ခေါ်လိုက်ရင် နှစ်ခါ အလုပ်လုပ်ပါတယ်။ တကယ်လို့ သင့်အနေနဲ့ ဒီအလုပ်ကို အကြိမ် (၁၀၀) လုပ်ချင်တယ် ဆိုရင်တောင် Code တွေကို အကြိမ် (၁၀၀) ထပ်ခါထပ်ခါ ကူးယူ ရေးသားနေစရာ မလိုတော့ပါဘူး။ Function ကို အကြိမ် (၁၀၀) ခေါ်လိုက်ရုံပါပဲ။
