---

ES6 (2015 ခုနှစ်) ဗားရှင်းက စတင်ပြီး JavaScript မှာ Function တွေကို ရေးသားဖို့ နည်းလမ်းသစ် တစ်ခု ထွက်ပေါ်လာပါတယ်။ အဲ့ဒါကတော့ **Arrow Function** ပဲ ဖြစ်ပါတယ်။ အရင်နည်းလမ်းထက် Code ပိုတိုတောင်းပြီး ပိုမို သပ်ရပ်ပါတယ်။ `=>` (မြှားပုံသင်္ကေတ) ကို အသုံးပြုထားတဲ့အတွက် Arrow Function လို့ အမည်တွင်တာ ဖြစ်ပါတယ်။

## Regular Function နှင့် Arrow Function နှိုင်းယှဉ်ချက်

### ရိုးရိုး Function (Regular Function)
```javascript
function sayHi() {
  return "Hi!";
}
```

### Arrow Function
```javascript
const sayHi = () => {
  return "Hi!";
}
```

## ပိုမို တိုတောင်းအောင် ရေးသားနည်း

အကယ်၍ Function ထဲမှာ Code က တစ်ကြောင်းတည်းပဲ ရှိပြီး၊ အဲ့ဒီ တစ်ကြောင်းကနေ `return` ကို တိုက်ရိုက် လုပ်မယ်ဆိုရင် တွန့်ကွင်း `{}` တွေနဲ့ `return` ဆိုတဲ့ စာသားကို ဖြုတ်ပစ်ပြီး အတိုဆုံး ပုံစံ ရေးလို့ ရပါတယ်။

```javascript
// အတိုဆုံးနဲ့ အသပ်ရပ်ဆုံး ပုံစံ
const add = (a, b) => a + b;

console.log(add(5, 10)); // ရလဒ်: 15
```

## ဘယ်လို နေရာတွေမှာ အသုံးပြုမလဲ?

`this` Keyword နဲ့ ပတ်သက်တဲ့ ရှုပ်ထွေးတဲ့ ကိစ္စရပ်တွေ (Object Methods) ကလွဲရင် ကျန်တဲ့ နေရာ အတော်များများမှာ အခုဆိုရင် Arrow Function ကိုပဲ ကျယ်ကျယ်ပြန့်ပြန့် သုံးလေ့ ရှိကြပါတယ်။ အထူးသဖြင့် React, Vue လိုမျိုး ခေတ်မီ Framework တွေမှာဆိုရင် Arrow Function ကို အဓိကထား အသုံးပြုကြပါတယ်။
