Function တွေဟာ ကိုယ့်ဘာသာ သီးသန့် အလုပ်လုပ်ရုံတင် မဟုတ်ဘဲ၊ အပြင်ဘက်ကနေ Data တွေကို လက်ခံရယူနိုင်သလို၊ တွက်ချက်လို့ ရလာတဲ့ ရလဒ်တွေကိုလည်း အပြင်ကို ပြန်လည် ပေးပို့နိုင်ပါတယ်။

## Parameters (လက်ခံသူများ)

Function ကို ခေါ်သုံးတဲ့ အချိန်မှာ Function ထဲကို Data တွေ ထည့်သွင်း ပေးလိုက်ချင်တဲ့ အခါမျိုးမှာ အသုံးပြုပါတယ်။

```javascript
function greet(name) {
  console.log("Hello " + name + "!");
}

greet("Mg Mg"); // "Hello Mg Mg!"
greet("Hla Hla"); // "Hello Hla Hla!"
```

ဒီနေရာမှာ `greet(name)` ထဲက `name` ဆိုတာကို **Parameter** လို့ ခေါ်ပြီး၊ အပြင်ကနေ အမှန်တကယ် ထည့်ပေးလိုက်တဲ့ "Mg Mg" နဲ့ "Hla Hla" ဆိုတဲ့ တန်ဖိုးတွေကိုကျတော့ **Argument** လို့ ခေါ်ဆိုပါတယ်။

## Return Values (ရလဒ် ပြန်လည်ပေးပို့ခြင်း)

Function ထဲမှာ `console.log` သုံးတာက မျက်နှာပြင်ပေါ်မှာ စာထုတ်ပြရုံ သက်သက်ပါပဲ။ တကယ့် လက်တွေ့ Project တွေမှာက တွက်ချက်လို့ ရလာတဲ့ အဖြေကို နောက်ထပ် လုပ်ငန်းစဉ်တွေမှာ ဆက်သုံးဖို့အတွက် အပြင်ကို ပြန်ထုတ်ယူချင်တာ များပါတယ်။ အဲ့ဒီအခါမျိုးမှာ `return` ဆိုတဲ့ Keyword ကို အသုံးပြုရပါတယ်။

```javascript
function add(num1, num2) {
  return num1 + num2; // တွက်ချက်ပြီး ရလာတဲ့ အဖြေကို အပြင်သို့ ပြန်ပေးပါမည်
}

let result = add(10, 20); // add Function က ပြန်ပေးလိုက်တဲ့ တန်ဖိုး 30 ကို result ဆိုတဲ့ Variable ထဲမှာ သိမ်းလိုက်ပါတယ်
console.log(result); // ရလဒ်: 30
```

🚨 **အထူး သတိပြုရန်:** Function တစ်ခုအတွင်းမှာ `return` လို့ ရေးထားတဲ့ လိုင်းကို ရောက်ပြီး အလုပ်လုပ်သွားတာနဲ့ တပြိုင်နက်၊ အဲ့ဒီ Function ရဲ့ လုပ်ငန်းစဉ်ဟာ ချက်ချင်း ရပ်တန့်သွားပါလိမ့်မယ်။ `return` ရဲ့ အောက်မှာ ဘယ်လောက်ပဲ Code တွေ ဆက်ရေးထားပါစေ လုံးဝ (လုံးဝ) ဆက်အလုပ်မလုပ်တော့ပါဘူး။
