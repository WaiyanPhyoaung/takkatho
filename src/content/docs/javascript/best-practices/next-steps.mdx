title: "What to Learn Next?"
description: "ဆက်လက်လေ့လာသင့်သော အရာများ"
---

ပထမဆုံး အနေနဲ့ ဂုဏ်ယူပါတယ်။ အခုဆိုရင် သင်ဟာ JavaScript ရဲ့ အရေးကြီးတဲ့ အခြေခံ သဘောတရားတွေကို ပိုင်နိုင်စွာ နားလည်သွားပြီပဲ ဖြစ်ပါတယ်။ ဒါပေမဲ့ Web Development ခရီးစဉ်က ဒီနေရာမှာတင် မဆုံးသေးပါဘူး။ ဆက်လက်ပြီး မြှင့်တင် လေ့လာစရာတွေ အများကြီး ကျန်ရှိနေပါသေးတယ်။

## ၁။ Advanced DOM Manipulation (အဆင့်မြင့် DOM အသုံးပြုနည်းများ)

* **Event Delegation:** Element အသစ်တွေ အများကြီးကို အချိန်နဲ့ တပြေးညီ ဖန်တီးတဲ့အခါ Event တွေကို ထိထိရောက်ရောက် ဘယ်လို ကိုင်တွယ်မလဲ ဆိုတာကို လေ့လာရပါမယ်။
* **Creating Complex Elements:** ရိုးရှင်းတဲ့ Element လေးတွေ သာမကဘဲ၊ Facebook News Feed လိုမျိုး ရှုပ်ထွေးတဲ့ ဖွဲ့စည်းပုံတွေကို JavaScript သီးသန့်နဲ့ ဘယ်လို တည်ဆောက်မလဲ။
* **Form Validation:** User က Form ဖြည့်တဲ့အခါ (ဥပမာ - Email ပုံစံ မှန်မမှန်၊ Password အားကောင်းမှု ရှိမရှိ) အချက်အလက်တွေကို Server ဆီ မပို့ခင် Client-side ကနေ ကြိုတင် စစ်ဆေးတဲ့ နည်းလမ်းများ။

## ၂။ ES6+ Features (ခေတ်မီ JavaScript ရေးထုံးများ)

JavaScript ဟာ အမြဲတမ်း Update ဖြစ်ပေါ် တိုးတက်နေတဲ့ ဘာသာစကား တစ်ခု ဖြစ်ပါတယ်။
* **Destructuring:** Array တွေ၊ Object တွေထဲက Data တွေကို အလွယ်တကူ ဆွဲထုတ် အသုံးပြုနည်း (`const { name } = person;`)
* **Spread / Rest Operators:** `...` သင်္ကေတကို အသုံးပြုပြီး Data တွေကို ဖြန့်ကျက်ခြင်း၊ စုစည်းခြင်းများ။
* **Promises & Async/Await:** အချိန်ယူရတဲ့ အလုပ်တွေ (ဥပမာ - Network ချိတ်ဆက်ခြင်း၊ Data ဆွဲယူခြင်း) တွေကို အဆင်ပြေချောမွေ့စွာ ကိုင်တွယ် ဖြေရှင်းနည်း။
* **Modules:** Code တွေကို သပ်ရပ်အောင် ဖိုင် (File) တွေ ခွဲထုတ်ပြီး `import` / `export` အသုံးပြုကာ ပြန်လည် ချိတ်ဆက်နည်း။

## ၃။ Fetch API & JSON

Backend Server တွေ၊ API တွေဆီကနေ Data တွေ လှမ်းတောင်းယူတာ (ဥပမာ - ရာသီဥတု အချက်အလက်ယူတာ၊ သတင်းတွေ ဆွဲယူတာ) နဲ့ ပြန်ရလာတဲ့ JSON Data တွေကို ကိုယ့် Website ပေါ်မှာ ဘယ်လို ဖော်ပြမလဲ ဆိုတာကို လေ့လာရပါမယ်။

## ၄။ JavaScript Frameworks (React, Vue, Angular စသည်)

**Vanilla JavaScript** (မည်သည့် Library/Framework မှ မပါဝင်သော မူရင်း တင်းလင်း JavaScript) ကို ကောင်းစွာ ကျွမ်းကျင်သွားပြီ ဆိုရင်တော့၊ လက်တွေ့ လုပ်ငန်းခွင်မှာ အများဆုံး အသုံးပြုကြတဲ့ **React** သို့မဟုတ် **Vue** လိုမျိုး ခေတ်မီ Framework တစ်ခုခုကို ဆက်လက် လေ့လာဖို့ အထူး အကြံပေး တိုက်တွန်း လိုပါတယ်။ Framework တွေကို တတ်ကျွမ်းထားခြင်းဖြင့် အလုပ်အကိုင် အခွင့်အလမ်း ပိုမို များပြားလာမှာ ဖြစ်ပါတယ်။

> **အရေးကြီး အကြံပြုချက်:** အခြေခံ JavaScript ကို သေချာ မခိုင်မာသေးဘဲ Framework တွေကို အလျင်စလို ခုန်မကူးပါနဲ့။ အခြေခံ ပိုင်နိုင်ထားမယ် ဆိုရင် မည်သည့် Framework ကိုမဆို လေ့လာတဲ့အခါ အလွန် လွယ်ကူ ချောမွေ့သွားပါလိမ့်မယ်။

**ရှေ့ဆက်ပြီး ကြိုးစားပါ! အောင်မြင်မှုတွေက သင့်ကို စောင့်ကြိုနေပါတယ်။**
