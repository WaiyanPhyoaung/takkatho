### HTML Structure
နံပါတ်ပြမယ့် နေရာနဲ့၊ အတိုးအလျှော့ လုပ်မယ့် ခလုတ်နှစ်ခု လိုပါမယ်။

```html
<div class="counter-app">
  <h1 id="count">0</h1>
  <button id="decrease">-</button>
  <button id="increase">+</button>
</div>
```

### JavaScript Logic
1.  Variable တစ်ခုထားပြီး နံပါတ်ကို မှတ်ထားမယ်။
2.  ခလုတ်တွေကို DOM နဲ့ ဖမ်းမယ်။
3.  Click event တွေမှာ Variable ကို ပြောင်းပြီး Screen ပေါ်မှာ ပြန်ပြမယ်။

```javascript
// 1. State (အခြေအနေ)
let count = 0;

// 2. Elements (ပစ္စည်းများ)
const countDisplay = document.getElementById("count");
const increaseBtn = document.getElementById("increase");
const decreaseBtn = document.getElementById("decrease");

// Function to update display
function updateDisplay() {
  countDisplay.textContent = count;
  
  // အရောင်လေးပါ ပြောင်းကြည့်မယ်
  if (count > 0) countDisplay.style.color = "green";
  else if (count < 0) countDisplay.style.color = "red";
  else countDisplay.style.color = "black";
}

// 3. Events (လုပ်ဆောင်ချက်များ)
increaseBtn.addEventListener("click", () => {
  count++; // ၁ တိုး
  updateDisplay();
});

decreaseBtn.addEventListener("click", () => {
  count--; // ၁ လျှော့
  updateDisplay();
});
```
