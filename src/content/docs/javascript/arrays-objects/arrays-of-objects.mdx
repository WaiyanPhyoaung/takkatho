တကယ့် လက်တွေ့ Project တွေမှာ Array နဲ့ Object ကို သီးခြားစီ သုံးတာထက်၊ သူတို့နှစ်ခုကို ပေါင်းစပ်ပြီး (Array ထဲမှာ Object တွေ ထည့်ပြီး) အသုံးပြုလေ့ ရှိကြပါတယ်။
ဥပမာ - ကျောင်းသားများ စာရင်း၊ ကုန်ပစ္စည်းများ စာရင်း၊ Facebook က Post တွေ စသဖြင့်ပေါ့။

## ဥပမာ - Products List (ကုန်ပစ္စည်း စာရင်း)

```javascript
let products = [
  { id: 1, name: "Phone", price: 500 },
  { id: 2, name: "Laptop", price: 1000 },
  { id: 3, name: "Mouse", price: 20 }
];
```

ဒါဟာ **Object တွေ အများကြီး ပါဝင်နေတဲ့ Array** တစ်ခု ဖြစ်ပါတယ်။ Database ကနေ Data တွေ ယူရင်လည်း အများအားဖြင့် ဒီလို ပုံစံမျိုးနဲ့ပဲ ရရှိမှာ ဖြစ်ပါတယ်။

## Data ဆွဲထုတ်ခြင်း

ဒီ Data အစုအဝေးကြီး ထဲကနေ ကိုယ်လိုချင်တဲ့ အချက်အလက်ကို ဘယ်လို ဆွဲထုတ်မလဲ?

```javascript
// ပထမဆုံး ပစ္စည်း (Index 0) ရဲ့ နာမည် (name) ကို ယူမယ်
console.log(products[0].name); // ရလဒ်: "Phone"

// ဒုတိယ ပစ္စည်း (Index 1) ရဲ့ ဈေးနှုန်း (price) ကို ယူမယ်
console.log(products[1].price); // ရလဒ်: 1000
```

## Looping through Arrays of Objects

ကုန်ပစ္စည်း အားလုံးရဲ့ နာမည်တွေကို တစ်ခုချင်းစီ ထုတ်ပြချင်ရင် For Loop ကို အသုံးပြုပြီး အလွယ်တကူ ထုတ်ပြလို့ ရပါတယ်။

```javascript
for (let i = 0; i < products.length; i++) {
  console.log(products[i].name);
}
// ရလဒ် (Outputs):
// Phone
// Laptop
// Mouse
```

Database သို့မဟုတ် Backend API တွေဆီကနေ Data တွေ တောင်းယူတဲ့အခါ ပြန်ကျလာမယ့် ပုံစံက အများအားဖြင့် အခုလို `Array of Objects` ပုံစံမျိုးပါပဲ။ ဒါကို သေချာ နားလည်ထားပြီ ဆိုရင်တော့ နောက်ပိုင်းမှာ API တွေနဲ့ ချိတ်ဆက် အလုပ်လုပ်တဲ့အခါ အများကြီး လွယ်ကူသွားပါလိမ့်မယ်။
