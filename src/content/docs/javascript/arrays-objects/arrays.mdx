Variable တစ်ခုတည်းမှာ Data တစ်ခုတည်း မဟုတ်ဘဲ၊ Data တွေ အများကြီးကို စုစည်းပြီး သိမ်းဆည်းထားချင်ရင် **Array** ကို အသုံးပြုရပါမယ်။
ဥပမာ - ဈေးဝယ်စာရင်း (Shopping List) လိုမျိုး အချက်အလက် အများကြီး ပါတဲ့ စာရင်းတစ်ခုပေါ့။

## Creating Arrays (Array တည်ဆောက်ခြင်း)

Array ကို တည်ဆောက်တဲ့အခါ `[]` (Square Brackets) ကို အသုံးပြုပြီး ရေးသားရပါတယ်။ Data တစ်ခုနဲ့ တစ်ခုကြားမှာ `,` (Comma) ခံပေးရပါတယ်။

```javascript
let fruits = ["Apple", "Orange", "Banana"]; // String Array
let numbers = [10, 20, 30, 40]; // Number Array
```

## Accessing Elements (Data ယူသုံးခြင်း)

Array ထဲမှာ သိမ်းထားတဲ့ Data တွေကို ပြန်ခေါ်သုံးချင်ရင် **Index Number** (သူတို့ရဲ့ အစဉ်လိုက် နံပါတ်) ကို အသုံးပြုရပါတယ်။
🚨 **အရေးကြီးဆုံး မှတ်ထားရန်:** Programming မှာ Index ဟာ **0 (သုည)** ကနေ စတင်ပါတယ်။ (၁ ကနေ မစဘူးဆိုတာ အမြဲတမ်း သတိရပါ!)

```javascript
console.log(fruits[0]); // ရလဒ်: "Apple" (ပထမဆုံး ဒေတာ)
console.log(fruits[1]); // ရလဒ်: "Orange" (ဒုတိယ ဒေတာ)
```

## Common Array Methods (အသုံးများသော နည်းလမ်းများ)

Array တွေထဲကို Data အသစ်ထပ်ထည့်တာ၊ ဖျက်တာ၊ အရေအတွက် စစ်ဆေးတာတွေ လုပ်ဖို့အတွက် JavaScript မှာ အဆင်သင့် သုံးလို့ရတဲ့ (Built-in) Function လေးတွေ ရှိပါတယ်။

### ၁။ `push()` - အနောက်ဆုံးမှ ထပ်ထည့်မည်
```javascript
fruits.push("Grape");
// ယခု fruits Array: ["Apple", "Orange", "Banana", "Grape"]
```

### ၂။ `pop()` - အနောက်ဆုံးက တစ်ခုကို ဖျက်မည်
```javascript
fruits.pop();
// အနောက်ဆုံးက "Grape" ပြုတ်သွားပြီး ["Apple", "Orange", "Banana"] ပြန်ဖြစ်သွားပါမည်
```

### ၃။ `unshift()` - အရှေ့ဆုံးမှ ထပ်ထည့်မည်
```javascript
fruits.unshift("Mango");
// အရှေ့ဆုံးကို ဝင်သွားမည်: ["Mango", "Apple", "Orange", "Banana"]
```

### ၄။ `shift()` - အရှေ့ဆုံးက တစ်ခုကို ဖျက်မည်
```javascript
fruits.shift();
// အရှေ့ဆုံးက "Mango" ပြုတ်သွားပြီး ["Apple", "Orange", "Banana"] ပြန်ဖြစ်မည်
```

### ၅။ `length` - စုစုပေါင်း အရေအတွက် (Length) ကို ကြည့်မည်
```javascript
console.log(fruits.length); // ရလဒ်: 3 (Data သုံးခု ရှိသောကြောင့်ဖြစ်သည်)
```
