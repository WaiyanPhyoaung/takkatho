description: "Viewport Units များနှင့် clamp() function များ အသုံးပြုပုံ"
---

Mobile မှာ စာလုံးသေးသေးလေးနဲ့ Desktop မှာ စာလုံးအကြီးကြီးဖြစ်အောင် အရင်က Media Queries တွေ အများကြီးရေးပြီး `font-size` တွေကို လိုက်ပြောင်းခဲ့ရပါတယ်။ အခုခေတ်မှာတော့ **Fluid Typography** စနစ်ကို သုံးလာကြပါပြီ။

Fluid Typography ဆိုတာ Screen ကြီးရင် ကြီးသလို စာလုံးက တဖြည်းဖြည်း (Smoothly) ချဲ့သွားပြီး၊ သေးရင် သေးသလို တဖြည်းဖြည်း ကျုံ့သွားတဲ့ သဘောတရားကို ဆိုလိုပါတယ်။

### Viewport Units များကို နားလည်ခြင်း
* `vw` (Viewport Width): မျက်နှာပြင်တစ်ခုလုံးရဲ့ အကျယ် ၁%။ (100vw = မျက်နှာပြင်အကျယ် အပြည့်)
* `vh` (Viewport Height): မျက်နှာပြင်တစ်ခုလုံးရဲ့ အမြင့် ၁%။
* `vi` (Viewport Inline): စာကြောင်းသွားတဲ့ဘက် (Inline) အတိုင်းအတာ (ခေတ်သစ်လော့ဂျစ်)

```css
h1 {
  font-size: 5vw; /* ဖုန်းမှာ သေးသွားပြီး ကွန်ပျူတာမှာ အကြီးကြီးဖြစ်သွားမည် */
}
```
*Note: `vw` ချည်းပဲသက်သက်သုံးလျှင် Mobile မှာ စာလုံးက မြင်ရုံလေးလောက်အထိ သေးသွားတတ်ပါတယ်။*

### `clamp()` Function (အကောင်းဆုံး နည်းလမ်း)
Responsive Typography အတွက် ခေတ်အမီဆုံးနဲ့ အကောင်းဆုံးနည်းလမ်းကတော့ CSS ၏ **`clamp()`** function ဖြစ်ပါတယ်။

`clamp(အငယ်ဆုံး, လက်ရှိပြောင်းလဲမှု, အကြီးဆုံး)` လို့ အလုပ်လုပ်ပါတယ်။

```css
h1 {
  /* 
    - အနည်းဆုံး 1.5rem (24px) ထက် မငယ်ရဘူး
    - Screen အရွယ်အစားပေါ်မူတည်ပြီး 2.5vw + 1rem နှုန်းနဲ့ ကြီး/ငယ်သွားမယ်
    - အကြီးဆုံး 3.5rem (56px) ကို ရောက်ရင် ထပ်မကြီးတော့ဘူး
  */
  font-size: clamp(1.5rem, 2.5vw + 1rem, 3.5rem);
}
```

### `min()` နှင့် `max()` 
Clamp အပြင် တခြား သင်္ချာ function (၂) ခုလည်း ရှိပါသေးတယ်။

#### 1. `min()` - "ဒီနှစ်ခုထဲက အငယ်ဆုံးကို ယူပါ" သဘောပါ။
```css
.container {
  /* 100% ရယ် 1200px ရယ် ယှဉ်ရင် ဘယ်ဟာပိုငယ်လဲ အဲ့ဒါကို ယူမယ်။
     Mobile မှာဆိုရင် 100% က ပိုငယ်လို့ 100% ဖြစ်ပြီး၊ 
     Desktop ကြီးကြီးမှာဆို 1200px က ပိုငယ်လို့ 1200px ပဲ တန့်နေမယ်။ */
  width: min(100%, 1200px);
}
```

#### 2. `max()` - "ဒီနှစ်ခုထဲက အကြီးဆုံးကို ယူပါ" သဘောပါ။
```css
.sidebar {
  /* 300px ရယ် 20vw ရယ် ယှဉ်ပြီး ပိုကြီးတဲ့တန်ဖိုးကို ယူမယ်။ */
  width: max(300px, 20vw);
}
```

`clamp()` ကတော့ Responsive Layouts နဲ့ Typography တွေမှာ မပါမဖြစ် လိုအပ်တဲ့ Modern CSS ရဲ့ အရေးအပါဆုံး အစိတ်အပိုင်းတစ်ခု ဖြစ်လာပါပြီ။
