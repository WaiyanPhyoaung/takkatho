### CSS ဆိုတာ ဘာလဲ?
**CSS (Cascading Style Sheets)** ဆိုတာ Webpage တွေကို လှပအောင် ဒီဇိုင်းဆွဲပေးတဲ့ ဘာသာစကား တစ်ခုပါ။
HTML နဲ့ ရေးထားတဲ့ စာသားတွေ၊ ပုံတွေကို အရောင်ထည့်တာ၊ နေရာချတာ၊ Font ပြောင်းတာတွေ လုပ်ပေးနိုင်ပါတယ်။

### ဘာလို့ CSS လိုအပ်တာလဲ?
CSS မပါတဲ့ Website တစ်ခုဟာ Word Document အလွတ်တစ်ခုလိုပဲ ပျင်းစရာ ကောင်းပါတယ်။
CSS က Website ကို -
-   **လှပစေတယ်:** အရောင်တွေ၊ ဒီဇိုင်းတွေ ထည့်လို့ရတယ်။
-   **Responsive ဖြစ်စေတယ်:** ဖုန်း၊ တက်ဘလက်၊ ကွန်ပျူတာ မျက်နှာပြင် အစုံမှာ ကြည့်လို့ကောင်းအောင် လုပ်ပေးနိုင်တယ်။
-   **User Experience ကောင်းစေတယ်:** သုံးရလွယ်ကူပြီး ဆွဲဆောင်မှုရှိတဲ့ Website တွေ ဖန်တီးနိုင်တယ်။

### CSS ကို ဘယ်လို ရေးမလဲ?
CSS ရေးနည်း (၃) မျိုး ရှိပါတယ်။

#### 1. Inline CSS (Tag ထဲမှာ တိုက်ရိုက်ရေးခြင်း)
HTML tag ထဲမှာ `style` attribute သုံးပြီး ရေးတာပါ။
```html
<h1 style="color: blue;">မင်္ဂလာပါ</h1>
```

#### 2. Internal CSS (Head ထဲမှာ ရေးခြင်း)
`<head>` tag ထဲမှာ `<style>` tag ဖွင့်ပြီး ရေးတာပါ။
```html
<head>
  <style>
    h1 {
      color: red;
    }
  </style>
</head>
```

#### 3. External CSS (သီးသန့် ဖိုင်ခွဲရေးခြင်း) - **အကောင်းဆုံး နည်းလမ်း**
`.css` ဖိုင် သီးသန့်ဆောက်ပြီး HTML နဲ့ ချိတ်ဆက်တာပါ။
```html
<!-- index.html -->
<head>
  <link rel="stylesheet" href="style.css">
</head>
```

```css
/* style.css */
h1 {
  color: green;
}
```

<Aside type="tip">
Project အကြီးတွေမှာ **External CSS** ကို အသုံးများပါတယ်။ ပြင်ရလွယ်ပြီး Code ရှင်းလင်းလို့ပါ။
