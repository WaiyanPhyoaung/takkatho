### နောက်တစ်ဆင့် ဘာဆက်လုပ်မလဲ?

CSS လောကက အရမ်းကျယ်ပြန့်ပါတယ်။ အခြေခံ ရပြီးပြီဆိုတော့ ပိုမိုနက်နဲတဲ့ နည်းပညာတွေကို ဆက်လေ့လာဖို့ အချိန်တန်ပါပြီ။
