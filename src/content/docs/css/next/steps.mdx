### နောက်တစ်ဆင့် ဘာဆက်လုပ်မလဲ?

CSS နည်းပညာ လောကကြီးက တော်တော်လေးကို ကျယ်ပြန့်ပါတယ်။ အခု အခြေခံကို ပိုင်ပိုင်နိုင်နိုင် ရသွားပြီဆိုတော့၊ ဒီထက် ပိုမိုနက်နဲပြီး ဆန်းသစ်တဲ့ နည်းပညာတွေကို ဆက်လက် လေ့လာဖို့ အချိန်တန်ပြီပေါ့ ခင်ဗျာ။

#### 1. Advanced CSS
CSS မှာ အသစ်ထွက်ပေါ်လာတဲ့ နောက်ဆုံးပေါ် Feature တွေ၊ ရှုပ်ထွေးတဲ့ Logic တွေ၊ အဆင့်မြင့် Animation တွေနဲ့ Project ကြီးတွေမှာ CSS ကို ဘယ်လို စနစ်တကျ ရေးရမလဲဆိုတဲ့ Architecture ပိုင်းတွေကို ဆက်လက် လေ့လာကြည့်ပါ။

#### 2. JavaScript
Website တွေကို ဒီအတိုင်း ရပ်နေတာမျိုး မဟုတ်ဘဲ၊ User တွေနဲ့ အပြန်အလှန် တုံ့ပြန်နိုင်အောင် (Interactive ဖြစ်ပြီး လှုပ်ရှားသက်ဝင်နေအောင်) ဖန်တီးဖို့ဆိုရင်တော့ JavaScript ကို မဖြစ်မနေ လေ့လာဖို့ လိုအပ်လာပါပြီ။

#### 3. CSS Libraries
Website တစ်ခုကို ရေးတဲ့အခါ CSS တွေကို အစကနေ အကုန်လုံး ကိုယ်တိုင် လိုက်ရေးနေစရာ မလိုတော့ဘဲ၊ မြန်မြန်ဆန်ဆန်နဲ့ လွယ်လွယ်ကူကူ တည်ဆောက်နိုင်ဖို့အတွက် အောက်ပါ နာမည်ကြီး Library / Framework တွေကိုလည်း လေ့လာကြည့်သင့်ပါတယ် -
*   **Tailwind CSS** (Utility-first)
*   **Bootstrap** (Component-based)
