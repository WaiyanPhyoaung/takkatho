Website တစ်ခုရဲ့ Mood လို့ခေါ်တဲ့ စိတ်ခံစားမှု အသက်ဝင်လာအောင် အရောင် (Colors) တွေက အဓိက ဖန်တီးပေးပါတယ် ခင်ဗျာ။

### Colors (စာလုံးအရောင်)
စာသားတွေရဲ့ အရောင်ကို ပြောင်းချင်တယ်ဆိုရင် `color` Property ကို အသုံးပြုနိုင်ပါတယ်။

CSS မှာ အရောင်တွေကို သတ်မှတ်ဖို့ အဓိက နည်းလမ်း (၃) မျိုး ရှိပါတယ် -

1.  **Color Name:** `red`, `blue`, `green` လိုမျိုး အရောင်နာမည်တွေကို တိုက်ရိုက် ခေါ်သုံးတာပါ။ (ဒါကတော့ အလွယ်တကူ စမ်းသပ်ဖို့အတွက် အကောင်းဆုံးပါ)
2.  **Hex Code:** `#FF0000`, `#000000` လိုမျိုး Code တွေနဲ့ ရေးတာပါ။ (မိမိလိုချင်တဲ့ အရောင် အတိအကျကို သုံးချင်ရင် ဒါကို အသုံးများပါတယ်)
3.  **RGB:** `rgb(255, 0, 0)` လိုမျိုး အနီ (Red)၊ အစိမ်း (Green) နဲ့ အပြာ (Blue) အရောင်သုံးခုကို ရောစပ်ပြီး သုံးတဲ့နည်းလမ်း ဖြစ်ပါတယ်။

```css
h1 {
  color: tomato; /* Color Name */
}

p {
  color: #333333; /* Hex Code */
}
```

### Backgrounds (နောက်ခံ)
Element တစ်ခုရဲ့ နောက်ခံ (Background) ကို ပြောင်းလဲချင်ရင်တော့ `background-color` သို့မဟုတ် `background-image` ကို သုံးလို့ရပါတယ်။

#### Background Color
```css
body {
  background-color: #f0f0f0; /* တစ်မျက်နှာလုံး မီးခိုးရောင်ဖျော့ဖျော့ */
}

.card {
  background-color: white;
}
```

#### Background Image
ပုံတစ်ပုံကို နောက်ခံအဖြစ် ထည့်ချင်တယ်ဆိုရင်တော့ `url()` ကို သုံးပြီး ပုံရှိတဲ့ လမ်းကြောင်း (Path) ကို ညွှန်းပေးရပါတယ်။

```css
.hero-section {
  background-image: url('hero-bg.jpg');
  background-size: cover; /* ပုံကို အပြည့်ဖြန့်မယ် */
  background-position: center; /* ပုံကို အလယ်တည့်တည့် ထားမယ် */
}
```

<Aside type="tip">
စာလုံးအရောင်နဲ့ နောက်ခံအရောင်ကို ရွေးချယ်တဲ့အခါ **Contrast** (အရောင် ကွဲပြားမှု) ရှင်းရှင်းလင်းလင်း ရှိအောင် သတိထားပါ။
ဥပမာ - နောက်ခံက အမည်းရောင်ဆိုရင် စာလုံးကို အဖြူရောင် ထားပေးမှသာ User တွေအတွက် ဖတ်ရ လွယ်ကူမှာ ဖြစ်ပါတယ် ခင်ဗျာ။
</Aside>
