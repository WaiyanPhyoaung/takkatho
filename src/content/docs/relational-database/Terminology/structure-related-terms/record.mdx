## Record (Row) ဆိုတာ ဘာလဲ?

**Record** ဆိုတာ Table တစ်ခုရဲ့ အေနာက်ဘက် ဒေါင်လိုက် Fields တွေနဲ့ ဖွဲ့စည်းထားတဲ့ အလျားလိုက် အတန်း (Row) တစ်ခုကို ခေါ်တာ ဖြစ်ပါတယ်။

Table တစ်ခုဟာ သီးသန့် Entity တစ်ခုအကြောင်း (ဥပမာ - User) ကို သိမ်းဆည်းထားတာဖြစ်ပြီး၊ အဲ့ဒီ Table ထဲက **Record** တစ်ခုချင်းစီဟာ Entity ရဲ့ သီးခြား Real-world Data Instance တစ်ခုစီ (ဥပမာ - သီးသန့် User တစ်ယောက်၏ အချက်အလက်များ) ကို ကိုယ်စားပြုပါတယ်။

### အခေါ်အဝေါ် ကွဲပြားမှုများ:
- **Row:** Database Systems များတွင် အသုံးအများဆုံး အခေါ်အဝေါ် ဖြစ်ပါတယ်။
- **Record:** Data Processing & Software Development တွင် အသုံးအများဆုံး အခေါ်အဝေါ် ဖြစ်ပါတယ်။
- **Tuple:** Relational Database Theory သီအိုရီ စာအုပ်များတွင် ခေါ်ဆိုလေ့ရှိသော အခေါ်အဝေါ် ဖြစ်ပါတယ်။

## လက်တွေ့ ဥပမာ - `employees` Table

| ID | Name | Position |
|----|------|----------|
| 1 | Alice | Director |
| 2 | Bob | Manager |

ဒီ `employees` Table မှာ ID `1`, Name `Alice`, Position `Director` ဆိုတဲ့ အလျားလိုက် အတန်းတစ်တန်းလုံးကို **Record (Row)** လို့ ခေါ်ဆိုပါတယ်။
