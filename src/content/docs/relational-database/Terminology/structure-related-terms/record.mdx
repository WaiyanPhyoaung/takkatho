## Record (Row) ဆိုတာ ဘာလဲ?

**Record** ဆိုတာ Table တစ်ခုရဲ့ အတန်း (Row) ကို ခေါ်တာပါ။
Table တစ်ခုဟာ (Entity လို့ခေါ်တဲ့) အကြောင်းအရာ တစ်ခုအကြောင်းကို သိမ်းပေးတာဖြစ်ပြီး၊ အဲဒီ Table ထဲက Record တစ်ခုချင်းစီဟာ အဲဒီ အကြောင်းအရာရဲ့ တကယ့် အချက်အလက် (Instance) တစ်ခုချင်းစီကို ကိုယ်စားပြုပါတယ်။

### ခေါ်ဝေါ်သုံးစွဲပုံ ကွဲပြားမှုများ
- **Row:** Database ပိုင်းမှာ အများဆုံး သုံးတဲ့ အခေါ်အဝေါ်ပါ။
- **Record:** ယေဘုယျအားဖြင့် အများဆုံး သုံးကြတဲ့ အခေါ်အဝေါ်ပါ။
- **Tuple:** Relational Database သီအိုရီ (Theory) အပိုင်းတွေမှာ သုံးလေ့ရှိတဲ့ အခေါ်အဝေါ်ပါ။ 

## ဥပမာ - ဝန်ထမ်းစာရင်း (Employees Table)

| ID | Name | Position |
|----|------|----------|
| 1 | Alice | Director |
| 2 | Bob | Manager |

ဒီ `Employees` Table မှာ ဝန်ထမ်းတစ်ယောက်စီရဲ့ အချက်အလက်တွေ (ဥပမာ - Alice ရဲ့ အချက်အလက်များ) ကို သိမ်းဆည်းထားတဲ့ အတန်း တစ်တန်းလုံးကို **Record (သို့မဟုတ်) Row** လို့ ခေါ်ပါတယ်။
