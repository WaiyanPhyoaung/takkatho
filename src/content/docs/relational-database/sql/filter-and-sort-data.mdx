## Data စစ်ထုတ်ခြင်း (The WHERE Clause)

Table တစ်ခုထဲမှာ Data တွေ သောင်းချီ သိမ်းထားပေမယ့်၊ တကယ်တမ်း လိုချင်တာက "လစာ ၅၀,၀၀၀ အထက် ဝန်ထမ်းများ" ဆိုတဲ့ အချက်အလက်မျိုးလေးပဲ ဖြစ်နိုင်ပါတယ်။ ဒီလို ကိုယ်လိုချင်တဲ့ Data အတိအကျကိုပဲ စစ်ထုတ် (Filter) ယူဖို့အတွက် `WHERE` Clause ကို အသုံးပြုရပါတယ်။

### Syntax
```sql
SELECT column1, column2 
FROM table_name 
WHERE condition;
```

### ဥပမာ - "IT" ဌာနက ဝန်ထမ်းများကိုသာ ရွေးထုတ်ပါ။
```sql
SELECT FirstName, LastName, Department
FROM Employees
WHERE Department = 'IT';
```
*(ဒီ Query က `Department` Column မှာ 'IT' လို့ ရေးထားတဲ့ Record တွေကိုပဲ ရွေးထုတ်ပေးပါလိမ့်မယ်။)*

## Comparison Operators (နှိုင်းယှဉ်ခြင်းများ)

`WHERE` ထဲမှာ စစ်ထုတ်ဖို့အတွက် အောက်ပါ နှိုင်းယှဉ်တဲ့ သင်္ကေတတွေကို အသုံးများပါတယ်။
- `=` (ညီမျှသည်)
- `<>` သို့မဟုတ် `!=` (မညီမျှပါ)
- `>` (ကြီးသည်)
- `<` (ငယ်သည်)
- `>=` (ကြီးသည် သို့မဟုတ် ညီသည်)
- `<=` (ငယ်သည် သို့မဟုတ် ညီသည်)

### ဥပမာ - လစာ $75,000 ထက်များသော ဝန်ထမ်းများကို ရွေးထုတ်ပါ။
```sql
SELECT FirstName, Salary 
FROM Employees
WHERE Salary > 75000;
```

---

## Logical Operators (ယုတ္တိဗေဒ သင်္ကေတများ)

Condition (အခြေအနေ) တစ်ခုထက်မကကို ပေါင်းစပ်စစ်ဆေးချင်ရင် `AND`, `OR`, `NOT` တို့ကို သုံးပါတယ်။

- **`AND`:** Condition နှစ်ခုစလုံး မှန်မှ အလုပ်လုပ်ပါတယ်။
- **`OR`:** Condition နှစ်ခုထဲက တစ်ခုမှန်တာနဲ့ အလုပ်လုပ်ပါတယ်။
- **`NOT`:** သတ်မှတ်ထားတဲ့ Condition နဲ့ မကိုက်ညီတဲ့ ဟာတွေကို ရွေးထုတ်ပါတယ်။

### ဥပမာ - "HR" ဌာနကလည်းဖြစ်၊ လစာလည်း $61,000 အောက် ရတဲ့သူကို ရှာပါ။
```sql
SELECT FirstName, Department, Salary
FROM Employees
WHERE Department = 'HR' AND Salary < 61000;
```

## Pattern Matching (LIKE Operator)

စာသား (Text) တွေထဲမှာ ကိုယ်လိုချင်တဲ့ စကားလုံး ပါ/မပါ ရှာဖွေဖို့ (Search လုပ်ဖို့) `LIKE` ကို အသုံးပြုပါတယ်။ 
`LIKE` နဲ့အတူ Wildcard သင်္ကေတ `%` ကို တွဲသုံးလေ့ရှိပါတယ်။ `%` ဟာ "မည်သည့်စာသား မဆို" လို့ အဓိပ္ပာယ်ရပါတယ်။

- `'A%'` = "A" နဲ့ စတဲ့ စာသား အားလုံး (Apple, Aung Aung)
- `'%son'` = "son" နဲ့ ဆုံးတဲ့ စာသား အားလုံး (Jackson, Johnson)
- `'%IT%'` = ကြားထဲမှာ "IT" ပါတဲ့ စာသား အားလုံး

### ဥပမာ - နောက်ဆုံးနာမည် (LastName) 'S' နှင့် စသော ဝန်ထမ်းများကို ရှာပါ။
```sql
SELECT FirstName, LastName 
FROM Employees
WHERE LastName LIKE 'S%';
```

## Data များကို စီစဉ်ပြသခြင်း (The ORDER BY Clause)

ရလာတဲ့ Data တွေကို အက္ခရာစဉ် (A-Z) အတိုင်းဖြစ်စေ၊ ကိန်းဂဏန်း အကြီး/အငယ် အတိုင်းဖြစ်စေ အစီအစဉ်တကျ ပြသချင်ရင် `ORDER BY` Clause ကို အသုံးပြုရပါတယ်။

- **`ASC` (Ascending):** အငယ်မှ အကြီးသို့ (A-Z) စီပေးပါတယ်။ (ပုံသေ သတ်မှတ်ချက် ဖြစ်ပါတယ်)
- **`DESC` (Descending):** အကြီးမှ အငယ်သို့ (Z-A) ပြောင်းပြန် စီပေးပါတယ်။

### ဥပမာ - လစာ အများဆုံးမှ အနည်းဆုံးသို့ အစဉ်လိုက် စီပြပါ။
```sql
SELECT FirstName, Salary
FROM Employees
ORDER BY Salary DESC;
```
*(ဒီ Query က လစာအများဆုံးရတဲ့ ဝန်ထမ်းကို ထိပ်ဆုံးကနေ စပြီး အစဉ်လိုက် ပြပေးပါလိမ့်မယ်။)*
