    Database ထဲက Table တွေမှာ Data အသစ်တွေ ထည့်တာ (`INSERT`)၊ ရှိပြီးသား Data ကို ပြင်တာ (`UPDATE`) နဲ့ မလိုတော့တဲ့ Data ကို ဖျက်တာ (`DELETE`) တွေကို ဘယ်လို လုပ်ဆောင်ရမလဲ ဆိုတာကို လေ့လာကြပါမယ်။

## 1. Data အသစ် ထည့်သွင်းခြင်း (INSERT INTO)

Table ထဲကို အတန်း (Row) အသစ်တစ်ခု ထည့်ချင်ရင် `INSERT INTO` ကို အသုံးပြုရပါတယ်။

### Syntax
```sql
INSERT INTO table_name (column1, column2, column3)
VALUES (value1, value2, value3);
```
- ပထမကွင်းစစ ထဲမှာ Data ထည့်ချင်တဲ့ Column နာမည်တွေ ရေးရပါတယ်။
- ဒုတိယ `VALUES` နောက်က ကွင်းစစ ထဲမှာတော့ တကယ့် Data (တန်ဖိုး) တွေကို အစဉ်လိုက် ရေးပေးရပါတယ်။ (စာသားဆိုရင် Single Quote `''` ထဲ ထည့်ရပါတယ်။)

### ဥပမာ - ဝန်ထမ်းအသစ် တစ်ယောက် ထည့်ပါ။
```sql
INSERT INTO Employees (EmployeeID, FirstName, LastName, Department, Salary, HireDate)
VALUES (106, 'Frank', 'Thomas', 'Sales', 55000.00, '2023-11-01');
```

> **မှတ်ချက်:** Primary Key `EmployeeID` က အလိုအလျောက် တိုးတဲ့ (Auto Increment) အမျိုးအစား ဆိုရင်တော့ Column စာရင်းမှာ `EmployeeID` ကို ထည့်ရေးစရာ မလိုပါဘူး။ Database က သူ့ဘာသာ နံပါတ်စဉ် တပ်ပေးသွားပါလိမ့်မယ်။

## 2. ရှိပြီးသား Data ကို ပြင်ဆင်ခြင်း (UPDATE)

Table ထဲမှာ ရှိပြီးသား Data တွေကို အချက်အလက် အသစ်နဲ့ အစားထိုး ပြင်ဆင်ချင်ရင် `UPDATE` ကို အသုံးပြုရပါတယ်။

### Syntax
```sql
UPDATE table_name
SET column1 = value1, column2 = value2
WHERE condition;
```

### ဥပမာ - Alice ရဲ့ လစာကို $65,000 သို့ တိုးပေးပါ။
```sql
UPDATE Employees
SET Salary = 65000.00
WHERE FirstName = 'Alice';
```

> **[WARNING] သတိပြုရန်:** `UPDATE` လုပ်တဲ့အခါ `WHERE` Clause မပါခဲ့ရင် Table ထဲမှာ ရှိသမျှ Row တွေ အကုန်လုံး ပြင်ဆင်သွားပါလိမ့်မယ်။ ဒါကြောင့် **`WHERE` ထည့်ဖို့ ဘယ်တော့မှ မမေ့ပါနဲ့။** (အကောင်းဆုံးကတော့ နာမည်နဲ့ မရှာဘဲ Primary Key ဖြစ်တဲ့ `EmployeeID` နဲ့ တိတိကျကျ ရှာပြီး ပြင်သင့်ပါတယ်။)

---

## 3. Data ကို ဖျက်ပစ်ခြင်း (DELETE)

Table ထဲက မလိုတော့တဲ့ Record တွေကို ဖယ်ရှားပစ်ချင်ရင် `DELETE` ကို အသုံးပြုရပါတယ်။

### Syntax
```sql
DELETE FROM table_name
WHERE condition;
```

### ဥပမာ - EmployeeID 105 ဝန်ထမ်းကို အလုပ်က ထွက်သွား၍ ဖျက်ပစ်ပါ။
```sql
DELETE FROM Employees
WHERE EmployeeID = 105;
```

> **[CAUTION] အလွန် သတိပြုရန်:** `DELETE` မှာလည်း `WHERE` Clause မပါရင် Table ထဲက Data တွေ အကုန်လုံး ပျက်သွားပါလိမ့်မယ်! Data အကုန် ဖျက်ချင်တယ် ဆိုရင်တောင် `DELETE` အစား `TRUNCATE TABLE` ကို သုံးတာက ပိုမြန်ဆန်ပါတယ်။
