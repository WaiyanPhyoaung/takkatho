## အခြေခံအကျဆုံး - SELECT နှင့် FROM

SQL မှာ Data ဆွဲထုတ်တဲ့ Query တိုင်းဟာ အောက်ပါ အဓိက Keyword ၂ ခုနဲ့ စတင်ရပါတယ်။
- **`SELECT`:** ကိုယ်လိုချင်တဲ့ Column (အကွက်) တွေကို သတ်မှတ်ဖို့ ("ဘာတွေကို လိုချင်တာလဲ")
- **`FROM`:** Data တွေ ဆွဲထုတ်ယူမယ့် Table ကို သတ်မှတ်ဖို့ ("ဘယ်ကနေ ယူမှာလဲ")

### 1. Column အားလုံးကို ဆွဲထုတ်ခြင်း

Table ထဲမှာရှိတဲ့ Column တွေ အကုန်လုံး (အချက်အလက် အပြည့်အစုံ) ကို ကြည့်ချင်တယ်ဆိုရင် ကြယ်ပွင့်သင်္ကေတ `*` (Asterisk) ကို အသုံးပြုရပါတယ်။ `*` ဆိုတာ "အားလုံး (All)" လို့ အဓိပ္ပာယ်ရပါတယ်။

```sql
SELECT * FROM Employees;
```
*(ဒီ Query က အပေါ်မှာ ပြထားတဲ့ `Employees` Table တစ်ခုလုံးကို ပြန်ထုတ်ပေးပါလိမ့်မယ်။)*

### 2. သတ်မှတ်ထားသော Column များကိုသာ ရွေးထုတ်ခြင်း

ကိုယ် တကယ်လိုအပ်တဲ့ Column နာမည်တွေကိုပဲ `SELECT` နောက်မှာ ကော်မာ (`,`) ခြားပြီး ရေးပေးလို့ ရပါတယ်။

**ဥပမာ:** ဝန်ထမ်းတွေရဲ့ နာမည်နဲ့ ဌာနကိုပဲ ကြည့်ချင်တယ် ဆိုပါစို့။

```sql
SELECT FirstName, LastName, Department 
FROM Employees;
```

> **အကြံပြုချက်:** `SELECT *` က အချက်အလက်တွေကို အမြန်စမ်းသပ်ကြည့်ဖို့ အဆင်ပြေပေမယ့်၊ တကယ့် Application တွေ ရေးတဲ့အခါ (Production မှာ) ကိုယ်လိုချင်တဲ့ Column တွေကိုပဲ တိတိကျကျ ရွေးထုတ်တာက ပိုမြန်ဆန်ပြီး အလုပ်တွင်စေပါတယ်။

## Column များကို အမည်ပြောင်းခြင်း (Alias - AS)

Database ထဲက Column နာမည်တွေ (ဥပမာ `emp_first_name`) က ဖတ်ရခက်နေရင်၊ Query ရလဒ်မှာ ပြမယ့် ခေါင်းစဉ်နာမည်ကို `AS` Keyword သုံးပြီး ကိုယ်ကြိုက်တဲ့ ယာယီနာမည် (Alias) ပြောင်းပေးလို့ ရပါတယ်။

```sql
SELECT FirstName AS "Name", Salary AS "Annual Salary"
FROM Employees;
```

- ဒီ Query မှာ `FirstName` အစား "Name" လို့ ပြပေးသွားမှာ ဖြစ်ပြီး၊ `Salary` အစား "Annual Salary" လို့ ပြပေးမှာ ဖြစ်ပါတယ်။
- နာမည်အသစ်မှာ Space တွေ ပါနေရင် (သို့) SQL Keyword တွေ ဖြစ်နေရင် Double Quote `""` နဲ့ ဝန်းရံပေးဖို့ လိုပါတယ်။

## Data အထပ်များကို ဖယ်ရှားခြင်း (DISTINCT)

Column တစ်ခုတည်းမှာ တန်ဖိုးတွေ ထပ်နေတတ်ပါတယ်။ ဥပမာ `Department` Column မှာ 'IT' ဆိုတာ ၂ ခါ ပါနေပါတယ်။ အဲဒီလို ထပ်နေတာတွေ မပါဘဲ မတူညီတဲ့ (Unique) ဌာနစာရင်း အတိအကျကိုပဲ လိုချင်ရင် `DISTINCT` ကို သုံးရပါတယ်။

```sql
SELECT DISTINCT Department 
FROM Employees;
```

**ရလဒ်:** (HR, IT, Marketing) ၃ ခုပဲ ထွက်လာပါလိမ့်မယ်။

## Query ထဲတွင် တိုက်ရိုက် တွက်ချက်ခြင်း

`SELECT` Statement ထဲမှာ Column တွေကို သုံးပြီး သင်္ချာတွက်ချက်မှု (`+`, `-`, `*`, `/`) တွေ တိုက်ရိုက် လုပ်လို့ ရပါတယ်။ 

**ဥပမာ:** ဝန်ထမ်းတွေရဲ့ လစဉ်လစာ (တစ်နှစ်စာ လစာကို ၁၂ နဲ့ စား) ကို တွက်ထုတ်ချင်တယ် ဆိုပါစို့။

```sql
SELECT FirstName, (Salary / 12) AS "Monthly Salary"
FROM Employees;
```
*(ဒါဆိုရင် `Monthly Salary` ဆိုတဲ့ Column အသစ်လေးတစ်ခု ထွက်လာပြီး လစဉ်လစာကို အလိုအလျောက် တွက်ပြသွားပါလိမ့်မယ်။)*
