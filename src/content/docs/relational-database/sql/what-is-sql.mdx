## SQL (Structured Query Language) ဆိုတာ ဘာလဲ?

**SQL** (အက်စ်-ကျူ-အယ်လ် သို့မဟုတ် စီးကွက် လို့ ဖတ်ပါတယ်) ဆိုတာ **Structured Query Language** ရဲ့ အတိုကောက်ပါ။ Relational Database တွေ (ဥပမာ - MySQL, PostgreSQL, SQL Server) ထဲမှာ ရှိတဲ့ Data တွေကို ဆက်သွယ်ဖို့၊ စီမံခန့်ခွဲဖို့၊ ရှာဖွေဖို့နဲ့ ပြင်ဆင်ဖို့အတွက် အသုံးပြုတဲ့ စံသတ်မှတ်ချက် (Standard) ဘာသာစကား တစ်ခု ဖြစ်ပါတယ်။

လူတွေအချင်းချင်း ဆက်သွယ်ဖို့ စကားပြောသလိုပဲ၊ Database ဆီကနေ ကိုယ်လိုချင်တဲ့ Data တောင်းဖို့၊ အသစ်ထည့်ဖို့ Database ကို လှမ်းပြောရတဲ့ (Query လုပ်ရတဲ့) ဘာသာစကား ဖြစ်ပါတယ်။

## SQL နဲ့ ဘာတွေ လုပ်လို့ ရလဲ?

SQL ကို အသုံးပြုပြီး အောက်ပါ အဓိက လုပ်ငန်းဆောင်တာ ၄ မျိုး (CRUD) ကို လုပ်ဆောင်နိုင်ပါတယ်။

1. **C**reate: Data အသစ်တွေ (Row အသစ်တွေ) ထည့်သွင်းခြင်း (ဥပမာ - `INSERT`)
2. **R**ead: ကိုယ်လိုချင်တဲ့ Data တွေကို ရှာဖွေ ဆွဲထုတ်ခြင်း (ဥပမာ - `SELECT`)
3. **U**pdate: ရှိပြီးသား Data တွေကို ပြင်ဆင်ခြင်း (ဥပမာ - `UPDATE`)
4. **D**elete: မလိုတော့တဲ့ Data တွေကို ဖျက်ပစ်ခြင်း (ဥပမာ - `DELETE`)

ဒါ့အပြင် Table အသစ်တွေ တည်ဆောက်တာ၊ Database လုံခြုံရေးအတွက် Permission တွေ ပေးတာ စတဲ့ Database တစ်ခုလုံးကို စီမံခန့်ခွဲတဲ့ အလုပ်တွေကိုလည်း SQL သုံးပြီး လုပ်ဆောင်နိုင်ပါတယ်။

## SQL က ခက်သလား?

SQL ဟာ Programmer တွေသုံးတဲ့ C++, Java တို့လို ရှုပ်ထွေးတဲ့ Programming Language မျိုး မဟုတ်ပါဘူး။ သူက အင်္ဂလိပ်စာကြောင်းတွေနဲ့ အတော်လေး တူပါတယ်။ 

ဥပမာ - "Customers Table ထဲကနေ နိုင်ငံက USA ဖြစ်တဲ့ လူတွေရဲ့ နာမည်ကို ပြပါ" ဆိုရင် `SELECT Name FROM Customers WHERE Country = 'USA';` လို့ ရေးရုံပါပဲ။ ဒါကြောင့် ဖတ်ရလွယ်ကူပြီး နားလည်ရ မြန်ဆန်ပါတယ်။
