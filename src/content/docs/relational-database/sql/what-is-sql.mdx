## SQL ဆိုတာဘာလဲ။

SQL (Structured Query Language) ဆိုတာ relational database တွေနဲ့ စကားပြောဖို့ အသုံးပြုတဲ့ standard programming language တစ်ခုဖြစ်ပါတယ်။ 
Database ထဲက data တွေကို ဆွဲထုတ်တာ၊ အသစ်ထည့်တာ၊ ပြင်ဆင်တာ၊ ဖျက်ပစ်တာတွေ လုပ်ဆောင်ဖို့နဲ့ database ရဲ့ ဖွဲ့စည်းပုံတွေကို စီမံခန့်ခွဲဖို့အတွက် အသုံးပြုပါတယ်။

ဥပမာ - 

database ဆိုတာက စာကြည့်တိုက်ကြီးတစ်ခု ဆိုရင် 
SQL ဆိုတာကတော့ အဲဒီစာကြည့်တိုက်မှာရှိတဲ့ စာကြည့်တိုက်မှူးကို "ဒီစာအုပ်ရှာပေးပါ"၊ "စာအုပ်အသစ်ထပ်ထည့်ပေးပါ"၊ "စာအုပ်အချက်အလက်ပြင်ပေးပါ" လို့ တိတိကျကျ ပြောဆိုညွှန်ကြားနိုင်တဲ့ အထူးဘာသာစကားတစ်ခုဖြစ်ပါတယ်။


## SQL ကိုဘာအတွက် အသုံးပြုလဲ။
SQL ကို အဓိကလုပ်ဆောင်ချက် (Create, Read, Update, Delete) လေးခုအတွက် အဓိက အသုံးပြုပေမဲ့ SQL က ဒါတွေထက် အများကြီး ပိုလုပ်နိုင်ပါသေးတယ်။

### 1. Querying Data

SQL ရဲ့ အသုံးအများဆုံး လုပ်ဆောင်ချက်ဖြစ်ပါတယ်။ Database ထဲမှာ သိမ်းဆည်းထားတဲ့ အချက်အလက်တွေကို သတ်မှတ်ထားတဲ့ commands တွေနဲ့ ပြန်လည်ဆွဲထုတ်ကြည့်ရှုတာ ဖြစ်ပါတယ်။

Commands: SELECT

### 2. Data Manipulation Language - DML

Table တွေထဲမှာသိမ်းဆည်းထားတဲ့ data တွေကို အသစ်ထည့်တာ၊ ပြင်ဆင်တာ၊ ဖျက်ပစ်တာတွေ လုပ်ဆောင်နိုင်ပါတယ်။

Commands: INSERT, UPDATE, DELETE

### 3. Data Definition Language - DDL

Database ရဲ့ ဖွဲ့စည်းပုံ (structure) တွေကို တည်ဆောက်တာ၊ ပြင်ဆင်တာ၊ ဖျက်သိမ်းတာတွေ လုပ်ဆောင်နိုင်ပါတယ်။ Table တွေ၊ Index တွေကို တည်ဆောက်တာမျိုးတွေ ပါဝင်ပါတယ်။

Commands: CREATE, ALTER, DROP

### 4. Data Control Language - DCL

Database ကို ဘယ်သူတွေက သုံးစွဲခွင့်ရှိလဲ၊ ဘယ်လိုအဆင့်အထိ သုံးစွဲခွင့်ရှိလဲဆိုတဲ့ ခွင့်ပြုချက် (permissions) တွေကို စီမံခန့်ခွဲနိုင်ပါတယ်။

Commands: GRANT, REVOKE


## Key Characteristics

SQL မှာ သိထားသင့်တဲ့ အဓိက အချက်နှစ်ခုရှိပါတယ်။

**Declarative:** သင်က "ဘာလုပ်ချင်တယ်" ဆိုတာကိုပဲ SQL နဲ့ ရေးရတာဖြစ်ပါတယ်။ အဲဒါကို "ဘယ်လိုလုပ်ရမယ်" ဆိုတဲ့ အဆင့်ဆင့်လုပ်ဆောင်ပုံကို ရေးစရာမလိုပါဘူး။
ဥပမာ - "ရန်ကုန်မြို့က customers အားလုံးကို ရှာပေးပါ" လို့ သင်က SQL နဲ့ရေးလိုက်တာနဲ့ database က အဲဒီအချက်အလက်ကို ဘယ်လိုအထိရောက်ဆုံးဖြစ်အောင် ရှာဖွေရမလဲဆိုတာကို သူ့ဘာသာသူ စီမံဆောင်ရွက်သွားမှာဖြစ်ပါတယ်။

**Standardized:** SQL ဟာ ANSI/ISO တို့ရဲ့ စံသတ်မှတ်ထားတဲ့ ဘာသာစကားတစ်ခုဖြစ်ပါတယ်။ ဒါကြောင့် PostgreSQL, MySQL, SQL Server, Oracle စတဲ့ မတူညီတဲ့ database system တွေကြားမှာ အနည်းငယ်စီ ကွဲပြားမှုတွေရှိပေမယ့်
SELECT, INSERT, CREATE စတဲ့ အဓိက command တွေကတော့ အကုန်အတူတူပဲ ဖြစ်ပါတယ်။
