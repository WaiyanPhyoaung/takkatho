## Database ဆိုတာ ဘာလဲ?

**Database** ဆိုတာ ကိုယ်သိမ်းချင်တဲ့ အချက်အလက် (Data) တွေကို စနစ်တကျ စုစည်းသိမ်းဆည်းပေးတဲ့ နေရာတစ်ခုပါ။ အပြင်ကမ္ဘာမှာဆိုရင်တော့ စာရွက်စာတမ်းတွေကို သပ်သပ်ရပ်ရပ် ဖိုင်တွဲပြီး သိမ်းထားတဲ့ စာရွက်စာတမ်းစင် (Filing Cabinet) ကြီးနဲ့ တူပါတယ်။ 

အခုခေတ်မှာတော့ Database အများစုကို ဒစ်ဂျစ်တယ်ပုံစံ (Digital Format) နဲ့ ကွန်ပျူတာတွေ၊ ဆာဗာ (Server) တွေပေါ်မှာ သိမ်းဆည်းလာကြပါပြီ။

### Database ရဲ့ အဓိက အခန်းကဏ္ဍ

Software တွေ၊ Website တွေ ဒါမှမဟုတ် Application တော်တော်များများက Data တွေနဲ့ပဲ အသက်ဆက်နေရတာပါ။ User တွေရဲ့ အချက်အလက်တွေ (ဥပမာ - နာမည်၊ Email၊ Password) ကို သေချာသိမ်းထားပေးရတယ်။ ပြီးရင် အဲဒီ Data တွေကို ပြန်ရှာတာ၊ အသစ်ထပ်ထည့်တာ၊ ပြင်တာ ဒါမှမဟုတ် ဖျက်ပစ်တာမျိုးတွေကို လုပ်ပေးဖို့ လိုအပ်လာပါတယ်။ ဒီလို အလုပ်တွေကို လုပ်ဖို့အတွက် **Database** တွေက မရှိမဖြစ်ကို လိုအပ်ပါတယ်။

Software Application တော်တော်များများမှာ အောက်ပါ အဓိက အစိတ်အပိုင်း ၃ ခု ပါဝင်လေ့ရှိပါတယ်:

1. **User Interface (UI):** User တွေ မျက်စိနဲ့ မြင်ရတဲ့ အပိုင်း (Frontend)။
2. **Business Logic:** ဘယ်လို အလုပ်လုပ်ရမယ်ဆိုတဲ့ တွက်ချက်မှုအပိုင်း (Backend)။
3. **Database:** အချက်အလက် (Data) တွေကို သိမ်းဆည်းပေးတဲ့ အပိုင်း။

ဒီသုံးခုမှာ **Database** ဟာ Application တစ်ခုလုံးရဲ့ အသက်သွေးကြော (Core Component) လို့တောင် ဆိုနိုင်ပါတယ်။
