### 1. Chatbots (အမေးအဖြေ လုပ်ရန်)

AI Chatbot တွေကို မေးခွန်းတွေ မေးဖို့၊ အကြံဉာဏ် တောင်းဖို့နဲ့ Code တွေ ရှင်းခိုင်းဖို့ အသုံးပြုနိုင်ပါတယ်။

*   **General Purpose & Logic:** အထွေထွေ ဗဟုသုတနဲ့ Logic အတွက် ကောင်းမွန်တဲ့ Chatbot တွေ ရှိပါတယ်။ (ဥပမာ - ChatGPT, Gemini)
*   **Coding & Large Context:** Code ရေးရာမှာ တိကျပြီး၊ စာလုံးရေ အများကြီး (Large Context) ဖတ်နိုင်တဲ့ Chatbot တွေကို အသုံးပြုသင့်ပါတယ်။ (ဥပမာ - Claude)
*   **Multimodal & Ecosystem:** ပုံ၊ ဗီဒီယိုတွေကို နားလည်နိုင်ပြီး မိမိသုံးနေတဲ့ Google/Microsoft Ecosystem နဲ့ ချိတ်ဆက်နိုင်တဲ့ Chatbot တွေလည်း ရှိပါတယ်။ (ဥပမာ - Gemini, Copilot)

### 2. AI Code Editors (Code ရေးရန်)

*   **Autocomplete Extensions:** VS Code လို Editor တွေမှာ ထည့်သွင်းအသုံးပြုရတဲ့ Extension တွေ ဖြစ်ပါတယ်။ (ဥပမာ - GitHub Copilot)
*   **AI-Native Editors:** AI ကို အဓိကထားပြီး တည်ဆောက်ထားတဲ့ Editor တွေ ဖြစ်ပြီး Codebase တစ်ခုလုံးကို နားလည်နိုင်စွမ်း ရှိကြပါတယ်။ (ဥပမာ - Cursor)

### 3. Other Tools

*   **UI Generators:** စာရိုက်ပြီး UI Design တွေ ရယူနိုင်တဲ့ Tool များ။ (ဥပမာ - v0.dev)
*   **Search Assistants:** Web ပေါ်က အချက်အလက်တွေကို ရှာဖွေပြီး အနှစ်ချုပ်ပေးနိုင်တဲ့ Tool များ။ (ဥပမာ - Perplexity)

### ဘယ်ဟာ သုံးရမလဲ?

*   **Beginner:** Free ရရှိနိုင်တဲ့ Chatbot တစ်ခုခုနဲ့ စတင် လေ့လာပါ။
*   **Pro:** AI-Native Editor တစ်ခုနဲ့ စွမ်းဆောင်ရည် မြင့်မားတဲ့ Chatbot (Logic ကောင်းမွန်သော သို့မဟုတ် Context များများ ဖတ်နိုင်သော bot) ကို တွဲဖက် အသုံးပြုပါ။
