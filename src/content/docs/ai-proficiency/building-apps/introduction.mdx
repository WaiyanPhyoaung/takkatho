<Aside title="Chat UI">
AI App တော်တော်များများက Chat ပုံစံနဲ့ လာပါတယ်။
User က စာရိုက်မယ်၊ AI က ပြန်ဖြေမယ်။ ဒီ UI ကို ကောင်းကောင်း တည်ဆောက်တတ်ဖို့ လိုပါတယ်။
</Aside>

### Vercel AI SDK

Next.js သုံးတဲ့ Developer တွေအတွက် **Vercel AI SDK** က မရှိမဖြစ်ပါ။
သူက Chat UI ဆောက်ဖို့ လိုအပ်တဲ့ Hook တွေ (`useChat`, `useCompletion`) ကို အဆင်သင့် ပေးထားပါတယ်။

### Streaming (စာလုံးချင်း ပေါ်လာခြင်း)

AI က စာအရှည်ကြီး ရေးရင် ကြာတတ်ပါတယ်။ User ကို စောင့်မခိုင်းဘဲ၊ စာလုံးတွေ တစ်လုံးချင်းစီ (Typewriter Effect) ပေါ်လာအောင် လုပ်တာကို **Streaming** လို့ ခေါ်ပါတယ်။
Vercel AI SDK က ဒါကို အလွယ်တကူ လုပ်ပေးနိုင်ပါတယ်။

### Optimistic UI

AI ဖြေတာ မစောင့်ဘဲ၊ User ပို့လိုက်တဲ့ စာကို UI မှာ ချက်ချင်း ပြပေးတာမျိုးပါ။
App ကို ပိုမြန်သယောင် ခံစားရစေပါတယ်။

### Markdown Rendering

AI က Code တွေ၊ List တွေ၊ Table တွေကို Markdown Format နဲ့ ပြန်ပေးလေ့ ရှိပါတယ်။
ဒါကို UI မှာ လှလှပပ ပြနိုင်ဖို့ `react-markdown` လို Library တွေ သုံးရပါမယ်။
