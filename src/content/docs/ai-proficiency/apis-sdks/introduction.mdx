Developer တစ်ယောက်အနေနဲ့ AI Model ကို ကိုယ်တိုင် တည်ဆောက်စရာ မလိုပါဘူး။
OpenAI, Google, Anthropic တို့က ထုတ်ပေးထားတဲ့ API တွေကို လှမ်းချိတ်ပြီး သုံးလိုက်ရုံပါပဲ။
</Aside>

### Popular AI APIs

1.  **OpenAI API:** GPT-4o, GPT-3.5 Turbo (Text Generation), DALL-E (Image Generation).
2.  **Google Gemini API:** Multimodal (စာ၊ ပုံ၊ ဗီဒီယို) အတွက် ကောင်းမွန်ပြီး Free Tier ပေးထားပါတယ်။
3.  **Anthropic API:** Claude 3.5 Sonnet (စာရေးကောင်း၊ Code ရေးကောင်း)။

### How to use (Concept)

API သုံးတာက ရိုးရိုး HTTP Request ပို့တာနဲ့ အတူတူပါပဲ။

1.  **API Key ရယူပါ:** Provider ရဲ့ Website မှာ Account ဖွင့်ပြီး Key ယူပါ။
2.  **Request ပို့ပါ:** User ရဲ့ မေးခွန်း (Prompt) ကို ပို့လိုက်ပါ။
3.  **Response ယူပါ:** AI ပြန်ဖြေတဲ့ စာကို ယူပြီး UI မှာ ပြပေးပါ။

### SDKs

Request တွေကို ကိုယ်တိုင် ရေးမယ့်အစား Library (SDK) တွေကို သုံးတာ ပိုလွယ်ပါတယ်။

*   JavaScript/TypeScript: `npm install openai`
*   Python: `pip install openai`

### Cost Management

API သုံးရင် ပိုက်ဆံကုန်ပါတယ်။ Token (စာလုံးရေ) အလိုက် ပေးရတာပါ။
ဒါကြောင့် မလိုအပ်ဘဲ မခေါ်မိအောင်၊ Caching လုပ်တာတွေ သတိထားရပါမယ်။
