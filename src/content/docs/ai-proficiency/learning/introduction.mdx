### 1. Roadmap ဆွဲခိုင်းပါ

*   **Prompt:** "I know React. I want to learn Next.js. Create a 2-week learning roadmap for me. Focus on the differences between React and Next.js."

### 2. Comparison (နှိုင်းယှဉ်လေ့လာခြင်း)

*   **Prompt:** "What is the difference between `useEffect` and `useLayoutEffect`? Give me a code example where `useLayoutEffect` is necessary."

### 3. Quiz (စာမေးပွဲ စစ်ခိုင်းခြင်း)

*   **Prompt:** "Ask me 5 interview questions about TypeScript Generics. Don't give me the answers yet. Wait for my response."

### 4. Explain Errors

Code ရေးရင်း Error တက်ရင်၊ အဲ့ဒီ Error က ဘာလဲဆိုတာ သေချာ မေးပြီး နားလည်အောင် လုပ်ပါ။ Copy-Paste ပြီး ပြင်ရုံနဲ့ မပြီးပါစေနဲ့။
