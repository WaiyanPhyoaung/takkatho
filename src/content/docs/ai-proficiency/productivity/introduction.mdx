Developer တစ်ယောက်ရဲ့ အလုပ်က Code ရေးတာ တစ်ခုတည်း မဟုတ်ပါဘူး။
Email ပို့ရတယ်၊ Meeting လုပ်ရတယ်၊ Data တွေ ပြင်ရတယ်။ ဒီနေရာတွေမှာ AI ကို သုံးပါ။

### 1. Writing Emails & Messages

Manager ဆီ ခွင့်တိုင်စာ ရေးမလား၊ Client ကို Project Update ပို့မလား။

*   **Prompt:** "Write a polite email to my manager asking for sick leave today because of a fever."
*   **Prompt:** "Rewrite this slack message to be more professional but friendly."

### 2. Data Conversion

JSON ကို CSV ပြောင်းချင်တာ၊ SQL Result ကို Table ပြောင်းချင်တာတွေအတွက် Script ရေးမနေပါနဲ့။ AI ကို ပစ်ထည့်လိုက်ပါ။

*   **Prompt:** "Convert this JSON data into a CSV format."

### 3. Learning New Concepts

Meeting ထဲမှာ နားမလည်တဲ့ စကားလုံး (ဥပမာ - "Kubernetes Ingress") ကြားလိုက်ရင် ချက်ချင်း မေးကြည့်ပါ။

*   **Prompt:** "What is Kubernetes Ingress? Explain like I'm 5."

### 4. CLI Commands

Terminal Command တွေ မေ့နေရင် မေးလိုက်ပါ။

*   **Prompt:** "How do I find all files larger than 100MB in Linux?"
