## Project Overview

ဒီ Capstone Project မှာ မြန်မာနိုင်ငံက Online Shop တစ်ခုအတွက် လိုအပ်မယ့် **Complete Automation System (အလိုအလျောက် စနစ်အပြည့်အစုံ)** တစ်ခုကို အတူတူ တည်ဆောက်ကြည့်ကြပါမယ်။ Course ထဲမှာ သင်ခဲ့သမျှ အချက်အလက်တွေ (Concepts) အကုန်လုံးကို ဒီနေရာမှာ လက်တွေ့ အသုံးချသွားမှာပါ -

| Module | ဘယ်နေရာမှာ သုံးမလဲ |
|--------|-----------|
| Webhook | Order အသစ် ဝင်လာတာကို လက်ခံဖို့ |
| Credentials | Stripe, Gmail, Slack API တွေနဲ့ ချိတ်ဆက်ဖို့ |
| Data Transformation | ဝင်လာတဲ့ Order Data ကို ဖတ်ရလွယ်အောင် ပြင်ဖို့ |
| IF / Branch | VIP Order လား၊ သာမန် (Standard) Order လား ခွဲခြားဖို့ |
| Parallel Actions | Email လည်း ပို့မယ်၊ Slack လည်း ပို့မယ်၊ Database ထဲလည်း ထည့်မယ် (တပြိုင်တည်း) |
| Error Handling | Payment လုပ်တာ Fail ဖြစ်သွားရင် Alert ပေးဖို့ |
| Sub-Workflows | Notification ပို့တဲ့အပိုင်းကို သီးသန့်ခွဲထုတ်ပြီး ပြန်သုံးဖို့ (Reusable) |
| Database | MySQL ထဲမှာ Order မှတ်တမ်းတွေ သွားသိမ်းဖို့ |
| Loop/Batch | Order အများကြီးကို Batch လိုက် Report ထုတ်ဖို့ |
| AI | Customer က လာမေးတဲ့ စာတွေကို AI နဲ့ Auto-reply ပြန်ပေးဖို့ |
| Schedule | နေ့တိုင်း ညနေပိုင်း Sales Report တွေကို Email ပို့ဖို့ |

### Order Processing Code

```javascript
// Code Node — ဝင်လာတဲ့ Order ကို Validate (မှန်မမှန် စစ်ဆေးခြင်း) နဲ့ Enrich (Data ပြည့်စုံအောင် လုပ်ခြင်း)
const order = $input.first().json;

// မဖြစ်မနေ ပါရမယ့် အချက်အလက် (Required Fields) တွေကို စစ်မယ်
const required = ['orderId', 'customer', 'total', 'items'];
const missing = required.filter(f => !order[f]);
if (missing.length > 0) {
  throw new Error(`အချက်အလက် မပြည့်စုံပါ: ${missing.join(', ')}`);
}

// Order Data တွေကို ပိုပြည့်စုံသွားအောင် (Enrich) ထပ်ဖြည့်မယ်
return [{
  json: {
    ...order,
    isVip: order.total > 50000,
    formattedTotal: `${Number(order.total).toLocaleString()} ကျပ်`,
    itemCount: order.items.length,
    orderDate: new Date().toLocaleDateString('my-MM', {
      year: 'numeric', month: 'long', day: 'numeric'
    }),
    estimatedDelivery: new Date(
      Date.now() + (order.isVip ? 2 : 5) * 24 * 60 * 60 * 1000 // VIP ဆို ၂ ရက်၊ သာမန်ဆို ၅ ရက် ကြာမယ်
    ).toLocaleDateString('my-MM')
  }
}];
```

## Workflow 2: Payment Processing

Stripe Payment ကနေ ပိုက်ဆံရှင်းတဲ့ Webhook ဝင်လာတဲ့အခါ -

<BranchFlow
  input="Webhook: Stripe Payment Event ဝင်လာပြီ"
  condition="Payment အောင်မြင်ရဲ့လား (succeeded)?"
  truePath={["MySQL: Update order status='paid'", "Gmail: ငွေရပြေစာ (Receipt) ပို့မယ်", "Sub-WF: Notification ပို့မယ့်အပိုင်းကို ခေါ်မယ်"]}
  falsePath={["MySQL: Update status='failed'", "Gmail: Payment Fail အကြောင်းကြားပြီး ပြန်စမ်းဖို့ Link ပို့မယ်", "Slack: #payments မှာ Urgent Alert ပို့မယ်"]}
  trueLabel="payment_intent.succeeded ✅"
  falseLabel="payment_intent.payment_failed ❌"
/>

```javascript
// Code Node — Stripe က ပို့လိုက်တဲ့ Event ကို Parse လုပ်မယ်
const event = $input.first().json;

const isSuccess = event.type === 'payment_intent.succeeded';
const amount = event.data.object.amount / 100; // cents ကနေ dollars ကို ပြောင်းမယ်

return [{
  json: {
    eventType: event.type,
    orderId: event.data.object.metadata.orderId,
    amount,
    currency: event.data.object.currency.toUpperCase(),
    status: isSuccess ? 'paid' : 'failed',
    failureReason: isSuccess ? null : event.data.object.last_payment_error?.message
  }
}];
```

<FlowDiagram

    { label: "Code: KPI တွေ တွက်ချက်မယ်", type: "code" },
    { label: "Gmail: Report ကို ပိုင်ရှင်ဆီ Email ပို့မယ်", type: "action" }
  ]}
  caption="နေ့တိုင်း မနက် ၈ နာရီမှာ မနေ့က Sales Summary (အရောင်းစာရင်း) ကို Owner ဆီ အလိုအလျောက် Email ပို့ခြင်း"
/>

```javascript
// Code Node — KPI Calculation (စာရင်းတွေ တွက်ချက်မယ်)
const orders = $input.all().map(i => i.json);

const stats = {
  date: new Date(Date.now() - 86400000).toLocaleDateString('my-MM'),
  totalOrders: orders.length,
  totalRevenue: orders.reduce((s, o) => s + o.total, 0),
  avgOrderValue: orders.length > 0
    ? Math.round(orders.reduce((s, o) => s + o.total, 0) / orders.length)
    : 0,
  vipOrders: orders.filter(o => o.is_vip).length,
  cancelledOrders: orders.filter(o => o.status === 'cancelled').length,
  topProducts: orders
    .flatMap(o => JSON.parse(o.items || '[]'))
    .reduce((acc, item) => {
      acc[item.name] = (acc[item.name] || 0) + item.quantity;
      return acc;
    }, {})
};

// အရောင်းရဆုံး ပစ္စည်း (Top product) ကို ရှာမယ်
stats.bestSeller = Object.entries(stats.topProducts)
  .sort(([,a], [,b]) => b - a)[0]?.[0] ?? 'N/A';

return [{ json: stats }];
```

## Implementation Checklist (လုပ်ဆောင်ရမယ့် အဆင့်များ)

```
Phase 1: Setup (ရက် ၁)

Phase 3: Reporting (ရက် ၄)

Phase 4: AI Integration (ရက် ၅)
  □ Customer Message ကောင်း/ဆိုး (Sentiment Analysis) ခွဲမယ်
  □ မကြာခဏ မေးလေ့ရှိတာ (FAQ) တွေကို Auto-reply ပြန်မယ်
```
