## Workflow 1: New Order Processing

New Order ဝင်လာသောအခါ အောက်ပါ Actions တွေကို Run ပေးသည်:

<BranchFlow
  input="Webhook: New Order (from Store)"
  condition="Order Total > 50,000 MMK?"
  truePath={["Set: VIP Flag", "Gmail: VIP Confirmation + Free Shipping", "Slack: #vip-orders alert"]}
  falsePath={["Set: Standard Flag", "Gmail: Standard Confirmation", "Slack: #orders alert"]}
  trueLabel="VIP Order ⭐"
  falseLabel="Standard Order"
/>

### Order Processing Code

```javascript
// Code Node — Order ကို Validate + Enrich လုပ်ပါ
const order = $input.first().json;

// Validate Required Fields
const required = ['orderId', 'customer', 'total', 'items'];
const missing = required.filter(f => !order[f]);
if (missing.length > 0) {
  throw new Error(`Missing fields: ${missing.join(', ')}`);
}

// Enrich Order Data
return [{
  json: {
    ...order,
    isVip: order.total > 50000,
    formattedTotal: `${Number(order.total).toLocaleString()} ကျပ်`,
    itemCount: order.items.length,
    orderDate: new Date().toLocaleDateString('my-MM', {
      year: 'numeric', month: 'long', day: 'numeric'
    }),
    estimatedDelivery: new Date(
      Date.now() + (order.isVip ? 2 : 5) * 24 * 60 * 60 * 1000
    ).toLocaleDateString('my-MM')
  }
}];
```

## Workflow 2: Payment Processing

Stripe Payment Webhook ဝင်လာသောအခါ:

<BranchFlow
  input="Webhook: Stripe Payment Event"
  condition="Payment succeeded?"
  truePath={["MySQL: Update order status='paid'", "Gmail: Receipt Email", "Sub-WF: Send Notification"]}
  falsePath={["MySQL: Update status='failed'", "Gmail: Payment Failed + Retry Link", "Slack: #payments urgent alert"]}
  trueLabel="payment_intent.succeeded ✅"
  falseLabel="payment_intent.payment_failed ❌"
/>

```javascript
// Code Node — Stripe Event ကို Parse လုပ်ပါ
const event = $input.first().json;

const isSuccess = event.type === 'payment_intent.succeeded';
const amount = event.data.object.amount / 100; // cents to dollars

return [{
  json: {
    eventType: event.type,
    orderId: event.data.object.metadata.orderId,
    amount,
    currency: event.data.object.currency.toUpperCase(),
    status: isSuccess ? 'paid' : 'failed',
    failureReason: isSuccess ? null : event.data.object.last_payment_error?.message
  }
}];
```

## Workflow 3: Daily Sales Report

<FlowDiagram
  nodes={[
    { label: "Schedule Trigger", sublabel: "Mon-Sat 8am", type: "trigger" },
    { label: "MySQL: Yesterday's Orders", type: "data" },
    { label: "Code: Calculate KPIs", type: "code" },
    { label: "Gmail: Report to Owner", type: "action" }
  ]}
  caption="နေ့တိုင်း မနက် ၈ နာရီ ကနေ့ Sales Summary ကို Owner ဆီ Email ပို့သည်"
/>

```javascript
// Code Node — KPI Calculation
const orders = $input.all().map(i => i.json);

const stats = {
  date: new Date(Date.now() - 86400000).toLocaleDateString('my-MM'),
  totalOrders: orders.length,
  totalRevenue: orders.reduce((s, o) => s + o.total, 0),
  avgOrderValue: orders.length > 0
    ? Math.round(orders.reduce((s, o) => s + o.total, 0) / orders.length)
    : 0,
  vipOrders: orders.filter(o => o.is_vip).length,
  cancelledOrders: orders.filter(o => o.status === 'cancelled').length,
  topProducts: orders
    .flatMap(o => JSON.parse(o.items || '[]'))
    .reduce((acc, item) => {
      acc[item.name] = (acc[item.name] || 0) + item.quantity;
      return acc;
    }, {})
};

// Top product ကို ရှာပါ
stats.bestSeller = Object.entries(stats.topProducts)
  .sort(([,a], [,b]) => b - a)[0]?.[0] ?? 'N/A';

return [{ json: stats }];
```
