## ဆောက်မည့် Workflow

ဒီသင်ခန်းစာမှာ ရိုးရှင်းတဲ့ Workflow တစ်ခုကို အတူတူ ဆောက်ကြည့်ကြပါမယ် -

Button နှိပ်လိုက်တာနဲ့ Data တွေ ထုတ်ပြပြီး Greeting Message ပြပေးမယ့် ပုံစံပါ။

## Step 1: Workflow အသစ် ဖန်တီးပါ

1. Dashboard ကနေ **"+ New Workflow"** ကို နှိပ်ပါ။
2. Workflow Editor ပွင့်လာပါလိမ့်မယ်။
3. ဘယ်ဘက်အပေါ်ထောင့်က Workflow Name မှာ **"My First Workflow"** လို့ နာမည်ပေးလိုက်ပါ။

## Step 2: Manual Trigger Node ထည့်ပါ

Canvas ပေါ်က **"+"** ကို နှိပ်ပြီး **"On clicking 'execute'"** ကို ရွေးပါ (ဒါဟာ Manual Trigger ပါ)။
Canvas ထဲမှာ "When clicking 'Execute Workflow'" ဆိုတဲ့ Node လေး ရောက်လာပါလိမ့်မယ်။

## Step 3: Set Node ထည့်ပြီး Data သတ်မှတ်ပါ

Manual Trigger Node ရဲ့ Output Point (●) ကနေ Drag ဆွဲပြီး **"Set"** Node ကို ရွေးထည့်ပါ။

**Set Node ကို Configure လုပ်ပါ:**

Set Node ရဲ့ Setting ထဲမှာ **"Add Field"** ကို နှိပ်ပြီး အောက်ပါအတိုင်း ထည့်ပါ -

| Field Name | Value |
|-----------|-------|
| `name` | `Aung Aung` |
| `course` | `n8n Beginner` |
| `message` | `Hello from n8n!` |

Mode ကို "Manual Mapping" အနေနဲ့ ထားပြီး Field တစ်ခုချင်းစီကို ထည့်ပေးရမှာပါ။

## Step 4: Code Node နဲ့ Greeting ဖန်တီးပါ

Set Node ကနေဆက်ပြီး **Code Node** ကို ထပ်ချိတ်ပါ။

Code Node ထဲမှာ အောက်ပါ JavaScript လေး ရေးထည့်ပါ -

```javascript
// Code Node - JavaScript
for (const item of $input.all()) {
  const name = item.json.name;
  const course = item.json.course;

  item.json.greeting = `🎉 မင်္ဂလာပါ ${name}! ${course} Course ကနေ ကြိုဆိုပါတယ်!`;
  item.json.timestamp = new Date().toISOString();
}

return $input.all();
```

<Aside type="note" title="$input.all() ဆိုတာ">
`$input.all()` က အရှေ့ Node ကနေ ဝင်လာတဲ့ Items တွေ အားလုံးကို Array ပုံစံနဲ့ ပြန်ပေးတာပါ။ ပြီးတော့ အဲ့ဒီ Array ကို Loop ပတ်ပြီး Item တစ်ခုချင်းစီကို ပြင်ဆင်တာ ဖြစ်ပါတယ်။
</Aside>

## Step 5: Workflow ကို Run ကြည့်ပါ

Editor ရဲ့ ညာဘက်အပေါ်ထောင့်မှာရှိတဲ့ **"Execute Workflow"** ကို နှိပ်ပါ (Shortcut: `Ctrl+Enter`) -

Expected Result — Code Node ကနေ ဒီလို Output ထွက်လာပါမယ် -

```json
[
  {
    "name": "Aung Aung",
    "course": "n8n Beginner",
    "message": "Hello from n8n!",
    "greeting": "🎉 မင်္ဂလာပါ Aung Aung! n8n Beginner Course ကနေ ကြိုဆိုပါတယ်!",
    "timestamp": "2025-01-15T09:30:00.000Z"
  }
]
```

Node တစ်ခုချင်းစီ အောင်မြင်သွားရင် **Green Checkmark** ✅ လေးတွေ ပေါ်လာမှာပါ။

## ဘာတွေ ဖြစ်သွားတာလဲ? (Data Flow နားလည်ခြင်း)

Data Flow အဆင့်ဆင့်ကို ပြန်ကြည့်ရအောင် -

1. **Manual Trigger** — Output: `[{ json: {} }]` (Empty item လေး ထွက်လာတယ်)
2. **Set Node** — Output: `[{ json: { name: "Aung Aung", course: "n8n Beginner", message: "Hello from n8n!" } }]` (Data တွေ ဖြည့်လိုက်တယ်)
3. **Code Node** — Input အနေနဲ့ အရှေ့က Output ကို ယူပြီး / Output: `[{ json: { ...same, greeting: "🎉...", timestamp: "..." } }]` (Greeting နဲ့ Time ထပ်ပေါင်းထည့်လိုက်တယ်)

## Node ကို Debug လုပ်ခြင်း

Node တစ်ခုကို Click နှိပ်ပြီး **"Input"** / **"Output"** Tab တွေကို ဖွင့်ကြည့်ရင် Data တွေ အဆင့်ဆင့် ဘယ်လို ပြောင်းသွားလဲ မြင်ရပါတယ်။

ဥပမာ Set Node မှာဆိုရင် Input Tab မှာ Empty Object `{}` ကို မြင်ရပြီး၊ Output Tab မှာတော့ ဖြည့်လိုက်တဲ့ `{ name: "Aung", course: "...", message: "..." }` Data တွေကို မြင်ရမှာပါ။

<Aside type="tip" title="Node ကို ခွဲပြီး Test လုပ်ပါ">
Workflow တစ်ခုလုံး Run စရာမလိုဘဲ Node တစ်ခုချင်းစီမှာရှိတဲ့ **"Test Step"** ကို နှိပ်ပြီး Test လို့ ရပါတယ်။ အဲ့ဒါက Debug လုပ်ရတာ ပိုမြန်စေပါတယ်။
</Aside>

## Workflow ကို Save လုပ်ပါ

`Ctrl+S` (သို့) **"Save"** Button နှိပ်ပြီး Workflow ကို သိမ်းထားလိုက်ပါ။

ဒါဆိုရင်တော့ ပထမဆုံး Workflow လေး အောင်မြင်စွာ ဆောက်ပြီးသွားပါပြီ။ နောက်သင်ခန်းစာမှာ **HTTP Request** နဲ့ **Webhook** အကြောင်း လေ့လာပြီး တကယ့် API Integration တွေ စလုပ်ကြပါမယ် ခင်ဗျာ။
