## ဒီသင်ခန်းစာမှာ ဆောက်မယ့် Workflows

ဒီမှာ လက်တွေ့သုံးလို့ရတဲ့ Notification Workflow (၃) ခု ဆောက်ပါမယ် -

1. Gmail → Slack Forwarding (Email ဝင်ရင် Slack ကနေ အကြောင်းကြားဖို့)
2. Form Submit → Email + Slack Notification (Contact Form တင်တာနဲ့ အကြောင်းကြားဖို့)
3. Schedule Daily Report → Email (နေ့စဉ် Report တွေကို အလိုအလျောက် ပို့ဖို့)

## Workflow 1: Gmail → Slack Forwarding

Important Label ပါတဲ့ Email ဝင်လာတိုင်း Slack ကနေ လှမ်း Notify ပေးပါမယ် -

<BranchFlow
  input="Gmail Trigger: New Email (Poll: 1 min)"
  condition='Label = "IMPORTANT"?'
  truePath={["Slack: Post Message in #alerts"]}
  falsePath={["Do nothing"]}
  trueLabel="YES ✅"
  falseLabel="NO ❌"
/>

### Gmail Trigger Setup

1. Node Panel ထဲမှာ **Gmail** ကို Search လုပ်ပါ
2. **Gmail Trigger** ကို ရွေးပါ
3. Credential အနေနဲ့ Google OAuth2 ချိတ်ဆက်ပါ (ရှေ့ Credential သင်ခန်းစာမှာ လုပ်ခဲ့သလိုပါပဲ)

Gmail Trigger Configuration -
- Event: New Email
- Label: IMPORTANT
- Poll Time: Every 1 minute

### IF Node — Label ကို စစ်ပါ

```
Condition:
  {{ $json.labelIds.includes("IMPORTANT") }} === true
```

### Slack Message Format

```
Slack Node Configuration:
  Resource:  Message
  Operation: Post Message
  Channel:   #alerts
  Text:
  *📧 Important Email!*
  From: {{ $json.from[0].address }}
  Subject: {{ $json.subject }}
  Preview: {{ $json.snippet }}
```

### Report Email Template

```javascript
// Code Node — Report ပြုစုခြင်း
const data = $input.first().json;

const report = {
  date: new Date().toLocaleDateString('en-US', { 
    weekday: 'long', year: 'numeric', month: 'long', day: 'numeric' 
  }),
  totalOrders: data.orders.length,
  revenue: data.orders.reduce((sum, o) => sum + o.total, 0),
  topProduct: data.orders
    .flatMap(o => o.items)
    .sort((a, b) => b.quantity - a.quantity)[0]?.name || 'N/A'
};

return [{ json: report }];
```

## Telegram Notification ထည့်ခြင်း

Slack အစား Telegram ကို သုံးချင်ရင် -

### Telegram Bot ဖန်တီးနည်း

1. Telegram မှာ **@BotFather** ကို ရှာပြီး Chat ဖွင့်ပါ
2. `/newbot` Command ရိုက်ထည့်ပါ
3. Bot ရဲ့ Name နဲ့ Username ကို ပေးပါ
4. **Bot Token** တစ်ခု ရလာပါလိမ့်မယ် (ဥပမာ: `123456:ABCdef...`)

### n8n Telegram Node

```
Telegram Node:
  Chat ID:  [သင့် Chat ID သို့မဟုတ် Group ID]
  Text:
  🔔 *New Order Received!*

  Order #{{ $json.orderId }}
  Customer: {{ $json.customerName }}
  Amount: ${{ $json.total }}
  Status: {{ $json.status }}
```

<Aside type="tip" title="Chat ID ရှာနည်း">
`@userinfobot` ကို Telegram မှာ Message ပို့ပြီးတော့ ကိုယ့်ရဲ့ Chat ID ကို သိနိုင်ပါတယ်။
</Aside>
